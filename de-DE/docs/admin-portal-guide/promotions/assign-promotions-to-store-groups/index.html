Promotions zu Store Groups zuweisen

Was diese Anleitung deckt​

Links bestehende Promotionen zu speichern Gruppen, so dass sie für alle Geschäfte innerhalb dieser Gruppen gleichzeitig aktiv.

Schritte​

Step 1: Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt Promotions.

Step 2: Klicken Sie auf die Schaltfläche Assign Promotions to Store Groups (oder gleichwertige Schaltfläche auf Ihrem Bildschirm).

Step 3: Wählen Sie die Aktion(en) aus, die Sie zuordnen möchten. Sie können:

  • ** Checkbox** neben jedem Werbenamen
  • ** Verwenden Sie die Option "Alle auswählen"*, um alle sichtbaren Aktionen auszuwählen (oder mit "Alle auswählen")
  • Suche* für bestimmte Aktionen mit der Suchleiste

Step 4: Sobald Sie Ihre Aktionen ausgewählt haben, klicken Sie auf Next oder klicken Sie auf die nächste Schrittanzeige, um fortzufahren.

Step 5: Wählen Sie die Speichergruppe(n), die diese Promotions erhalten sollte. Sie können:

  • ** Überprüfen Sie das Kontrollkästchen* neben jedem Speichergruppennamen
  • Search* für bestimmte Speichergruppen mit der Suchleiste

Step 6: Überprüfen Sie Ihre Auswahl und klicken Sie auf die Schaltfläche Assign, um die Aktionen auf die ausgewählten Speichergruppen anzuwenden.

Spitze

Promotionen können nur Storegruppen zugeordnet werden, nicht an einzelne Stores. Einmal zugewiesen, werden die Promotionen sofort für alle Geschäfte in diesen Gruppen aktiv und werden auf ihren digitalen Bestellkanälen angezeigt.

Spitze

Sie können auch Aktionen aus dem Bereich Store Groups zuordnen. SiehePromotionen zuweisenfür den Workflow.
