Veröffentlichung 2026.03.16

Veröffentlicht auf März 16, 2026.

Diese Veröffentlichung erfrischt die Atlas Wiki-Homepage mit einem kinematischeren KFC-Branded Hero und verbessert die Byte Portal-Navigationsstruktur, so dass die Menschen durch den Führer mit weniger Reibung bewegen können.

Highlights​

  • Der Homepageheld nutzt jetzt filmische KFC-Filme Die Atlas Wiki Landingpage verfügt nun über einen vollbluteten Bewegungshelden mit KFC-Kochaufnahmen hinter der Headline, der Suche und primären Aufrufen zur Aktion.

  • Die Homepage fühlt sich eher an wie eine gebrannte Produktoberfläche Die Hero-Behandlung trägt jetzt mehr visuelle Energie, während die Hauptsuche und Onboarding Eintrittspunkte vor und Mitte.

  • Homepage-Markt-Flags machen jetzt zuverlässig unter Windows Die Marktkarten verwenden jetzt bildbasierte Flag-Symbole anstelle von plattformabhängigem Emoji, so dass aktuelle Marktkarten korrekt über Nicht-Mac-Geräte anzeigen.

  • *Eigene Marktkarten wurden für deutlichere Labels neu formatiert Die Top-Marktkarten wurden erweitert, so dass Labels wie Australien sauber passen, ohne sich zu verpacken oder zu klopfen.

  • Byte Portal Navigation wird deutlicher organisiert Die Byte Portal Sidebar Struktur wurde gereinigt, so große Abschnitte bewusst mehr lesen und unterstützen eine kohärentere Brotkrust und Surf-Erfahrung.

  • Die aktualisierte Homepage unterstützt noch mobile Layouts Die Heldenüberlagerung und Inhaltsschichtung wurden abgestimmt, so dass der Text, die Suche und die CTA-Kontrollen auf kleineren Bildschirmen lesbar bleiben.

  • Die Homepage enthält jetzt einen Atlas Peek Eintrittspunkt Ein neuerAtlas PeekEin Eintragspunkt wurde der Homepage hinzugefügt, die auf eine dedizierte Vorschauseite verlinkt, die einen app-ähnlichen Bestellprototyp zur Überprüfung beherbergt.

Was verändert​

Homepage Erfahrung​

  • Ein filmischer Vollblut-Videoheld der Homepage hinzugefügt
  • Kept die Held Headline, Suche und CTAs über dem Video
  • Hinzugefügt KFC-branded Overlay-Behandlung, um die Lesbarkeit über bewegte Aufnahmen zu schützen
  • Erhalten Sie die vorhandenen Onboarding-Eintrittspunkte für Frontend Guide und Admin Portal Guide
  • ZusätzlichAtlas PeekCTA und unterstützende Homepage Callout
  • Ersetzte Homepage-Markt-Flag-Emoji mit lokalen Image-Assets für eine konsequentere plattformübergreifende Rendering
  • Erhöht die vorgestellte Marktkarte Größe so Marktnamen sauber auf Windows-Geräten anzeigen

Atlas Peek Vorschau​

  • Hinzugefügt eine gewidmet/sneak-peekSeite im Atlas Wiki für ein reicheres Testerlebnis
  • Eingebettet einen Standalone Atlas Peek Prototyp, der sich mehr wie eine mobile Bestell-App verhält als ein inszeniertes Diadeck
  • Reworked the Prototyp in einen Stateful iPhone-Stil Fluss mit Speicherauswahl, Menübrowsing, Produktdetails, Warenkorb, Checkout und Bestätigung
  • Hinzufügen einer sichtbaren Add-to-cart-Reise mit klareren Produktaktionen, so dass der Pfad von Menü zu Checkout einfach zu verstehen ist
  • Hinzugefügt in-prototype Englisch und lokalisierte Vermittlung, so dass Akteure können beide Einheimische in einer Sitzung inspizieren
  • Tuned die Vorschau Viewport und Scroll-Verhalten, so dass der Prototyp in kleineren Browserfenstern nutzbar bleibt

Navigation und Information Architektur​

  • Reworked the Byte Portal Sidebar Gruppierung, um die Hauptleitstruktur besser reflektieren
  • Geförderte klarere Top-Level-Eintragspunkte über Byte Portal-Inhalte
  • Verbessert, wie geschachtelte Abschnitte für das Brauen und den Zusammenhang mit dem Brotkrümel aufgetaucht sind

Responsive Verhalten​

  • Bestimmte Heldenschichtung so Hintergrundvideo bleibt hinter interaktiven Bedienelementen
  • Tuned Small-Screen Overlay-Verhalten, um die Homepage-Inhalte auf mobilen

In Plain Englisch​

Diese Veröffentlichung macht das Atlas Wiki unmissverständlicher KFC, wenn Sie darauf landen, gibt den Stakeholdern einen klareren Atlas Peek-Prototyp, um zu überprüfen, und macht die Byte Portal-Dokumentation einfacher, wenn Sie beginnen zu erkunden.
