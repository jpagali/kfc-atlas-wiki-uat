Veröffentlichung 2026.03.13

Veröffentlicht auf März 13, 2026.

Diese Veröffentlichung konzentrierte sich darauf, das Atlas Wiki in ein umfassenderes, navigierbares Produkt zu verwandeln. Die größten Verbesserungen rund um das Verlagswesen, die Customer Journey Covering und das Frontend-Guide fühlen sich strukturierter und einfacher zu durchsuchen.

Highlights​

  • *GitHub Pages wurde eingerichtet und veröffentlicht Die Atlas Wiki hat jetzt einen klareren Einsatzpfad und einen veröffentlichten Demo-Snapshot, wodurch es einfacher ist, außerhalb der lokalen Entwicklung zu überprüfen und zu teilen.

  • *Search und Routing wurden für Seiten stabilisiert Das Routing und das eingebettete Diagrammverhalten wurden aufgeräumt, so dass die gehostete Erfahrung zuverlässiger ist.

  • Kundenreisedokumentation wurde ein echter Dokumentationsbaum Anstelle einer einzigen Übersichtsseite wird nun die Customer Journey in Schritt-für-Schritt-Abschnitte mit einzelnen Unterseiten für den Hauptauftragsablauf und unterstützende Fahrten zerlegt.

  • *Customer Engagement Surveys wurden dem Frontend Guide hinzugefügt Ein dedizierter Umfragebereich existiert nun, einschließlich Qualtrics Deckung, so dass das Feedback-Verhalten nach der Bestellung einfacher zu lokalisieren und zu erklären ist.

  • lokalisierte Navigationsunterstützung wurde erweitert Die Website gewann zusätzliche lokalisierte Customer Journey Dokumentation und Navigation Übersetzung Abdeckung.

Was verändert​

Leitfaden für Frontend​

  • Hinzugefügt ein strukturiertes *Customer Journey Abschnitt mit Unterseiten für:
    • Setup Bestellung
    • Finden Sie Produkte
    • Angebote und Rewards finden
    • Produktvorschau und anpassen
    • Bestellen und erhalten
    • Track Order
  • Supporting Journeys für:
    • Zugang Konto und Verwalten Profil
    • Hilfe
    • Finden Sie einen Store
    • Lernen Sie KFC
  • Kundenverlobung Umfragen Dokumentation, einschließlich einer Qualtrics-Seite
  • Aktualisierte Übersicht und Seitenkopie, um die neuere Frontendstruktur zu reflektieren

Plattform- und Site Experience​

  • GitHub Pages-Bereitstellung einrichten
  • Veröffentlicht ein aktuelles Demo Snapshot
  • Feste Atlas Chat Links und Diagramme
  • Verbesserte Seitensuche Routing

Lokalisierung​

  • Hinzugefügt lokalisierte Customer Journey Docs und Navigation Übersetzungen

In Plain Englisch​

Dies war die Veröffentlichung, die die Website fühlte sich mehr wie ein nutzbares Produkt und weniger wie eine lose Sammlung von Seiten. Navigation ist stärker, Bereitstellung ist deutlicher, und die kundenorientierte Dokumentation hat jetzt eine viel bessere Struktur für zukünftige Inhalte.
