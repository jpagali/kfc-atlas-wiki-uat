# Step 5 - Track Order

This step covers the post-purchase experience after the customer places an order.

## What This Step Covers

Track Order should eventually document order confirmation, preparation status, pickup readiness, delivery progress, handoff instructions, receipts, and support entry points. The latest prototype does not yet provide a complete post-checkout tracking journey.

## Journey Map

| Sequence | Customer Need |
|---|---|
| 1 | Confirm the order was placed successfully |
| 2 | Understand preparation or delivery progress |
| 3 | Know when and where to collect pickup orders |
| 4 | Track delivery status and handoff timing |
| 5 | Access help if the order is delayed or incorrect |
