Release 2026.03.16

Released on March 16, 2026.

This release refreshes the Atlas Wiki homepage with a more cinematic KFC-branded hero and improves the Byte Portal navigation structure so people can move through the guide with less friction.

Highlights​

  • The homepage hero now uses cinematic KFC footage The Atlas Wiki landing page now features a full-bleed motion hero using KFC cooking footage behind the headline, search, and primary calls to action.

  • The homepage feels more like a branded product surface The hero treatment now carries more visual energy while keeping the main search and onboarding entry points front and center.

  • Homepage market flags now render reliably on Windows The market cards now use image-based flag icons instead of platform-dependent emoji, so current market cards display correctly across non-Mac devices.

  • Featured market cards were resized for clearer labels The top market cards were widened so labels like Australia fit cleanly without awkward wrapping or truncation.

  • Byte Portal navigation is organized more clearly The Byte Portal sidebar structure was cleaned up so major sections read more intentionally and support a more coherent breadcrumb and browsing experience.

  • The updated homepage still supports mobile layouts The hero overlay and content layering were tuned so the text, search, and CTA controls remain readable on smaller screens.

  • The homepage now includes an Atlas Peek entry point A new Atlas Peek entry point was added to the homepage, linking to a dedicated preview page that hosts a more app-like ordering prototype for review.

What Changed​

Homepage Experience​

  • Added a cinematic full-bleed video hero to the homepage
  • Kept the hero headline, search, and CTAs layered above the video
  • Added KFC-branded overlay treatment to protect readability over moving footage
  • Preserved the existing onboarding entry points for Front-end Guide and Admin Portal Guide
  • Added an Atlas Peek CTA and supporting homepage callout
  • Replaced homepage market flag emoji with local image assets for more consistent cross-platform rendering
  • Increased the featured market card size so market names display cleanly on Windows devices

Atlas Peek Preview​

  • Added a dedicated /sneak-peek page in the Atlas Wiki for a richer prototype review experience
  • Embedded a standalone Atlas Peek prototype that behaves more like a mobile ordering app than a staged slide deck
  • Reworked the prototype into a stateful iPhone-style flow with store selection, menu browsing, product detail, cart, checkout, and confirmation
  • Added a visible add-to-cart journey with clearer product actions so the path from menu to checkout is easy to understand
  • Added in-prototype English and localized switching so stakeholders can inspect both locales in one session
  • Tuned the preview viewport and scrolling behavior so the prototype remains usable in smaller browser windows

Navigation and Information Architecture​

  • Reworked the Byte Portal sidebar grouping to better reflect the main guide structure
  • Promoted clearer top-level section entry points across Byte Portal content
  • Improved how nested sections are surfaced for browsing and breadcrumb context

Responsive Behavior​

  • Adjusted hero layering so background video remains behind interactive controls
  • Tuned smaller-screen overlay behavior to keep the homepage content legible on mobile

In Plain English​

This release makes the Atlas Wiki feel more unmistakably KFC when you land on it, gives stakeholders a clearer Atlas Peek prototype to review, and makes the Byte Portal documentation easier to move through once you start exploring.
