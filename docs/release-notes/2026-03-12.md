# Release 2026.03.12

Released on **March 12, 2026**.

This release was centered on a major expansion of the Admin Portal Guide. The work was mostly about scale and cleanup: adding a large amount of operational content, fixing navigation and links, and making article pages more usable.

## Highlights

- **The Admin Portal Guide was expanded in a major way**
  A very large set of admin topics was added, bringing much broader operational coverage into the Atlas Wiki.

- **Navigation and slugs were cleaned up**
  Sidebar document IDs, markdown links, and index paths were corrected so the growing guide would resolve properly inside Docusaurus.

- **Article structure was improved**
  Admin Portal Guide content was reworked into a more structured format, including article reorganization and image handling improvements.

- **Image behavior became cleaner and more stable**
  Figma splash images were removed, image zoom was added, and an SSR-related image issue was fixed.

- **General docs polish continued**
  The footer content and design-system references were improved to better connect sections of the site.

## What Changed

### Admin Portal Guide

- Expanded the Admin Portal Guide with a large set of new topics and images
- Restructured admin articles to work better as documentation pages
- Fixed sidebar IDs to match Docusaurus slugs
- Fixed internal markdown links in admin content
- Replaced directory-style links with explicit index paths
- Appended `/index` to sidebar IDs after the subdirectory restructure

### Image Handling

- Removed Figma splash images from articles
- Added image zoom behavior
- Replaced the previous location-based image zoom approach to avoid SSR crashes

### Supporting Improvements

- Improved admin guide quality and consistency
- Refreshed footer content
- Added stronger design system links

## In Plain English

This was the release that gave the Admin Portal Guide real weight. It added a lot more operational material, fixed the rough edges that come with a large content expansion, and made the guide much easier to navigate and maintain.
