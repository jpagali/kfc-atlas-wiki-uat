Setting Daypart Visibility on Categories

Summary​

Dayparts are available globally throughout Byte Portal and can be used to configure category visibility via daypart. Using this feature does not impact the orderability of items within a category - it merely provides a hint to the consuming interface (Byte POS, Byte Kiosk, Byte Commerce, etc.) to hide or de-emphasize a category outside of the configured dayparts.

Setting daypart visibility on categories​

Step 1: Starting from the dashboard, click Menus in the sidebar.

Step 2: Click the Categories tab.

Step 3: Select a specific category.

Screenshot placeholder: Put screenshot showing the Menus page with the Categories tab selected and a category row ready to open here.

Step 4: Scroll down to the Dayparts section.

Step 5: Select the dayparts that you would like this category to be visible during. Multiple dayparts can be selected. Note that if no dayparts are selected, the category will be visible throughout the entire business day.

Screenshot placeholder: Put screenshot showing the category drawer with the Dayparts section and one or more selected dayparts here.

Step 6: Click Save on the drawer form.

tip

Category visibility does not change whether the items inside the category are orderable. It only provides a visibility hint to the consuming interface.
