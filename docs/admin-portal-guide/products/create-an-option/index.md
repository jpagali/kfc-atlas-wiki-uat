# Create an Option

## What this guide covers

Creates a customisation group (e.g., “Size”, “Spice Level”) that can be linked to products, allowing customers to personalise their order with choices.

## Steps

**Step 1:** Navigate to the **Products** section using the left navigation menu.

**Step 2:** Click the **Options** tab.

![Step 2](./img/step-2.png)

**Step 3:** Click the **+ Create New Option** button.

**Step 4:** Fill in the option details. Fields marked with * are required.

| Field | What to enter | Notes |
|-------|--------------|-------|
| **Option Code** * | Unique identifier for this option | Use uppercase letters, numbers, and hyphens (e.g., “OPT-SIZE”) |
| **Option Name** * | The customisation category name shown to customers | e.g., “Size”, “Flavor”, “Spice Level” |

![Step 3](./img/step-3.png)

**Step 5:** Click **Create Option** to save the option.

## Notes

:::caution
Clicking **Cancel** discards all unsaved information.
:::

:::tip
The **Option Code** must be unique and cannot be changed after creation, so choose it carefully.
:::
