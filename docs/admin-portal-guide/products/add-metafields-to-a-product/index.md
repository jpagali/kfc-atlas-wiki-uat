# Add Metafields to a Product

## What this guide covers

Attaches custom metadata to a product for market-specific data requirements, such as regulatory tags, integration fields, or compliance information.

## Steps

**Step 1:** Navigate to the **Products** section using the left navigation menu.

**Step 2:** Find the product you want to add metafields to. You can search by Product Name or Product Code.

**Step 3:** Click the three-dot menu next to the product name, then select **Meta**.

![Step 2](./img/step-2.png)

**Step 4:** A drawer will open showing both **Public** and **Private** metafield sections.

**Step 5:** Click the **Add** button to add a new metafield.

**Step 6:** Fill in each metafield with the exact key and value your technical team has specified.

![Step 3](./img/step-3.png)

### To Edit an Existing Metafield

**Step 7:** Click the three-dot menu next to the metafield you want to edit, then select **Edit**.

**Step 8:** Update the key and value as needed, then click **Save**.

![Step 4](./img/step-4.png)

### To Copy a Metafield

**Step 9:** Click the three-dot menu next to the metafield you want to copy, then select **Copy**.

**Step 10:** A new metafield entry will be created with the same key and value. You can edit it if needed.

### To Delete a Metafield

**Step 11:** Click the three-dot menu next to the metafield you want to delete, then select **Delete**.

**Step 12:** A confirmation modal will appear. Click the red **Delete** button to permanently remove the metafield.

**Step 13:** When you are finished adding or modifying metafields, click **Close** to close the drawer.

![Step 5](./img/step-5.png)

## Notes

:::caution
Only add metafields if your technical team has specified the exact keys and values to use. Incorrect metafields may cause integration failures.
:::

:::tip
You can search for metafields by looking in the dropdown or by typing the key name in the search field.
:::

:::tip
Adding metafields to **Public** or **Private** follows the same process.
:::

:::caution
Deleting a metafield is permanent. Confirm you are removing the correct entry before clicking Delete.
:::
