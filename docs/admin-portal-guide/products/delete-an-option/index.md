# Delete an Option

## What this guide covers

Removes an option group from the system when it is no longer needed.

## Steps

**Step 1:** Navigate to the **Products** section using the left navigation menu.

**Step 2:** Click the **Options** tab.

![Step 2](./img/step-2.png)

**Step 3:** Search for the option you want to delete by entering the Option Name, Option Code, or Catalog Tag in the search field.

**Step 4:** Click the three-dot menu next to the option, then select **Delete**.

**Step 5:** A confirmation modal will appear showing all the areas of the system where this option is used. Review this carefully to ensure you are deleting the correct option.

**Step 6:** Click the red **Delete** button to permanently remove the option.

![Step 3](./img/step-3.png)

## Notes

:::caution
Deleting an option is permanent and cannot be undone. The option will be removed from all products that use it.
:::

:::tip
You can search options by Option Name, Option Code, or Catalog Tag to quickly find the item you want to delete.
:::

:::caution
Click **Cancel** if you do not want to proceed with deletion.
:::
