# Copy an Option Value

## What this guide covers

Duplicates an option value to quickly create similar choices within an option group.

## Steps

**Step 1:** Navigate to the **Products** section using the left navigation menu.

**Step 2:** Click the **Option Values** tab.

**Step 3:** Search for the option value you want to copy by entering the Name, Code, or Catalog Tag in the search field.

**Step 4:** Click the three-dot menu next to the option value, then select **Copy**.

![Step 2](./img/step-2.png)

**Step 5:** The copy form will appear with the original option value's information. Update the fields as needed. Fields marked with * are required.

| Field | What to enter | Notes |
|-------|--------------|-------|
| **Option Value Code** * | Unique identifier for the new value | Must be different from the original (e.g., “SIZE-XL-COPY”) |
| **Option Value Name** * | The choice displayed to customers | Can be the same or customized |
| **Display Name** | Shorter label for limited screen space | Defaults to Option Value Name if left blank |
| **Image** | Optional image for this choice | Toggle **Primary Image** to Yes if needed. Click **Add Another Image** to add more. |

**Step 6:** When you are finished editing, click the **Create Option Value** button.

![Step 3](./img/step-3.png)

## Notes

:::caution
The **Option Value Code** must be unique. You cannot use the same code as the original option value.
:::

:::tip
You can search option values by Name, Code, or Catalog Tag.
:::

:::caution
Clicking **Cancel** discards all unsaved changes.
:::
