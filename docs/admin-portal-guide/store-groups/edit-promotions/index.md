# Edit Promotions

## What this guide covers

Adds or removes promotion assignments for a specific store group, allowing you to modify which promotions are active at member stores.

## Steps

**Step 1:** Navigate to the **Store Groups** section using the left-hand navigation menu.

**Step 2:** Find the store group whose promotions you want to modify. Click the **action menu button** (three dots) next to the store group name.

![Step 2](./img/step-2.png)

**Step 3:** Click **Promotions** from the dropdown menu.

![Step 3](./img/step-3.png)

**Step 4:** A promotions drawer will open. Click the **Edit Promotions** button to enter edit mode.

![Step 4](./img/step-4.png)

**Step 5:** Make changes to the promotion assignments:

- **To add a promotion:** Check the checkbox next to promotion names you want to add
- **To remove a promotion:** Uncheck the checkbox next to promotion names you want to remove
- **Search** for specific promotions using the search bar
- Toggle the **"Show other promotions"** switch to view promotions not currently assigned (if checked, you can see all available promotions; if unchecked, you see only currently assigned ones)

![Step 5](./img/step-5.png)

**Step 6:** Review your changes in the summary section showing promotions added and removed. Click the **Assign** button to apply the changes. (The Assign button is only enabled after you make changes.)

:::note
Changes to promotion assignments take effect immediately for all stores in this store group.
:::

:::tip
You can search promotion names in the search bar to quickly find promotions you want to add or remove.
:::
