# Unassign Promotions from Store Group

## What this guide covers

Removes one or more promotions from a store group, immediately deactivating them across all member stores.

## Steps

**Step 1:** Navigate to the **Store Groups** section using the left-hand navigation menu.

**Step 2:** Find the store group whose promotions you want to remove. Click the **action menu button** (three dots) next to the store group name, then click **Promotions**.

![Step 2](./img/step-2.png)

**Step 3:** A drawer will open showing all promotions currently assigned to this store group. To remove a promotion, click the **Unassign** button next to the promotion name.

![Step 3](./img/step-3.png)

**Step 4:** A confirmation dialog will appear. Click **Unassign Promotion** to confirm removal of the promotion from this store group.

![Step 4](./img/step-4.png)

:::caution
Unassigning a promotion immediately deactivates it across all stores in this store group. Customers will no longer see this promotion on their digital ordering channels.
:::

:::tip
If you want to add or remove multiple promotions at once, use the [Edit Promotions](/docs/admin-portal-guide/store-groups/edit-promotions/) workflow instead.
:::
