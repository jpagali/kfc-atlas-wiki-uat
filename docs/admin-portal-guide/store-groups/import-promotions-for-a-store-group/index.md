# Import Promotions for a Store Group

## What this guide covers

Bulk-imports promotion assignments from one store group to another for large-scale campaign setup or configuration duplication.

## Steps

**Step 1:** Navigate to the **Store Groups** section using the left-hand navigation menu.

**Step 2:** Find the store group that will **receive** the promotions (the destination group). Click the **action menu button** (three dots) next to the store group name.

![Step 2](./img/step-2.png)

**Step 3:** Click **Import Promotions** from the dropdown menu.

![Step 3](./img/step-3.png)

**Step 4:** A dialog will open. Select the store group you want to import promotions **from** (the source group). You can search for it using the search bar.

**Step 5 (Optional):** Decide whether to import all or specific promotions:

| Option | Action |
|--------|--------|
| **Import All** | Leave the toggle ON to import every promotion from the source group |
| **Import Specific** | Toggle OFF the "Import All" switch, then check the boxes next to only the promotions you want to import |

![Step 4](./img/step-4.png)

**Step 6:** Review the import summary and click **Save** to complete the import.

:::caution
**Important:** Importing promotions will **replace** any existing promotions linked to the destination store group. This action cannot be undone. Make sure you are importing to the correct store group before confirming.
:::

:::tip
If you only want to add specific promotions without replacing existing ones, use the [Edit Promotions](/docs/admin-portal-guide/store-groups/edit-promotions/) workflow instead.
:::
