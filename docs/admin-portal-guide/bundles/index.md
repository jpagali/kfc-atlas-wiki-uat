# Bundles

This section covers all tasks for managing bundles (meal deals and combos) and their configurable choices in the Byte Commerce Admin Portal.

## Topics

### Bundles

- [Create a Bundle](/docs/admin-portal-guide/bundles/create-a-bundle/) — Build a new combo or meal deal by grouping products together.
- [Edit a Bundle](/docs/admin-portal-guide/bundles/edit-a-bundle/) — Update an existing bundle's name, description, choices, or other details.
- [Copy a Bundle](/docs/admin-portal-guide/bundles/copy-a-bundle/) — Duplicate a bundle to quickly create similar offerings.
- [Add Metafields to a Bundle](/docs/admin-portal-guide/bundles/add-metafields-to-a-bundle/) — Attach custom metadata for integrations or compliance.
- [Add an Image to a Bundle](/docs/admin-portal-guide/bundles/add-an-image-to-a-bundle/) — Upload a display image for the bundle on ordering channels.
- [Delete a Bundle](/docs/admin-portal-guide/bundles/delete-a-bundle/) — Remove a bundle from the catalogue.

### Choices

- [Create a Choice](/docs/admin-portal-guide/bundles/create-a-choice/) — Add a selection slot to a bundle (e.g., "Choose Your Side").
- [Edit a Choice](/docs/admin-portal-guide/bundles/edit-a-choice/) — Update a choice's options or configuration.
- [Copy a Choice](/docs/admin-portal-guide/bundles/copy-a-choice/) — Duplicate a choice to speed up bundle setup.
- [Delete a Choice](/docs/admin-portal-guide/bundles/delete-a-choice/) — Remove a choice from a bundle.
