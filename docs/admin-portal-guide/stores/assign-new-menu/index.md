# Assign New Menu

## What this guide covers

Links a published menu to a store's ordering channel (e.g., Digital, Kiosk, In-Store), determining what customers see when ordering from that location.

## Steps

**Step 1:** Navigate to the **Stores** section using the left-hand navigation menu.

**Step 2:** Search for the store by **Name**, **Store Number**, or **Franchise Code** using the search box.

![Step 2](./img/step-2.png)

**Step 3:** Once you find the store, click the **three-dot menu** (•••) icon to open the options menu.

![Step 3](./img/step-3.png)

**Step 4:** Click **Menus** from the dropdown menu.

**Step 5:** Click the **more menu** button (⋯) or **+ Assign New Menu** button to reveal additional options.

**Step 6:** Click **Assign New Menu**.

**Step 7:** Select a published menu from the list. This menu will be assigned to a specific ordering channel for this store.

| Field | What to enter | Notes |
|-------|--------------|-------|
| **Menu** * | Select from published menus | Only published menus appear here |
| **Channel** * | Select the ordering channel | e.g., Digital, Kiosk, In-Store |

**Step 8:** Click **Assign** to confirm the assignment.

![Step 4](./img/step-4.png)

:::tip
Only published menus are available for assignment. If the menu you need is not in the list, publish it first using [Publish a Menu](/docs/admin-portal-guide/stores/publish-a-menu/).
:::

:::note
Each store can have different menus assigned to different channels. For example, your Digital channel might use one menu while your In-Store Kiosk uses another.
:::
