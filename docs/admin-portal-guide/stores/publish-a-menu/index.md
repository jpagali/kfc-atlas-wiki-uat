# Publish a Menu

## What this guide covers

Publishes the assigned menu to the store's live ordering channels, making menu changes visible to customers.

## Steps

**Step 1:** Navigate to the **Stores** section using the left-hand navigation menu.

**Step 2:** Click the **Publish Menu** button or tab. This opens the publish workflow.

![Step 2](./img/step-2.png)

**Step 3:** Select the **store(s)** you want to publish menus for. You can search by store name, number, or code if the list is long.

![Step 3](./img/step-3.png)

**Step 4:** Select the **channels** you want to publish to (e.g., Digital, Kiosk, In-Store). You can publish to multiple channels at once.

![Step 4](./img/step-4.png)

**Step 5:** Review the changes to be published. Click **Review Changes** if you need to verify what will be sent to customers.

**Step 6:** Once you've reviewed and confirmed, click the **Publish** button to push the menu live to the selected channels.

![Step 5](./img/step-5.png)

:::tip
**Before publishing:** Verify that the correct menu is assigned to each channel in the store's Menu settings. Use [View a Store's Menu](/docs/admin-portal-guide/stores/view-a-stores-menu/) to confirm the assignments.
:::

:::caution
Publishing a menu immediately makes changes visible to customers. Review all changes before clicking Publish.
:::
