# Publish Menu

## What this guide covers

Triggers a menu publish directly from the store's Menus view, pushing the current menu configuration live to ordering channels.

## Steps

**Step 1:** Navigate to the **Stores** section using the left-hand navigation menu.

**Step 2:** Search for the store by **Name**, **Store Number**, or **Franchise Code** using the search box.

![Step 2](./img/step-2.png)

**Step 3:** Once you find the store, click the **three-dot menu** (•••) icon to open the options menu.

![Step 3](./img/step-3.png)

**Step 4:** Click **Menus** from the dropdown menu. This displays the menus assigned to this store by channel.

**Step 5:** Locate the channel you want to publish, and click the **more menu** button (⋯) on that row.

**Step 6:** Click **Publish Menu** from the options menu.

**Step 7:** Review the menu configuration and any pending changes.

**Step 8:** Click the **Publish** button to push the menu live to customers.

![Step 4](./img/step-4.png)

:::tip
**When to use:** Use this method when publishing from within a store's menu view. Use [Publish a Menu](/docs/admin-portal-guide/stores/publish-a-menu/) if you need to publish menus across multiple stores or channels at once.
:::

:::caution
Publishing a menu immediately makes changes visible to customers. Review all changes before clicking Publish.
:::
