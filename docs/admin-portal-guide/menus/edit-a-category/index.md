# Edit a Category

## What this guide covers

Updates a category’s name, display settings, or configuration.

## Steps

**Step 1:** Navigate to the **Menus** section using the left-hand navigation menu.

**Step 2:** Click on the **Categories** folder to view all categories.

![Step 2](./img/step-2.png)

**Step 3:** Find the category you want to edit, click the **action menu** (three dots) in the same row, and select **Edit**.

![Step 3](./img/step-3.png)

**Step 4:** Update any of the following fields:

| Field | What to enter | Notes |
|-------|--------------|-------|
| **Category Name** | The display name for this category | e.g., "Chicken", "Sides", "Desserts". Changes will be visible to customers. |
| **Display Name** | Optional alternate name for specific channels | Use a shorter label if needed on certain platforms. Leave blank to use the Category Name. |
| **Description** | Optional internal notes about this category | Visible only to your team — not shown to customers. |
| **Available Hours** | Optional time window for this category | Restrict availability to specific hours — e.g., "6:00 AM - 11:00 AM". Leave blank for all-day availability. |

![Step 4](./img/step-4.png)

**Step 5:** Once you have made your changes, click **Save** to apply them.

:::note
The category code cannot be edited — it’s permanent and used for system integrations. Changes to category name and availability take effect when the menu is republished.
:::
