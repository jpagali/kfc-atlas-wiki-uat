# Copy a Menu

## What this guide covers

Duplicates an existing menu to use as a starting point for a new menu, saving significant setup time.

## Steps

**Step 1:** Navigate to the **Menus** section using the left-hand navigation menu.

**Step 2:** Find the menu you want to copy in the menus list, click the **action menu** (three dots) in the same row, and select **Copy**.

![Step 2](./img/step-2.png)

**Step 3:** The menu details form will appear. Update the following fields as needed:

| Field | What to enter | Notes |
|-------|--------------|-------|
| **Menu Code** * | A unique identifier for the new menu | Use uppercase letters, numbers, and hyphens only — e.g., `AU-BREAKFAST-2025`. Must be different from the original menu. Cannot be changed after creation. |
| **Menu Name** * | A human-readable name for this menu | e.g., "Australia Breakfast Menu 2025". Shown in the Menus list and when assigning menus to stores. |

All categories, products, and bundles from the original menu are copied automatically.

![Step 3](./img/step-3.png)

**Step 4:** Once you have entered the new menu code and name, click **Create** to save the copied menu.

![Step 4](./img/step-4.png)

:::note
The copied menu will have the same category structure and items as the original. Edit it after creation if you need to make changes.
:::
