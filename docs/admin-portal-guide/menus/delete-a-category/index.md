# Delete a Category

## What this guide covers

Permanently removes a category from the system when it is no longer needed.

## Steps

**Step 1:** Navigate to the **Menus** section using the left-hand navigation menu.

**Step 2:** Click on the **Categories** folder to view all categories.

![Step 2](./img/step-2.png)

**Step 3:** Find the category you want to delete, click the **action menu** (three dots) in the same row, and select **Delete**.

![Step 3](./img/step-3.png)

**Step 4:** A confirmation dialog will appear. Click **Delete** to permanently remove the category.

![Step 4](./img/step-4.png)

:::caution
Deleting a category will remove it from all menus that use it. Products previously in this category will be unassigned and can be reassigned to other categories. This action cannot be undone.
:::
