# Find Serialized Code

## What this guide covers

Locates a specific serialized promotional code within Atlas, used when verifying redemption status, troubleshooting issues, or managing code validity.

## Steps

**Step 1:** Navigate to the **Promotions** section using the left-hand navigation menu.

**Step 2:** Click the **Find Serialized Code** button (usually visible near the promotions list).

![Step 2](./img/step-2.png)

**Step 3:** Enter the serialized code you want to look up in the **search field** and click the **Search** button.

![Step 3](./img/step-3.png)

**Step 4:** The system will display the code details including:

- **Code Value** — The actual code text
- **Promotion Name** — The promotion this code is tied to
- **Status** — Whether the code is active, redeemed, or voided
- **Expiry Date** — When the code expires

![Step 4](./img/step-4.png)

**Step 5 (Optional):** If you need to deactivate a code, click the **Void Code** button. A voided code cannot be redeemed.

:::note
Use this feature to verify that a code exists, check its redemption status, or troubleshoot customer issues with specific codes.
:::
