# Assign Promotions to Store Groups

## What this guide covers

Links existing promotions to store groups, making them active for all stores within those groups simultaneously.

## Steps

**Step 1:** Navigate to the **Promotions** section using the left-hand navigation menu.

**Step 2:** Click the **Assign Promotions to Store Groups** button (or equivalent button on your screen).

![Step 2](./img/step-2.png)

**Step 3:** Select the promotion(s) you want to assign. You can:

- **Check the checkbox** next to each promotion name
- **Use the "Select All" option** to select all visible promotions (or deselect using "Deselect All")
- **Search** for specific promotions using the search bar

![Step 3](./img/step-3.png)

**Step 4:** Once you've selected your promotions, click **Next** or click on the next step indicator to proceed.

**Step 5:** Select the store group(s) that should receive these promotions. You can:

- **Check the checkbox** next to each store group name
- **Search** for specific store groups using the search bar

**Step 6:** Review your selections and click the **Assign** button to apply the promotions to the selected store groups.

![Step 4](./img/step-4.png)

:::note
Promotions can only be assigned to store groups, not to individual stores. Once assigned, the promotions become active immediately for all stores in those groups and are displayed on their digital ordering channels.
:::

:::tip
You can also assign promotions from the Store Groups section. See [Assign Promotions](/docs/admin-portal-guide/store-groups/assign-promotions/) for that workflow.
:::
