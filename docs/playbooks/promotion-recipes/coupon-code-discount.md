# Coupon Code Discount

## Goal

$10 off for customers who enter the code `SAVE10`.

## Basic Information

| Field | Value |
|---|---|
| **Automatic** | `No` |
| **Add Redemption Code** | `SAVE10` |

## Requirements

| Area | Field | Value |
|---|---|---|
| **Optional** | **Max Total Uses Per Customer** | `1` to limit to one use per customer |
| **Optional** | **Max Number of Times a Promotion Can Be Used** | Total cap on redemptions |

## Effects

| Area | Field | Value |
|---|---|---|
| **Order Level** | **Pricing Effect Type** | `Amount Off` |
| **Order Level** | **Amount Off** | `$10.00` |
