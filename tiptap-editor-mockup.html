Link popover

Replaces window.prompt(). Paste a URL, hit Enter or Apply. In real Tiptap: pre-fills if cursor is already on a link so you can edit it in place.

Image popover + live preview

Type or paste a URL and a thumbnail appears immediately so you can confirm the image loads before inserting. Broken URLs stay flagged in the Readiness panel.

Keyboard friendly

Enter confirms, Escape closes. Clicking anywhere outside the popover also dismisses it — no stray dialogs left open.
