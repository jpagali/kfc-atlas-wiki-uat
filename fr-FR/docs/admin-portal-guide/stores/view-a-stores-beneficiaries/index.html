Voir un magasin Bénéficiaires

Ce que ce guide couvre​

Liste les bénéficiaires liés à un magasin spécifique pour la visibilité et la gestion.

Étapes​

Step 1: Naviguez dans la section Stores en utilisant le menu de navigation de gauche.

Step 2: Recherchez le magasin par Nom, Numéro de magasin ou Code de franchise à l'aide de la boîte de recherche.

Step 3: Une fois que vous trouvez le magasin, cliquez sur le menu ** à trois points** (••) pour ouvrir le menu des options.

Step 4: Cliquez sur Bénéficiaires dans le menu déroulant. Cela affiche tous les bénéficiaires associés au magasin sélectionné.

Step 5: Utilisez la boîte de recherche pour filtrer les bénéficiaires par name si la liste est longue.

Le tableau montre:

  • Nom du bénéficiaire — Le nom de l'organisme de bienfaisance ou de la cause
  • Acceptation des dons — Indicateur de l'état d'avancement (clause verte = dons actifs)
astuce

Pour retirer un magasin spécifique d'un bénéficiaire, allez àModifier/supprimer un bénéficiaireet désélectionnez le magasin de la liste des magasins associés.
