# Almuerzo especial

## Objetivo

$2 fuera de cualquier pedido colocado entre las 11am y las 2pm, de lunes a viernes.

## Necesidades

| Zona | Campo | Valor |
|---|---|---|
| *El marco temporal* | *El rango de día* | `Mo Tu We Th Fr` |
| *El marco temporal* | ** Hora de inicio de la intervención** | `11:00:00` |
| *El marco temporal* | # Interval End Time # | `14:00:00` |
| *El marco temporal* | **Valida contra el tiempo actual** | `Yes` |

## Efectos

| Zona | Campo | Valor |
|---|---|---|
| * Nivel de ordenación* | **Tipo de efecto de fijación** | `Amount Off` |
| * Nivel de ordenación* | **Suficiente* | `$2.00` |
