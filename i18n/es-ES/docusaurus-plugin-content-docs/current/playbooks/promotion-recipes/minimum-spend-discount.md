# Descuento mínimo para gastar

## Objetivo

El cliente obtiene $5 de cualquier pedido más de $30.

## Necesidades

| Zona | Campo | Valor |
|---|---|---|
| * Subtotal* | **Minimum Subtotal** | `$30.00` |
| * Subtotal* | **Modo de cálculo** | `Actual Subtotal` |

## Efectos

| Zona | Campo | Valor |
|---|---|---|
| * Nivel de ordenación* | **Tipo de efecto de fijación** | `Amount Off` |
| * Nivel de ordenación* | **Suficiente* | `$5.00` |
