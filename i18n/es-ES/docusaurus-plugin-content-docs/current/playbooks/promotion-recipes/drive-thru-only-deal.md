# Drive-Thru Only Deal

## Objetivo

Actualización gratuita disponible sólo en Drive Thru.

## Necesidades

| Zona | Campo | Valor |
|---|---|---|
| Ocasión** | Ocasión** | `Drive Thru` |

## Efectos

| Zona | Campo | Valor |
|---|---|---|
| * Nivel de emisión* | **Tipo de efecto de fijación** | `Fixed Price` |
| * Nivel de emisión* | **Precio fijo** | `Target price` |
| * Nivel de emisión* | ** Tema de calificación** | `The item being upgraded` |
