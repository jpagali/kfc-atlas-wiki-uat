# Coupon Code Discount

## Objetivo

$10 para clientes que entran en el código`SAVE10`.

## Información básica

| Campo | Valor |
|---|---|
| *Automático* | `No` |
| **Agregar el Código de la Redención** | `SAVE10` |

## Necesidades

| Zona | Campo | Valor |
|---|---|---|
| *Opcional* | **Max Total Uses Por Cliente** | `1`limitar a un uso por cliente |
| *Opcional* | **Max Number of Times a Promotion Can be used** | Gorro total de redención |

## Efectos

| Zona | Campo | Valor |
|---|---|---|
| * Nivel de ordenación* | **Tipo de efecto de fijación** | `Amount Off` |
| * Nivel de ordenación* | **Suficiente* | `$10.00` |
