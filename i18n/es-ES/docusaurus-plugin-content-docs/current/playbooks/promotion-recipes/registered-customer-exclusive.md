# Exclusivo para clientes registrados

## Objetivo

15% de descuento para los miembros de lealtad conectados solamente.

## Necesidades

| Zona | Campo | Valor |
|---|---|---|
| **Estatus de usuario** | **¿Es cliente registrado?** | `Yes` |

## Efectos

| Zona | Campo | Valor |
|---|---|---|
| * Nivel de ordenación* | **Tipo de efecto de fijación** | `Percent Off` |
| * Nivel de ordenación* | *Porcentaje* | `15` |
