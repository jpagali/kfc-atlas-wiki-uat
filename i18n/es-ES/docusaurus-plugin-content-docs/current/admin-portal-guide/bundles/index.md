# Bundles

Esta sección cubre todas las tareas para la gestión de paquetes (paquetes de comida y combos) y sus opciones configurables en el Portal de Admin de Byte Commerce.

## Temas

### Bundles

- [Crear un Bundle](/docs/admin-portal-guide/bundles/create-a-bundle/)— Construir un nuevo acuerdo de combo o comida agrupando productos juntos.
- [Editar un Bundle](/docs/admin-portal-guide/bundles/edit-a-bundle/)— Actualizar el nombre, descripción, opciones u otros detalles del paquete existente.
- [Copiar un Bundle](/docs/admin-portal-guide/bundles/copy-a-bundle/)— Duplicar un paquete para crear rápidamente ofertas similares.
- [Añadir Metafields a un Bundle](/docs/admin-portal-guide/bundles/add-metafields-to-a-bundle/)— Adjunte metadatos personalizados para integraciones o cumplimiento.
- [Añadir una imagen a un Bundle](/docs/admin-portal-guide/bundles/add-an-image-to-a-bundle/)— Sube una imagen de visualización para el paquete en los canales de pedidos.
- [Borrar un Bundle](/docs/admin-portal-guide/bundles/delete-a-bundle/)- Quitar un paquete del catálogo.

### Opciones

- [Crear una elección](/docs/admin-portal-guide/bundles/create-a-choice/)— Añade una ranura de selección a un paquete (por ejemplo, "Elige tu lado").
- [Editar una elección](/docs/admin-portal-guide/bundles/edit-a-choice/)— Actualizar las opciones o configuración de una opción.
- [Copiar una Elección](/docs/admin-portal-guide/bundles/copy-a-choice/)- Duplicar una opción para acelerar la configuración del paquete.
- [Eliminar una elección](/docs/admin-portal-guide/bundles/delete-a-choice/)- Quitar una opción de un paquete.
