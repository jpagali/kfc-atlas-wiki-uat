# Crear una elección

## Qué cubre esta guía

Crea una ranura de selección dentro de un paquete (por ejemplo, "Elige tu lado" o "Selecciona tu bebida"), permitiendo a los clientes personalizar su combo seleccionando de los productos disponibles.

## Pasos

**Step 1:** Navegue a la sección **Bundles** utilizando el menú de navegación de la mano izquierda.

**Step 2:** Haga clic en la pestaña **Choices** en la parte superior de la pantalla Bundles.

![Paso 2](./img/step-2.png)

**Step 3:** Haga clic en el botón **+ Crear una nueva elección**.

![Paso 3](./img/step-3.png)

**Step 4:** Rellene los detalles de la elección:

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| * Código de Justicia* | Unico identificador | Use letras mayúsculas, números e hifénes — por ejemplo,`CHOICE-SIDE` |
| ** Nombre de la oficina** | Label mostrada a clientes | Por ejemplo, "Elige tu lado", "Selecciona tu bebida" |
| **Choice Display Name** | Nombre más corto para espacio limitado | Defaults to Choice Name if left blank |
| *Min Quantity* | Selección mínima requerida | Set to`0`para hacer la elección opcional |
| **Max Quantity** | Máximas selecciones permitidas | por ejemplo,`1`para una sola elección,`2`para doble selección |

**Step 5:** Añadir productos y variantes a esta opción:

- Haga clic en el campo **Productos** y busque el producto que desee incluir
- Seleccione el producto — aparece una lista de variantes disponibles
- Compruebe las variantes que desea que los clientes sean capaces de seleccionar (por ejemplo, "Risas modulares", "Fres Large")
- Haga clic en **Añadir** para incluir este producto

**Step 6:** Repita el Paso 5 para cualquier producto o variante adicional que desee en esta elección.

**Step 7:** Haga clic en **Crear elección** para guardar la elección.

![Paso 4](./img/step-4.png)

:::
Usted puede buscar opciones más tarde por Nombre de Elección, Código de Elección, Nombre de Pantalla de Elección, o Catálogo Tags para encontrar rápidamente lo que necesita.
:::
