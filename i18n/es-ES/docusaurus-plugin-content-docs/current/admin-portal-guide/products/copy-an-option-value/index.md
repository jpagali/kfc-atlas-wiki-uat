# Copiar un valor de opción

## Qué cubre esta guía

Duplica un valor de opción para crear rápidamente opciones similares dentro de un grupo de opciones.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Option Values**.

**Step 3:** Busque el valor de opción que desea copiar introduciendo el Nombre, Código o etiqueta del catálogo en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto al valor de opción, luego seleccione **Copy**.

![Paso 2](./img/step-2.png)

**Step 5:** El formulario de copia aparecerá con la información del valor de opción original. Actualizar los campos según sea necesario. Se requieren campos marcados con *.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| ** Código de Valor de la Aplicación** | Unico identificador para el nuevo valor | Debe ser diferente del original (por ejemplo, “SIZE-XL-COPY”) |
| **Option Value Name** | La elección mostrada a los clientes | Puede ser el mismo o personalizado |
| *Nombre del juego* | Etiquetas más cortas para espacio de pantalla limitado | Defaults to Option Value Name if left blank |
| **Imagen* | Imagen opcional para esta elección | Toggle **Imagen primaria** a Sí si es necesario. Haga clic en **Añadir otra imagen** para añadir más. |

**Step 6:** Cuando termine de editar, haga clic en el botón **Crear valor de opción**.

![Paso 3](./img/step-3.png)

## Notas

:::caution
El **Option Value Code** debe ser único. No puede utilizar el mismo código que el valor de opción original.
:::

:::
Puedes buscar valores de opción por Nombre, Código o etiqueta de catálogo.
:::

:::caution
Clicking **Cancel** descarta todos los cambios sin salvar.
:::
