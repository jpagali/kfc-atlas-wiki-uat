# Copiar un Modificador

## Qué cubre esta guía

Duplica un modificador para construir rápidamente complementos similares sin volver a introducir toda la información.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Modificadores**.

**Step 3:** Busque el modificador que desea copiar introduciendo el Nombre, Código o etiqueta de catálogo en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto al modificador, luego seleccione **Copiar**.

![Paso 2](./img/step-2.png)

**Step 5:** El formulario de copia aparecerá con la información del modificador original. Actualizar los campos según sea necesario. Se requieren campos marcados con *.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| * Código modificador* | Identificador único para el nuevo modificador | Debe ser diferente del original (por ejemplo, “MOD-EXTRA-CHEESE-COPY”) |
| ** Nombre del modificador** | Nombre mostrado a los clientes | Puede ser el mismo o personalizado |
| *Precio* | Cargo adicional para este modificador | Entra`0`si no hay cargo extra |
| **Imagen* | Imagen opcional para este modificador | Toggle **Imagen primaria** a Sí si es necesario. Haga clic en **Añadir otra imagen** para añadir más. |

**Step 6:** Cuando termine de editar, haga clic en el botón **Crear Modificador**.

![Paso 3](./img/step-3.png)

## Notas

:::caution
El ** Código modificador** debe ser único. No puede utilizar el mismo código que el modificador original.
:::

:::
Puede buscar modificadores por Nombre, Código o etiqueta de catálogo.
:::

:::caution
Clicking **Cancel** descarta todos los cambios sin salvar.
:::
