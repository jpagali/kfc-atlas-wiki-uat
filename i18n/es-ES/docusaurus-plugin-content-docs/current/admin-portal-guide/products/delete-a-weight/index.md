# Eliminar un peso

## Qué cubre esta guía

Elimina una configuración de peso cuando ya no es aplicable.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Pesas**.

![Paso 2](./img/step-2.png)

**Step 3:** Busque el peso que desee eliminar introduciendo el nombre de peso o el código de peso en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto al peso, luego seleccione **Delete**.

**Step 5:** Una confirmación modal aparecerá mostrando todas las áreas del sistema donde se utiliza este peso. Revise esto cuidadosamente para asegurarse de que está eliminando el peso correcto.

**Step 6:** Haga clic en el botón rojo **Eliminar** para eliminar permanentemente el peso.

![Paso 3](./img/step-3.png)

## Notas

:::caution
Eliminar un peso es permanente y no se puede deshacer. El peso se eliminará de todas las ranuras y modificadores que lo utilizan.
:::

:::
Puede buscar pesos por Peso Nombre o Código de Peso para encontrar rápidamente el artículo que desea eliminar.
:::

:::caution
Haga clic en **Cancel** si no desea proceder con la eliminación.
:::
