# Suprimir un valor de opción

## Qué cubre esta guía

Elimina una elección específica de un grupo de opciones cuando se suspende o ya no se necesita.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Option Values**.

![Paso 2](./img/step-2.png)

**Step 3:** Busque el valor de opción que desea eliminar introduciendo el Nombre, Código o etiqueta del catálogo en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto al valor de opción, luego seleccione **Delete**.

**Step 5:** Un modal de confirmación aparecerá mostrando todas las áreas del sistema donde se utiliza este valor de opción. Revise esto cuidadosamente para asegurarse de que está eliminando la elección correcta.

**Step 6:** Haga clic en el botón rojo **Eliminar** para eliminar permanentemente el valor de opción.

![Paso 3](./img/step-3.png)

## Notas

:::caution
Eliminar un valor de opción es permanente y no se puede deshacer. La elección se eliminará de todos los productos que lo ofrecen.
:::

:::
Usted puede buscar valores de opción por Nombre, Código o etiqueta de catálogo para encontrar rápidamente el artículo que desea eliminar.
:::

:::caution
Haga clic en **Cancel** si no desea proceder con la eliminación.
:::
