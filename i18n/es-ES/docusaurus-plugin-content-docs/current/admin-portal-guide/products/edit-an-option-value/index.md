# Editar un valor de opción

## Qué cubre esta guía

Actualiza el nombre del valor de opción existente, las propiedades de visualización o las imágenes para reflejar los cambios del menú.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Option Values**.

**Step 3:** Busque el valor de opción que desea editar introduciendo el Nombre, Código o etiqueta de catálogo en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto al valor de opción, luego seleccione **Editar**.

![Paso 2](./img/step-2.png)

**Step 5:** Actualizar los detalles de valor de opción. Se requieren campos marcados con *.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| ** Código de Valor de la Aplicación** | Unico identificador | No se puede cambiar después de la creación |
| **Option Value Name** | La elección mostrada a los clientes | por ejemplo, “Large”, “Receta Original”, “Hot &apos; Spicy” |
| *Nombre del juego* | Etiquetas más cortas para espacio de pantalla limitado | Defaults to Option Value Name if left blank |
| **Imagen* | Imagen opcional para esta elección | Toggle **Primary Image** a Sí si esta es la imagen principal de la pantalla. Haga clic en **Añadir otra imagen** para añadir más. |

**Step 6:** Cuando termine de editar, haga clic en el botón **Guardar**.

![Paso 3](./img/step-3.png)

## Notas

:::caution
Clicking **Cancel** descarta todos los cambios sin salvar.
:::

:::
Toggle **Primary Image** a **Sí** para establecer esta imagen como la imagen principal de la pantalla para este valor de opción.
:::

:::
Puede añadir varias imágenes haciendo clic en **Añadir otra imagen**.
:::

:::
Puedes buscar valores de opción por Nombre, Código o etiqueta de catálogo.
:::
