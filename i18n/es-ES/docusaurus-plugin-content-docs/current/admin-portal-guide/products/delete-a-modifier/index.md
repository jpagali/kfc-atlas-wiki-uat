# Eliminar un Modificador

## Qué cubre esta guía

Elimina permanentemente un modificador del sistema cuando ya no es necesario.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Modificadores**.

![Paso 2](./img/step-2.png)

**Step 3:** Busque el modificador que desea eliminar introduciendo el Nombre, Código o etiqueta de catálogo en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto al modificador, luego seleccione **Eliminar**.

**Step 5:** Un modal de confirmación aparecerá mostrando todas las áreas del sistema donde se utiliza este modificador. Revise esto cuidadosamente para asegurarse de que está eliminando el modificador correcto.

**Step 6:** Haga clic en el botón rojo **Eliminar** para eliminar permanentemente el modificador.

![Paso 3](./img/step-3.png)

## Notas

:::caution
Eliminar un modificador es permanente y no se puede deshacer. El modificador se eliminará de todas las ranuras y productos que lo utilizan.
:::

:::
Puede buscar modificadores por Nombre, Código o etiqueta de catálogo para encontrar rápidamente el artículo que desea eliminar.
:::

:::caution
Haga clic en **Cancel** si no desea proceder con la eliminación.
:::
