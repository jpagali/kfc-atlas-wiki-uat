# Eliminar una Ranura

## Qué cubre esta guía

Elimina una ranura del sistema cuando la configuración ya no es necesaria.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Slots**.

![Paso 2](./img/step-2.png)

**Step 3:** Busque la ranura que desea eliminar al entrar en el Nombre de Ranura, Código de Ranura, o Etiqueta en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto a la ranura, a continuación, seleccione **Delete**.

**Step 5:** Una confirmación modal aparecerá mostrando todas las áreas del sistema donde se utiliza esta ranura. Revise esto cuidadosamente para asegurarse de que está eliminando la ranura correcta.

**Step 6:** Haga clic en el botón rojo **Eliminar** para eliminar permanentemente la ranura.

![Paso 3](./img/step-3.png)

## Notas

:::caution
Eliminar una ranura es permanente y no se puede deshacer. La ranura se eliminará de todos los productos que lo utilizan.
:::

:::
Usted puede buscar ranuras por Nombre de Ranura, Código de Ranura, o Tag para encontrar rápidamente el artículo que desea eliminar.
:::

:::caution
Haga clic en **Cancel** si no desea proceder con la eliminación.
:::
