# Copiar una opción

## Qué cubre esta guía

Duplica un grupo de opción para ahorrar tiempo al crear conjuntos de personalización similares a través de múltiples productos.

## Pasos

**Step 1:** Navegue a la sección **Productos** usando el menú de navegación izquierdo.

**Step 2:** Haga clic en la pestaña **Opciones**.

**Step 3:** Busque la opción que desea copiar introduciendo el Nombre de opción, Código de opción o etiqueta del catálogo en el campo de búsqueda.

**Step 4:** Haga clic en el menú de tres puntos junto a la opción, luego seleccione **Copiar**.

![Paso 2](./img/step-2.png)

**Step 5:** El formulario de copia aparecerá con la información de la opción original. Actualizar los campos según sea necesario. Se requieren campos marcados con *.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| * Código de aprobación* | Unico identificador para la nueva opción | Debe ser diferente al original. Use letras mayúsculas, números e hifénes (por ejemplo, “OPT-SIZE-COPY”) |
| **Option Name** | El nombre de la categoría de personalización mostrado a los clientes | Puede ser el mismo que el original o personalizado |

**Step 6:** Cuando termines con tus ediciones, haz clic en **Crear**.

![Paso 3](./img/step-3.png)

## Notas

:::caution
El código **Option** debe ser único. No puede utilizar el mismo código que la opción original.
:::

:::
Puedes buscar opciones por Option Name, Option Code o Catalog Tag.
:::

:::caution
Clicking **Cancel** descarta todos los cambios sin salvar.
:::
