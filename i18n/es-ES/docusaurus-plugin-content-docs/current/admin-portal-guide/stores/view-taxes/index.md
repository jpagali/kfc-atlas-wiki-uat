# Ver impuestos

## Qué cubre esta guía

Muestra todos los grupos de reglas tributarias y las reglas tributarias asociadas aplicadas a una tienda, separadas por grupo de tiendas, utilizados para verificar la configuración impositiva con fines de cumplimiento.

## Pasos

**Step 1:** Navegue a la sección **Stores** utilizando el menú de navegación de la mano izquierda.

**Step 2:** Buscar en la tienda por **Name**, **Número de página**, o ** Código de Franquicia** utilizando el cuadro de búsqueda.

![Paso 2](./img/step-2.png)

**Step 3:** Una vez que encuentre la tienda, haga clic en el menú ** de tres puntos** (••••) icono para abrir el menú de opciones.

**Step 4:** Haga clic en **Taxes** del menú desplegable. Esto muestra todos los grupos de reglas de impuestos y reglas configuradas para la tienda seleccionada.

![Paso 3](./img/step-3.png)

**Step 5:** Revisa las reglas de impuestos mostradas. La mesa está organizada por **Store Group**, mostrando:
- **Store Group Name** — El grupo de la tienda esta regla de impuestos pertenece a
- **Tax Rule Group** El nombre del grupo de reglas fiscales
- **Reglas de impuestos** - Reglas fiscales individuales aplicadas (por ejemplo, tasas federales, estatales y locales)
- **Tax Rate** Porcentaje o cantidad aplicada

:::note
La información fiscal se separa por grupo de tiendas porque diferentes grupos de tiendas pueden tener diferentes requisitos fiscales por región o jurisdicción. Si tu tienda pertenece a varios grupos de tiendas, verás secciones separadas para cada uno.
:::

:::
Utilice esta vista para verificar que las tasas de impuestos correctas están configuradas para la ubicación de su tienda antes de publicar menús para canales de pedidos en vivo.
:::
