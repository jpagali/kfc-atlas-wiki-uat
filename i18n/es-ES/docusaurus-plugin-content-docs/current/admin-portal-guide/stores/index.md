# Tiendas

Esta sección cubre todas las tareas relacionadas con las tiendas en el Portal de Admin de Byte Commerce, incluyendo la configuración de tiendas, la gestión de menús, la visibilidad de los elementos y el manejo de donaciones.

:::note Byte POS Caveat
Tareas de almacén que mencionan **POS**, ** autenticación de dispositivos**, o **store números que coinciden con los registros de POS** asumen que el mercado está utilizando **Byte POS**.

Si el mercado no está utilizando Byte POS, los equipos deben tratar **Byte Connect** como parte de Byte Commerce a bordo, por lo que el camino de integración de la tienda es **Byte Commerce - prenda Byte Connect - título POS**.
:::
