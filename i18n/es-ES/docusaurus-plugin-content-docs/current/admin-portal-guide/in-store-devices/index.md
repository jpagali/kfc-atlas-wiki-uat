# Dispositivos intraestables

Esta sección cubre la gestión de terminales y quioscos POS (punto de venta) en restaurantes KFC, incluyendo configuración de dispositivos, estado y autenticación.

:::note Byte POS Caveat
Las guías de dispositivo en tienda a continuación están escritas para los mercados usando **Byte POS**.

Si un mercado no está utilizando Byte POS, **Byte Connect** debe ser incluido en el paquete Byte Commerce onboarding, y los equipos deben entender el camino del pedido como **Byte Commerce - título Byte Connect - Propiedad POS** en lugar de asumir la comunicación directa de Byte Commerce al mercado POS.
:::

## Temas

- [Generar contraseña de un tiempo](/docs/admin-portal-guide/in-store-devices/generate-one-time-password/)— Crear un código de autenticación temporal para la configuración o re-authenticación de dispositivos.
- [Ver detalles del dispositivo dentro de la página](/docs/admin-portal-guide/in-store-devices/view-in-store-device-details/)— Mostrar detalles completos de un dispositivo que incluye nombre, tipo, estado y historia de publicación del menú.
- [Desactivar In-Store](/docs/admin-portal-guide/in-store-devices/deactivate-in-store/)- Desactivar un dispositivo de recibir actualizaciones del menú o pedidos de procesamiento.
