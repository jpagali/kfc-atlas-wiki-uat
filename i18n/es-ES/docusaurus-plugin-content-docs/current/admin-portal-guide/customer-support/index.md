# Atención al cliente

Esta sección cubre las herramientas de gestión de pedidos y clientes para los agentes de soporte del Portal de Admin de Byte Commerce.

## Temas

### Órdenes

- [Búsqueda de orden](/docs/admin-portal-guide/customer-support/order-search/)— Busque pedidos específicos de clientes y administre reembolsos o entrega de recibos.

### Clientes

- [Búsqueda de clientes](/docs/admin-portal-guide/customer-support/customer-search/)— Buscar cuentas de clientes por nombre, correo electrónico o teléfono para ayudar con problemas de cuenta.
