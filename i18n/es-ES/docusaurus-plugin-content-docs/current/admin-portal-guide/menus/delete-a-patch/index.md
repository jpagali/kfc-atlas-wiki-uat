# Eliminar un parche

## Qué cubre esta guía

Elimina permanentemente un parche del sistema.

## Pasos

**Step 1:** Navegue a la sección **Menus** usando el menú de navegación de la mano izquierda.

**Step 2:** Haga clic en la pestaña **Patches** para ver todos los parches.

![Paso 2](./img/step-2.png)

**Step 3:** Busque el parche que desee eliminar, haga clic en el menú **action** (tres puntos) en la misma fila, y seleccione **Eliminar**.

![Paso 3](./img/step-3.png)

**Step 4:** Aparecerá un diálogo de confirmación. Haga clic en **Eliminar** para eliminar permanentemente el parche.

![Paso 4](./img/step-4.png)

:::caution
Eliminar un parche lo eliminará de todas las tiendas donde se le asigna activamente. Si el parche está en la lista de parches de una tienda, se eliminará de esa lista. Esta acción no puede ser aprobada.
:::
