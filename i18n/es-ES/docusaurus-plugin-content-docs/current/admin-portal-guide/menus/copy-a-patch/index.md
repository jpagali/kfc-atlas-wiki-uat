# Copiar un parche

## Qué cubre esta guía

Duplica un parche existente para usar como punto de partida para un conjunto similar de overrides.

## Pasos

**Step 1:** Navegue a la sección **Menus** usando el menú de navegación de la mano izquierda.

**Step 2:** Haga clic en la pestaña **Patches** para ver todos los parches.

![Paso 2](./img/step-2.png)

**Step 3:** Busque el parche que desee copiar, haga clic en el menú **action** (tres puntos) en la misma fila, y seleccione **Copiar**.

![Paso 3](./img/step-3.png)

**Step 4:** Actualice el nombre del parche. Por defecto, el sistema lo nombrará “copia de [nombre de parche original]”.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| **Patch Name** | Un nombre descriptivo para este nuevo parche | e.g., “Sydney Q2 Price Override” (copied de “Sydney Q1 Pricing Override”). Cambiar el nombre para reflejar para qué se utilizará este parche. |

Todas las operaciones y elementos del parche original se copian automáticamente.

![Paso 4](./img/step-4.png)

**Step 5:** Revise las operaciones copiadas para asegurar que coincidan con sus necesidades. Puede editar, reordenar, añadir o eliminar operaciones antes de guardar.

**Step 6:** Haga clic en **Crear** para guardar el parche copiado.

![Paso 5](./img/step-5.png)

![Paso 6](./img/step-6.png)

:::note
El parche copiado es independiente del original. Los cambios a un parche no afectarán al otro. Edite el parche copiado después de la creación si necesita modificar las operaciones o elementos.
:::
