# Menús

Esta sección cubre todas las tareas de gestión de menús en el Portal de Admin de Byte Commerce, incluyendo crear menús, organizarlos en categorías, configurar overrides con parches y gestionar estructuras de precios.

## Menús

- [Crear un menú](/docs/admin-portal-guide/menus/create-a-menu/)— Construir una nueva estructura de menús para tiendas y canales
- [Editar un menú](/docs/admin-portal-guide/menus/edit-a-menu/)— Actualizar la configuración del menú existente
- [Copiar un menú](/docs/admin-portal-guide/menus/copy-a-menu/)— Duplicar un menú como punto de partida
- [Asignar un menú](/docs/admin-portal-guide/menus/assign-a-menu/)- Vincular un menú a tiendas y canales específicos
- [Publicar un menú](/docs/admin-portal-guide/menus/publish-a-menu/)— Empuja un menú a los canales de pedidos en vivo
- [Eliminar un menú](/docs/admin-portal-guide/menus/delete-a-menu/)— Quitar un menú del sistema

## Categorías

- [Crear una categoría](/docs/admin-portal-guide/menus/create-a-category/)— Añadir una etiqueta de agrupación para organizar productos dentro de un menú
- [Editar una categoría](/docs/admin-portal-guide/menus/edit-a-category/)— Actualizar el nombre o la configuración de una categoría
- [Copiar una categoría](/docs/admin-portal-guide/menus/copy-a-category/)— Duplicar una categoría con su configuración
- [Suprímase una categoría](/docs/admin-portal-guide/menus/delete-a-category/)— Quitar una categoría de un menú
- [Añadir Metafields a una categoría](/docs/admin-portal-guide/menus/add-metafields-to-a-category/)— Adjuntar datos personalizados a una categoría para las integraciones

## Patrones

- [Crear un parche](/docs/admin-portal-guide/menus/create-a-patch/)— Construir una anulación dirigida para los cambios de menú
- [Editar un parche](/docs/admin-portal-guide/menus/edit-a-patch/)— Actualizar las operaciones o los elementos de un parche
- [Copiar un parche](/docs/admin-portal-guide/menus/copy-a-patch/)— Duplicar un parche como punto de partida
- [Eliminar un parche](/docs/admin-portal-guide/menus/delete-a-patch/)- Quitar un parche del sistema
- [Asignar un parche (Añadir a la lista de parches)](/docs/admin-portal-guide/menus/assign-a-patch-add-to-patch-list/)— Añadir un parche a la lista activa de una tienda
- [Assign a Patch (Replace Existing List)](/docs/admin-portal-guide/menus/assign-a-patch-replace-existing-list/)— Reemplazar toda la lista de parches de una tienda

## Precios nombrados

- [Crear un precio llamado](/docs/admin-portal-guide/menus/create-a-named-price/)— Define una etiqueta de precio reutilizable para múltiples productos
- [Precio Editado](/docs/admin-portal-guide/menus/edit-named-price/)— Actualizar el valor de un precio nombrado existente
- [Precio suprimido](/docs/admin-portal-guide/menus/delete-named-price/)- Quitar un precio llamado del sistema

## Dominio relacionado

- [Salidas del día](/docs/admin-portal-guide/dayparts/)— Manage org-level dayparts, store-level overrides, time-based pricing, availability, and category visibility
