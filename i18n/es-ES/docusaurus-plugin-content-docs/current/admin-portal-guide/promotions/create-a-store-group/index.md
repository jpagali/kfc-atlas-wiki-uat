# Crear un grupo de tiendas

## Qué cubre esta guía

Crea una agrupación nombrada de tiendas que comparten menú, promoción o configuraciones tributarias, una estructura fundamental para gestionar múltiples ubicaciones a escala. (Esta guía crea un grupo de tiendas de la sección Promociones. También puede crear uno de la sección Grupos de Tienda en la navegación principal.)

## Pasos

**Step 1:** Navegue a la sección **Promociones** utilizando el menú de navegación de la mano izquierda.

**Step 2:** Haga clic en la pestaña ** Grupos de tareas**.

![Paso 2](./img/step-2.png)

**Step 3:** Haga clic en el botón **+ Crear nuevo grupo de tiendas**.

![Paso 3](./img/step-3.png)

**Step 4:** Introduzca los detalles del grupo de la tienda. Se requieren campos marcados con *.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| **Store Group Name** | Un nombre descriptivo para este grupo | e.g., “NSW Franchise Group”, “Corporate Owned Stores”, “Breakfast Pilot 2024”. Debería diferenciarse claramente de otros grupos. |
| **Store Group Tags** | Etiquetas opcionales para filtrar y reportar | por ejemplo, “pilot”, “corporate”, “franchise”. Puedes introducir múltiples etiquetas. |

![Paso 4](./img/step-4.png)

**Step 5:** Seleccione las tiendas para añadir a este grupo. Utilice la tabla proporcionada para encontrar y seleccionar tiendas:

- Puede **filter por número de tienda, nombre de tienda o código de franquicia** para encontrar rápidamente tiendas específicas.
- **Toggle the switch** next to each store name to add it to the group.
- Una tienda puede pertenecer a múltiples grupos de tiendas.

![Paso 5](./img/step-5.png)

**Step 6:** Revise el resumen que muestra todas las tiendas seleccionadas y detalles del grupo. Haga clic en **Crear** para guardar el grupo de la tienda.

![Paso 6](./img/step-6.png)

:::note
Una tienda puede pertenecer a múltiples grupos de tiendas. Las promociones, reglas fiscales y menús se gestionan a nivel de grupo de la tienda y se aplican a todas las tiendas miembros.
:::
