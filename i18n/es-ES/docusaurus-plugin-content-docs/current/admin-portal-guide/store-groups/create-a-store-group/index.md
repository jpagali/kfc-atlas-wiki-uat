# Crear un grupo de tiendas

## Qué cubre esta guía

Crea una agrupación nombrada de tiendas que comparten menú, promoción o configuraciones tributarias, una estructura fundamental para gestionar múltiples ubicaciones a escala.

## Pasos

**Step 1:** Navegue a la sección **Store Groups** utilizando el menú de navegación de la mano izquierda.

**Step 2:** Haga clic en el botón **+ Crear nuevo grupo de tiendas**.

![Paso 2](./img/step-2.png)

**Step 3:** Introduzca los detalles del grupo de la tienda. Se requieren campos marcados con *.

| Campo | Qué entrar | Notas |
|-------|--------------|-------|
| **Store Group Name** | Un nombre descriptivo para este grupo | Por ejemplo, "Grupo de franquicias de NV", "Corporate Owned Stores", "Breakfast Pilot 2024". Debería diferenciarse claramente de otros grupos. |
| **Store Group Tags** | Etiquetas opcionales para filtrar y reportar | por ejemplo, "pilot", "corporate", "franchise". Útil para organizar y buscar múltiples grupos de tiendas. |

![Paso 3](./img/step-3.png)

**Step 4:** Seleccione tiendas para añadir a este grupo utilizando la tabla de tiendas. Puedes:

- **Filter by Store Number, Store Name o Franchise Code** para encontrar rápidamente tiendas específicas
- **Toggle the switch** next to each store name to add or remove stores from the group
- Ver qué tiendas ya están seleccionadas buscando los interruptores comprobados

![Paso 4](./img/step-4.png)

**Step 5:** Revisa el resumen de las tiendas que has seleccionado y otros detalles. Haga clic en **Crear** para guardar el grupo de la tienda.

![Paso 5](./img/step-5.png)

![Paso 6](./img/step-6.png)

:::note
Una tienda puede pertenecer a múltiples grupos de tiendas simultáneamente. Las promociones, reglas fiscales y menús se gestionan a nivel de grupo de la tienda y se aplican a todas las tiendas miembros.
:::
