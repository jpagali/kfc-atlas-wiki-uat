# View Stores in a Store Group

## Qué cubre esta guía

Lists all stores currently belonging to a store group for visibility and verification purposes.

## Pasos

**Step 1:** Navegue a la sección **Store Groups** utilizando el menú de navegación de la mano izquierda.

**Step 2:** Examine la tabla de Grupos de Tienda para encontrar el grupo que desee ver, o utilice la barra de búsqueda para localizarla. Haga clic en el nombre del grupo **store** para ver sus detalles.

![Paso 2](./img/step-2.png)

**Step 3:** Se abrirá una vista detallada:

- **Store Group Name** — El nombre del grupo seleccionado
- **Store Group Tags** — Cualquier etiqueta asignada a este grupo
- **Stores Tabla** - Una lista completa de todas las tiendas de este grupo

La tabla muestra:
- **Número de archivo**: identificador único para cada tienda
- **Store Name** — Name of the store location
- ** Código de Franquicia**: identificador de franquicia si es aplicable
- Reglas de texto** Si este grupo tiene reglas fiscales asociadas

![Paso 3](./img/step-3.png)

:::note
La tabla de tiendas lista todas las tiendas que pertenecen actualmente a este grupo de tiendas. Para modificar la membresía de la tienda, vea[Editar un grupo de tiendas](/docs/admin-portal-guide/store-groups/edit-a-store-group/).
:::
