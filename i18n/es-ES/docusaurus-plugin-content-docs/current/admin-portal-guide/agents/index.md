# Agentes

Esta sección cubre todas las tareas para crear y gestionar agentes de usuario (miembros de personal con acceso a portal) en el Portal de Admin de Comercio Byte.

## Temas

- [Crear un agente](/docs/admin-portal-guide/agents/create-an-agent/)— Establecer una nueva cuenta de usuario con funciones y permisos específicos.
- [Editar un agente](/docs/admin-portal-guide/agents/edit-an-agent/)— Actualizar los detalles, roles o permisos de la cuenta de un agente existente.
