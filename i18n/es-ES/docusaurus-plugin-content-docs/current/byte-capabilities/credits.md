# Créditos

Byte Capabilities fue investigado y documentado por:

[**Varima Henry** y Wesley Weidemann** en Confluencia](https://yumbrands.atlassian.net/wiki/spaces/ITBFR/overview?homepageId=3880354415)para ver su trabajo.

Este trabajo se construyó a partir de 11 documentos fuente que abarcan las especificaciones funcionales y técnicas de Byte Helium, Byte Commerce Backend y Byte Portal, destiladas en un formato accesible al mercado, lanzamiento y equipos multifuncionales.
