# KFC Deeplink ← Enlace Universal

## Lo que hace

La función KFC Universal Link permite que cualquier URL KFC compatible se abra directamente dentro de la aplicación móvil de KFC, llevando al usuario directamente a la pantalla de aplicación relevante en lugar de una página del navegador.

Cuando la **app está instalada**, los usuarios de la ruta de enlaces soportados directamente a la pantalla de juego (por ejemplo, una página de producto, un promo o un aterrizaje de restaurante). Cuando la **app no se instala**, el mismo enlace se remonta con gracia a la experiencia web.

El resultado es un camino inigualable y sin fricción desde cualquier punto de contacto: un email de marketing, un código QR in-restaurant, un anuncio social, una notificación de empuje — directamente al lugar correcto en la aplicación.

## Por qué importa

Sin Enlaces Universales, cada enlace externo al contenido de KFC se abre en un navegador, incluso cuando el cliente tiene la aplicación instalada. Eso crea fricción innecesaria y pierde oportunidades de conversión.

Enlaces Universales cierran esa brecha. La URL sigue igual. El destino se adapta a lo que el cliente ha instalado.

## Ámbito

| Zona | Detalle |
|---|---|
| **Matrimonios reconocidos** | URLs de propiedad de KFC configuradas por mercado |
| *Routing* | Base de ruta — cada ruta URL mapa a un destino de aplicación |
| **Aplicación instalada** | El usuario se lleva directamente a la pantalla de aplicación |
| **App no instalado** | El usuario se envía a la experiencia web equivalente |
| **Platforms** | iOS, Android |

## Puntos de entrada

Los enlaces universales se activan típicamente de:

- Correos electrónicos de marketing y campañas SMS
- Códigos QR (in-restaurante, embalaje, exterior)
- Anuncios de redes sociales y puestos orgánicos
- Notificaciones push
- Corrientes de despido Web a App
