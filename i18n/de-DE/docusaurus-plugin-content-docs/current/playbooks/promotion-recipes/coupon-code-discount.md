# Gutscheincode Rabatt

## Ziel

$10 off für Kunden, die den Code eingeben`SAVE10`.

## Grundlegende Informationen

| Feld | Wert |
|---|---|
| ** Automatische** | `No` |
| **Erklärungscode hinzugefügt** | `SAVE10` |

## Anforderungen

| Gebiet | Feld | Wert |
|---|---|---|
| ** Option** | **Max Gesamteinsatz pro Kunde** | `1`um eine Nutzung pro Kunde zu begrenzen |
| ** Option** | **Max Anzahl der Zeiten, die eine Promotion verwendet werden kann** | Gesamtkappe für Rücknahmen |

## Auswirkungen

| Gebiet | Feld | Wert |
|---|---|---|
| **Orderstufe** | ** Druckeffekttyp** | `Amount Off` |
| **Orderstufe** | ** Betrag ab** | `$10.00` |
