# Täglicher Deal (nach Kunden pro Tag)

## Ziel

Ein kostenloses Getränk mit jeder Mahlzeit, limitiert auf einen pro Kunde pro Tag.

## Anforderungen

| Gebiet | Feld | Wert |
|---|---|---|
| **Ihre Präsenz** | **Minimum Menge** | `1` |
| **Ihre Präsenz** | **Qualifikationsartikel* | `Any Meal product` |
| **Max Gesamtbenutzer pro Tag** | **Max Gesamtbenutzer pro Tag** | `1` |

## Auswirkungen

| Gebiet | Feld | Wert |
|---|---|---|
| **Siehe Ebene** | ** Druckeffekttyp** | `Fixed Price` |
| **Siehe Ebene** | **Fixed Price* | `$0.00` |
| **Siehe Ebene** | **Qualifikationsartikel* | `The drink to be made free` |
