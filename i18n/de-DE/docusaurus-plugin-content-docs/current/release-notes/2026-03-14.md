# Veröffentlichung 2026.03.14

Veröffentlicht auf **März 14, 2026**.

Diese Veröffentlichung konzentrierte sich auf die Erweiterung der Beitrags-Workflows und die Wiederherstellung der Stabilität in der gemeinsamen Website-Navigation. Die Headline-Ergänzung ist ein neues, geführtes Beitragsstudio zur Einreichung von Atlas Wiki-Inhalten, gepaart mit einem Rücksetzen der lokalen Navigation auf das sicherere eingebaute Docusaurus-Verhalten.

## Highlights

- ** Ein neues Beitragsstudio ist jetzt verfügbar**
Contributors können Atlas Wiki-Inhalte in einem Markeneditor erstellen, den Artikel vorsehen, Markdown herunterladen und eine Veröffentlichungs-Request-E-Mail öffnen, ohne Repository benötigen oder Zugriff auf Anfrage ziehen zu müssen.

- **Contribute CTAs sind jetzt im Hauptstandort Chrom* sichtbar
Ein neuer`Contribute`CTA erscheint in der navbar, und ein fußseitiges Callout führt den Beitragsfluss als`The 12th Secret`.

- **lokalisierte Abdeckung beinhaltet jetzt das Beitragsstudio**
Das Studio-Erlebnis hat nun lokalisierte Etiketten und Kopien, so dass der neue Workflow in beiden Standorten verfügbar ist.

- **Lokales Schalten wurde stabilisiert, indem man auf den Standard des Rahmens zurückkehrt*
Nach benutzerdefinierten Locale-Switcher-Experimenten erwies sich spröde, wurde die Navigation auf Docusaurus' eingebauten Locale Dropdown zurückgesetzt, so dass das Sprachverhalten von der Plattform statt der benutzerdefinierten clientseitigen Pfadlogik behandelt wird.

- **Die neue Autorisierungserfahrung wurde leicht gehalten*
Der Beitragsfluss wurde implementiert, ohne eine schwere Editor-Abhängigkeit oder einen Roh-Media-Upload-Backend einzuführen.

## Was verändert

### Beitrag Workflow

- Neu hinzugefügt`/contribute/`Eingabeseite mit:
  - geführte Metadatenerfassung
  - Vorlagenstarter
  - eine WYSIWYG Autoroberfläche
  - kollapsible Vorschau
  - Ausfuhr von Markisen
  - Veröffentlichungs-Request E-Mail-Entwurf
- Hinzufügen von verlinkten Bildvorschau-Verhalten mit einer Anhang-Erinnerung, wenn eine Bild-URL nicht zum Rendern
- Zusätzlicher lokaler Kopierschutz für das Beitragsstudio
- Zusätzlich prominente Beitrag Eintrittspunkte in der Navbar und Fußzeile

### Site Experience

- Hinzufügen eines Footer Callout markiert`The 12th Secret`
- Styled das Beitragsstudio als dunkle KFC-Branded Publishing-Oberfläche
- Die Hauptlese Erfahrung leicht gemacht, indem Beitragstooling zu seiner eigenen Seite isoliert

### Lokalisierung und Stabilität

- Entfernen der benutzerdefinierten Locale-Switcher-Implementierung, nachdem es sich als unzuverlässig erwiesen
- Restaurierte Docusaurus' integrierte lokale Dropdown für eine sicherere Sprachnavigation
- Wiederherstellen der statischen Leistung nach dem Locale-Navigation-Reset

### Erfahrung mit dem Entwickler

- Kept die Implementierung leicht durch Vermeidung eines schweren Editor-Pakets
- Erhalten Sie den bestehenden Docs-Building-Fluss, während Sie die neue Beitragsroute hinzufügen
- Weiterführend mit einem einzigen Port gebaut Vorschau als der sicherste Weg, um Cross-locale-Browsing lokal zu validieren

## In Plain Englisch

Diese Veröffentlichung fügte einen polierten neuen Weg für Menschen, um Inhalte zum Atlas Wiki zu beitragen, ohne git zu berühren, während auch weg von risikobenutzerdefinierten Locale-routing-Verhalten zugunsten der sicheren Standard-Navigation des Rahmens. Das Ergebnis ist eine angenehmere Beitragserfahrung mit einer stabileren Lokalisierungsbasis.
