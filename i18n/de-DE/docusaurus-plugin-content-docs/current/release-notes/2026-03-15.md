# Veröffentlichung 2026.03.15

Veröffentlicht auf **März 15, 2026**.

Diese Veröffentlichung hat eine eigene`What's New`Erfahrung für die Atlas Wiki und verbunden es mit der Hauptseite Chrom, so dass Start Kommunikation ist einfacher zu entdecken. Die Arbeit sorgte auch dafür, dass sich die neuen Einstiegspunkte und Seitenerfahrung sowohl für englische als auch für lokalisierte Nutzer korrekt verhalten.

## Highlights

- **Ein neues`What's New`Seite ist jetzt live*
Die Atlas Wiki hat nun eine eigene Startseite, auf der Oberflächen Releases, neue Features, neue Verbesserungen und bevorstehende Arbeiten in einem strukturierteren Produktmarketing-Format vorgestellt wurden.

- **`What's New`ist jetzt Teil der Hauptnavigation*
Ein neuer Top-Nav-Eintrag wurde neben`Playbooks`, während`About`wurde von der Top-Navigation entfernt, aber in der Fußzeile zur Verfügung gehalten.

- **Der Footer enthält jetzt eine Twin Launch Card**
Die bestehenden`The 12th Secret`Footer Callout ist jetzt mit einem`What's New`karte so starten Updates haben einen starken Desktop-Eintrittspunkt.

- **Mobilverhalten bleibt absichtlich leicht*
Auf mobil, die`What's New`Seite bleibt durch die Hamburger-Navigation ohne zusätzliche Fußabschneider entdecken.

- **lokalisierte Routing- und Seiteninhalte werden unterstützt**
Die neue Seite und Release-Note-Abdeckung wurden lokalisiert, so dass die Funktion funktioniert in beiden`en-US`und`retired locale`.

## Was verändert

### Navigation und Entry Points

- hinzugefügt`What's New`Artikel zum Top navbar
- Entfernt`About`von oben navbar
- Kept`About Atlas Wiki`vom Fuß erreichbar
- Hinzugefügt einen Desktop-Fußspieler Twin-Card-Eintragspunkt für`What's New`
- Kept mobiler Zugriff auf die Hamburger-Navigation

### Was ist neue Erfahrung?

- Neu hinzugefügt`/whats-new`Seite in der App
- Styled the page as a dark KFC-branded launch hub
- Strukturiert die Seite um:
  - Roadmap Snapshot
  - Einführung
  - neue Features
  - Neuere Verbesserungen
  - bald kommen
- verstärkte visuelle Behandlung für den vorgestellten Startbereich

### Lokalisierung

- Hinzugefügt lokalisierte navbar Kopie für`What's New`
- Hinzugefügt lokalisierte Seiteninhalt für`/retired locale/whats-new`
- Lokalisierte Release-Note-Berichterstattung hinzugefügt, so dass die Launch-Kommunikation in beiden Lokalen navigierbar bleibt
- Behoben eines Atlas-Chat-Routing-Bugs, der lokalisierte Benutzer auf eine Seite-nicht-gefundene Route senden könnte, wenn Sie verknüpfte Punkte aus den Chat-Vorschlägen öffnen
- Aktualisiert den Atlas Peek Prototyp, so dass beide Shells einen In-Context halten`language`toggle: die mobile App Shell nutzt die obere Symbolleiste, während die simulierte Web Shell den Schalter im Web nav zeigt

### Layout Polnisch

- Einstellbarer Fußabstand, so dass das neue Twin-Card-Layout die Fußzeilen nicht übermäßig einschränkt
- Verbesserte lokale Footer Lesbarkeit durch Link Text und CTA Kopie mehr Atemraum

## In Plain Englisch

Diese Veröffentlichung verwandelte sich in eine reale Produktoberfläche anstelle eines versteckten Changelogs. Menschen können jetzt finden`What's New`von der Hauptnavigation und Fußzeile auf dem Desktop, immer noch erreichen es sauber auf dem Handy, und verwenden Sie es in Englisch und lokalisiert, ohne das Lokalisierungsverhalten zu brechen.
