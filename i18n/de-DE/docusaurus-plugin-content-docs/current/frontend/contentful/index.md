# Inhaltlich

Inhaltlich ist das Content-Management-System (CMS), das die redaktionelle Schicht des KFC Atlas Web-Erlebnisses betreibt. Es gibt Marktteams – Content-Autor, Menü-Autoren und Übersetzer – die Fähigkeit, Front-End-Inhalte zu verwalten, ohne dass eine Produkt-Bereitstellung oder technische Unterstützung erforderlich ist.

Denken Sie daran als Bedienfeld für das, was Kunden * siehe* über KFC-digitale Touchpoints: welche Produkte angezeigt werden, wie Werbeaktionen erscheinen, welche Kopie angezeigt wird und wie Seiten strukturiert werden - sowohl auf dem **web** als auch auf der **mobile App***.

## Wer benutzt es

| Laufsohle | Was sie tun in Zufriedenheit |
|---|---|
| **Content Creator** | Veröffentlichen und Updates Homepage-Banner, Werbeinhalte und Seitenkopie |
| **Menu Autor** | Verwaltet Produktlisten, Kategorien, Pakete und Angebote |
| **Übersetzer*** | Lokalisierte Inhalte für jede unterstützte Sprache in einem Markt verwalten |

## Wie Veröffentlichung funktioniert

Contentful verwendet einen strukturierten Publishing-Workflow:

1. Inhalt ist **drafted* von einem Content-Autor oder Menüautor
2. Es ist **reviews** und intern genehmigt
3. Es ist **veröffentlicht über Launch** — Contentfuls geplantes Publishing Tool — das koordiniert, wenn Inhalte gleichzeitig über mehrere Einträge hinwegleben.

Dies ermöglicht es den Märkten, Kampagnen im Voraus vorzubereiten und sie zu einer genauen Zeit zu veröffentlichen, ohne im Moment von go-live online sein zu müssen.

## Unterstützung der Plattform

Contentful-managed Content-Oberflächen über Web und Mobile App. Die gleichen Inhaltseinträge – Produktbeschreibungen, Werbekopien, Banner und Kategorie-Etiketten – werden von beiden Plattformen über die Atlas API-Schicht verbraucht.

| Plattform | Unterstützung |
|---|---|
| **Web** | Vollständige Unterstützung — alle Inhaltstypen gelten |
| **iOS / Android* | ✅ Unterstützt – Inhalt wird über die gleiche API an die App geliefert; einige Layout-spezifische Felder (z.B. Banner-Aspektverhältnisse) können je Plattform unterschiedlich sein |

> **Anmerkung:** Einige App-only-Konfigurationen (wie Push-Benachrichtigungskopie oder App-Store-Metadaten) werden außerhalb von Contentful verwaltet. Dieser Leitfaden deckt Inhalte ab, die innerhalb der KFC-App erscheinen.
