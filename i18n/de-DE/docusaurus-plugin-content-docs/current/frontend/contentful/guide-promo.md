# Leitfaden: Inhalte fördern

Werbeinhalte werden verwendet, um Gutschein- und Werbeinformationen an Kunden im Warenkorb und beim Checkout anzuzeigen. Es kann auch eine Promotion als Menükarte oberflächen - so dass es im Menü als wäre es ein regelmäßiges Produkt.

:::tip
Eine Promotion ** muss zuerst im Byte Admin Portal erstellt werden*, bevor sie als Promotion Content-Eintrag in Contentful konfiguriert werden kann.
:::

---

## Wo Sie es im Frontend sehen

**Basket / Checkout* — Wenn ein Kunde einen qualifizierten Artikel in seinem Warenkorb hat, erscheint eine Werbe-Fliese, die den Anzeigenamen der Promotion, Beschreibung und jeden Anforderungstext (z.B. "Spend $20 to entlock this Deal") zeigt.

**Menu-Karte** — Promotionen können optional als Karte direkt im Menüraster erscheinen – mit einem Bild, Titel und Beschreibung – so können Kunden sie entdecken und in ihren Warenkorb hinzufügen, ohne einen Gutscheincode zu benötigen.

---

## Inhaltlich konfigurieren

1. In Contentful, navigieren Sie zum **Promotion** Inhaltstyp und öffnen Sie es
2. Klicken Sie auf **Auswahl von Yum**, um eine Liste von Aktionen zu sehen, die in Byte Admin Portal erstellt wurden
3. Wählen Sie die Aktion aus, die Sie konfigurieren möchten

**Configure Display Name* — Der Name, der dem Kunden auf der UI angezeigt wird (z.B. "Zinger Box Deal"). Setzen Sie das pro Einheimischer.

**Configure Beschreibung** — Zusätzlicher Kontext, der neben der Promotion angezeigt wird (z.B. "Includes a Zinger Burger, regular fries, and a drink"). Set per locale.

**Anforderungstext konfigurieren* — Alle Bedingungen, die der Kunde erfüllen muss, um die Promotion zu erhalten (z.B. "Eine große Mahlzeit zu qualifizieren"). Set per locale.

**Konfigurieren Sie Menükarte Bild* — Fügen Sie ein Bild hinzu, um die Förderung als vertikale Menükarte im Menüraster zu beleuchten. Hochladen oder aus vorhandenen Medien auswählen.

5. Klicken Sie auf **Publish** – oder benutzen Sie den Dropdown, um die Aktion zu einem bestimmten Datum und Uhrzeit live zu planen

---

## Anwendungsfälle

**Zur Anzeige eines neuen Gutscheins im Checkout** Konfigurieren Sie eine Werbekopie und Bildsprache, so dass Kunden sehen es deutlich im Warenkorb, wenn sie berechtigt sind, es zu verwenden.

**Eine menü-visible Vereinbarung festlegen* Fügen Sie ein Menükartenbild hinzu, um die Promotion als ein browsables Element im Menü zu oberflächen, so dass es ohne Promo-Code entdeckt werden kann.

** Zeitbegrenzte Kampagnen* Verwenden Sie die geplante Veröffentlichung, um die Inhalte einer Promotion in Synchronisierung mit einem Marketing-Kampagne-Fenster zu aktivieren und zu deaktivieren.

**Lokalisierte Promotionen** — Setzen Sie verschiedene Anzeigenamen und Beschreibungen pro Lokal ein, um die Werbenachricht auf die Sprache und den Ton des jeweiligen Marktes zu richten.
