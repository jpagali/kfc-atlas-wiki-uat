# Leitfaden: Homepage Inhalt

Die Homepage ist die erste Seite, auf der Kunden beim Besuch der KFC-Website landen. Es stellt den Ton für die gesamte Erfahrung fest - so dass es visuell auf dem neuesten Stand und lokal relevant ist Schlüssel.

---

## Wo Sie es im Frontend sehen

Die Homepage Oberflächen in zwei Hauptbereichen:

**Hero Banner** — das Vollbreiten-Banner oben auf der Homepage. Dies ist die prominenteste Immobilien auf der Website und wird in der Regel für Kampagnen-Starts, saisonale Werbeaktionen oder Produkt-Strahler verwendet. Das Banner ist **locale-spezifisch*, so dass jeder Markt ein anderes Bild zeigen kann.

**Homepage Titel** — die Rubrik auf der Homepage, auch editierbar per locale. Dies wird oft verwendet, um Kunden mit marktspezifischem Exemplar zu begrüßen.

:::tip Leitfaden für visuelle Merchandising
Homepage-Banner sollten **vereinfachte Nachrichten** haben – speichern Sie das komplette Deal-Konstrukt für die Deals-Seite. Konzentrieren Sie sich auf 1–2 Kernelemente, begrenzen Sie den Bildtext und stellen Sie sicher, dass der Held nicht vom Hauptziel der Homepage ablenkt: Lokalisierung.
:::

Das Bannerbild wird mit einer einzigen Auflösung von **3840×606px (PNG)* hochgeladen. Dabei definiert ein Mittel 2:1-Verhältnis die "sichere Zone":

- **Rechtsseite** — empfohlen für Marketingtext
- **Linksseite** — durch das Lokalisierungs-Widget auf mittleren und großen Breakpoints abgedeckt
- **Grauer Ausblutungsbereich** — Ernten oder Erweiterungen je nach Bildschirmbreite; nur zur Unterstützung der Bilder

## Inhaltlich konfigurieren

### Aktualisierung des Homepage Titels

1. In Contentful, Suche nach der **Homepage** Inhaltsart
2. Homepage-Eintrag öffnen
3. Suchen Sie das Feld **Title - English** und aktualisieren Sie die Kopie
4. Änderungen werden in der Vorschau sofort reflektiert — Doppel-Check vor dem Verlag

### Aktualisieren des Hero Banners

1. Öffnen Sie den Homepage-Eintrag und scrollen Sie auf **Background Images**
2. Klicken Sie auf den **Hero**-Eintrag für die lokale, die Sie aktualisieren möchten – das öffnet ein neues Bearbeitungsfenster
3. Sie sehen **5 Bildformat-Slots*: Extra Small, Small, Medium, Large, Extra Large
4. Klicken Sie auf das Bild oder die Schaltfläche Einstellungen, um zu bearbeiten

** Um den aktuellen Vermögenswert zu ersetzen (empfohlen):**

1. Klicken Sie auf den Bildbereich, um den **Header Banner Asset** Editor zu öffnen
2. Klicken Sie auf **Open File Selector**, um eine neue Datei hochzuladen
3. Contentful unterstützt JPEG, PNG, WebP und mehr
4. Hochladen oder ziehen Sie Ihr Bild
5. Klicken Sie auf **Veröffentlichen Sie Änderungen*, um es live zu machen

Wiederholen Sie diesen Prozess für jeden Einheimischen, der ein anderes Banner benötigt.

## Anwendungsfälle

**Seasonal Kampagnenstart* — Laden Sie ein neues Helden-Banner hoch, wenn eine große Förderung (z.B. eine neue Produkt- oder Urlaubskampagne) beginnt. Verwenden Sie die Start-App, um es zu planen, um live um Mitternacht zu gehen, ohne dass jemand online in diesem Moment benötigen.

**Market Lokalisierung** — Jeder Einheimische erhält seinen eigenen Titel und Banner, so dass ein australischer Markt eine Sommerförderung zeigen kann, während ein britischer Markt einen Winter zeigt — gleichzeitig.

**A/B-Bannertest** — Swap-Banner zwischen Einheimischen, um Messaging-Ansätze ohne Einsatz zu testen.

**Keeping content fresh** — Regelmäßig Drehen des Homepage-Helfers verbessert das Kundenengagement und sorgt dafür, dass sich die Website nicht zwischen großen Kampagnen fühlt.
