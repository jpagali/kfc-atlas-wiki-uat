# Kundenbindungserhebungen

Kundenengagement-Umfragen helfen KFC, strukturiertes Gästefeedback nach einer Bestellung, einem Besuch oder einer Support-Interaktion zu erfassen. In Atlas wird diese Fähigkeit typischerweise für **KFC Listen**-Stil-Programme verwendet, die Zufriedenheit messen, Service-Probleme identifizieren und Oberflächeneinsichten für Marktteams.

## Was diese Fähigkeit tut

Kundenbefragungen lassen Märkte:

- Laden Sie die Kunden ein, das Feedback nach dem Kauf zu teilen
- Messen Sie die Zufriedenheit über Kanäle, Geschäfte und Kampagnen
- Routenerhebungsreaktionen in eine Berichtsplattform zur Analyse
- Trigger Follow-up-Aktionen für Ablenker, Beschwerden oder Recovery-Workflows

## Wo es in der Customer Journey erscheint

Umfrage-Einladungen können je nach Markteinstellung an verschiedenen Touchpoints platziert werden:

| Touchpoint | Beispiel Verhalten |
|---|---|
| Bestellbestätigung | Prompt nach dem Checkout oder auf dem Bestätigungsbildschirm |
| E-Mail senden | Fügen Sie eine Umfrage CTA in der Empfangs-E-Mail ein |
| Push-Benachrichtigung | Senden Sie eine Folgeeinladung nach Erfüllung |
| Sicherheiten im Lager | QR Code oder Empfangs-URL zur Umfrage |

## Warum Märkte es nutzen

KFC Listen und ähnliche Programme geben den Betreibern ein direktes Signal von Kunden über Lebensmittelqualität, Geschwindigkeit, Personalfreundlichkeit und Auftragsgenauigkeit. Das Feedback hilft den Märkten:

- Benchmark Store Performance
- Erkennen Sie wiederkehrende Kunden Schmerzpunkte
- Gültige Kampagne oder Produktänderungen
- Prioritäre Service-Recovery für mind-scoring Antworten

## Typisches Betriebsmodell

| Gebiet | Eigentümer |
|---|---|
| Design und Logik der Umfrage | CX / Insights Team |
| Einladung Trigger Regeln | Digitales Produkt oder CRM Team |
| Antwort | Marktgeschäfte / Insights |
| Service Erholung Workflows | Kundenbetreuung oder -speicherung |
