# Agenten

Dieser Abschnitt umfasst alle Aufgaben zum Erstellen und Verwalten von Benutzern (Staff-Mitgliedern mit Portalzugriff) im Byte Commerce Admin Portal.

## Themen

- [Einen Agenten erstellen](/docs/admin-portal-guide/agents/create-an-agent/)— Ein neues Benutzerkonto mit bestimmten Rollen und Berechtigungen einrichten.
- [Einen Agenten bearbeiten](/docs/admin-portal-guide/agents/edit-an-agent/)— Aktualisieren Sie die Kontodetails, Rollen oder Berechtigungen eines bestehenden Agenten.
