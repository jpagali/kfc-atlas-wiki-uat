# Daypart Sichtbarkeit auf Kategorien festlegen

## Zusammenfassung

Dayparts sind weltweit im Byte Portal verfügbar und können genutzt werden, um die Sichtbarkeit der Kategorie über den Tag zu konfigurieren. Mit dieser Funktion wird die Bestellbarkeit von Gegenständen innerhalb einer Kategorie nicht beeinträchtigt - sie bietet lediglich einen Hinweis auf die verbrauchende Schnittstelle (Byte POS, Byte Kiosk, Byte Commerce, etc.) um eine Kategorie außerhalb der konfigurierten Tagesteile zu verstecken oder zu de-emphasieren.

**Screenshot placeholder:** Screenshot zeigen die Kategorie Sichtdiagramm Verknüpfung Organisation Dayparts zu einer Kategorie hier.

## Tageslichtsicht auf Kategorien einstellen

**Step 1:** Klicken Sie auf **Menus** in der Seitenleiste.

**Step 2:** Klicken Sie auf die Registerkarte **Categories**.

**Step 3:** Wählen Sie eine bestimmte Kategorie aus.

**Screenshot placeholder:** Legen Sie Screenshot mit der Menü-Seite mit dem ausgewählten Kategorien-Tab und einer Kategorie-Reihe bereit, hier zu öffnen.

**Step 4:** Scrollen Sie bis zum Abschnitt **Dayparts**.

**Step 5:** Wählen Sie die Tagesparts aus, die Sie möchten, dass diese Kategorie während des Besuchs sichtbar ist. Mehrere Tagesteile können ausgewählt werden. Beachten Sie, dass, wenn keine Dayparts ausgewählt werden, die Kategorie während des gesamten Geschäftstages sichtbar ist.

**Screenshot placeholder:** Screenshot mit der Kategorie Schublade mit dem Dayparts Abschnitt und einem oder mehreren ausgewählten Dayparts hier.

**Step 6:** Klicken Sie auf **Save** im Schubladenformular.

:::tip
Kategorie Sichtbarkeit ändert sich nicht, ob die Artikel innerhalb der Kategorie geordnet sind. Es bietet nur einen Sichthinweis auf die verbrauchende Schnittstelle.
:::
