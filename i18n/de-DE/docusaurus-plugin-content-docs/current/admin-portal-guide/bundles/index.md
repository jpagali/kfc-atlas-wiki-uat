# Bund

Dieser Abschnitt umfasst alle Aufgaben zur Verwaltung von Paketen (Meal Deals und Combos) und ihre konfigurierbaren Entscheidungen im Byte Commerce Admin Portal.

## Themen

### Bund

- [Einen Bundle erstellen](/docs/admin-portal-guide/bundles/create-a-bundle/)— Erstellen Sie einen neuen Kombi- oder Mahlzeiten-Deal, indem Sie Produkte zusammen gruppieren.
- [Einen Bundle bearbeiten](/docs/admin-portal-guide/bundles/edit-a-bundle/)— Aktualisieren Sie den Namen eines bestehenden Bündels, die Beschreibung, die Auswahl oder andere Details.
- [Einen Bundle kopieren](/docs/admin-portal-guide/bundles/copy-a-bundle/)— Duplizieren Sie ein Bündel, um schnell ähnliche Angebote zu erstellen.
- [Metafelder zu einem Bundle hinzufügen](/docs/admin-portal-guide/bundles/add-metafields-to-a-bundle/)— Benutzerdefinierte Metadaten für Integrationen oder Compliance anhängen.
- [Bild hinzufügen zu einem Bundle](/docs/admin-portal-guide/bundles/add-an-image-to-a-bundle/)— Laden Sie ein Anzeigebild für das Bündel auf Bestellkanälen hoch.
- [Löschen Sie einen Bundle](/docs/admin-portal-guide/bundles/delete-a-bundle/)— Entfernen Sie ein Bündel aus dem Katalog.

### Wahlen

- [Eine Auswahl erstellen](/docs/admin-portal-guide/bundles/create-a-choice/)— Fügen Sie einen Auswahlschlitz zu einem Bündel hinzu (z.B. "Choose Your Side").
- [Eine Wahl bearbeiten](/docs/admin-portal-guide/bundles/edit-a-choice/)— Optionen oder Konfiguration einer Wahl aktualisieren.
- [Eine Wahl kopieren](/docs/admin-portal-guide/bundles/copy-a-choice/)— Duplizieren Sie die Wahl, um die Bündel-Setup zu beschleunigen.
- [Löschen einer Wahl](/docs/admin-portal-guide/bundles/delete-a-choice/)— Entfernen Sie die Wahl aus einem Bündel.
