# Löschen Sie einen Bundle

## Was diese Anleitung deckt

Entfernt permanent ein Paket aus dem Katalog und alle Menüs, in denen es zugewiesen wird.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Bundles**.

**Step 2:** Finden Sie das Paket, das Sie löschen möchten, indem Sie durch Bundle Name, Bundle Code, Katalog Tags oder Promo Tags.

**Step 3:** Klicken Sie auf die Schaltfläche **** (Dreipunktmenü) in der gleichen Zeile wie das Paket, dann wählen Sie **Delete***.

![Schritt 2](./img/step-2.png)

**Step 4:** Eine Bestätigungsmodalität erscheint, um die Löschung zu bestätigen. Klicken Sie **Bestätigen Sie*, um das Paket zu löschen, oder klicken Sie außerhalb des Modal oder **Cancel**, um es zu halten.

![Schritt 3](./img/step-3.png)

:::caution
Diese Aktion ist dauerhaft. Das Löschen eines Pakets wird es automatisch aus allen Menüs und Kategorien entfernen, in denen es zugewiesen wird. Sie können diese Aktion nicht rückgängig machen.
:::
