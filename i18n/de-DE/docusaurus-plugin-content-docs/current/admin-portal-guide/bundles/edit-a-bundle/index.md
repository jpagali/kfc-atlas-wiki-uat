# Einen Bundle bearbeiten

## Was diese Anleitung deckt

Aktualisiert den Namen eines bestehenden Bündels, Code, Auswahl, Preisgestaltung oder Beschreibung.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Bundles**.

**Step 2:** Finden Sie das Paket, das Sie bearbeiten möchten. Sie können mit der Suchleiste nach Bundle Name, Bundle Code, Katalog Tags oder Promo Tags suchen.

**Step 3:** Klicken Sie auf die Schaltfläche **** (Dreipunktmenü) in der gleichen Zeile wie das Paket, dann wählen Sie **Bearbeiten***.

![Schritt 2](./img/step-2.png)

**Step 4:** Das gleiche mehrseitige Formular aus der Bündelerstellung öffnet sich. Alle Felder aktualisieren:
- Klicken Sie auf blaue Abschnitt Kopfzeilen, um zwischen Seiten zu springen (Basic Info, Auswahl, Bewertung)
- Klicken Sie auf **Weiter**, um durch Seiten sequentiell zu bewegen

![Schritt 3](./img/step-3.png)

**Step 5:** Nachdem Sie alle Änderungen vorgenommen haben, klicken Sie auf **Save**, um die Updates zu verpflichten.

:::caution
Klicken Sie auf **Cancel** verworfen alle unerwünschten Änderungen.
:::
