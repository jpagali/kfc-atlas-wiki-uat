# Einen Bundle kopieren

## Was diese Anleitung deckt

Dupliziert ein bestehendes Paket, um schnell ähnliche Angebote mit den gleichen Wahlen und Konfiguration zu erstellen.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Bundles**.

**Step 2:** Finden Sie das Paket, das Sie kopieren möchten, indem Sie nach Bundle Name, Bundle Code, Catalog Tags oder Promo Tags suchen.

**Step 3:** Klicken Sie auf die Schaltfläche **** (Drei-Punkt-Menü) in der gleichen Zeile wie das Paket, dann wählen Sie **Copy***.

![Schritt 2](./img/step-2.png)

**Step 4:** Ein Formular öffnet sich, um das neue Paket zu erstellen. Geben Sie einen einzigartigen **Bundle Code* ein (erforderlich).

![Schritt 3](./img/step-3.png)

**Step 5:** Stellen Sie den **Bundle Name* bei Bedarf ein. Standardmäßig wird es "Copy of [Original Name]" sein – Sie können diese auf etwas deskriptiver ändern.

**Step 6:** Überprüfen Sie alle anderen Felder (Beschreibung, Auswahl, etc.). Das kopierte Bündel enthält alle Optionen aus dem Original. Ändern Sie alle Felder nach Bedarf.

**Step 7:** Klicken Sie *****, um das Paket zu kopieren.

:::tip
Das kopierte Bündel erbt alle Optionen und Konfiguration aus dem Original. Sie können diese Details vor oder nach der Erstellung bearbeiten.
:::
