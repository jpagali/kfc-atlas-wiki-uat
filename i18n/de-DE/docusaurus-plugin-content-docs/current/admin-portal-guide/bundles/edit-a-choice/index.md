# Eine Wahl bearbeiten

## Was diese Anleitung deckt

Aktualisiert den Namen, die Produkte, Varianten oder die Mengeneinstellungen von min/max.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Bundles**.

**Step 2:** Klicken Sie auf die Registerkarte **Choices* am oberen Rand des Bildschirms Bundles.

![Schritt 2](./img/step-2.png)

**Step 3:** Finden Sie die Wahl, die Sie bearbeiten möchten. Sie können nach Wahl Name, Wahlcode, Wahl Anzeige Name oder Katalog Tags suchen.

**Step 4:** Klicken Sie auf die Schaltfläche **** (Dreipunktmenü) in der gleichen Zeile wie die Wahl, dann wählen Sie **Bearbeiten***.

![Schritt 3](./img/step-3.png)

**Step 5:** Aktualisieren Sie alle folgenden Felder:
- **Choice Name** oder **Choice Display Name***
- **Min. Quantität** oder **Max.
- **Produkte*: Produkte und deren Varianten hinzufügen oder entfernen

![Schritt 4](./img/step-4.png)

**Step 6:** Klicken Sie auf **Save**, um Ihre Änderungen zu verpflichten.

:::tip
Sie können Produkte und Varianten jederzeit hinzufügen oder entfernen, ohne dass Sie Bündel, die diese Wahl verwenden, beeinflussen.
:::
