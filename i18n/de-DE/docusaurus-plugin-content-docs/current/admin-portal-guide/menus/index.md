# Menüs

Dieser Abschnitt umfasst alle Menüverwaltungsaufgaben im Byte Commerce Admin Portal, einschließlich der Erstellung von Menüs, der Organisation in Kategorien, der Konfigurierung gezielter Überschreitungen mit Patches und der Verwaltung von Preisstrukturen.

## Menüs

- [Menü erstellen](/docs/admin-portal-guide/menus/create-a-menu/)— Erstellen Sie eine neue Menüstruktur für Speicher und Kanäle
- [Menü bearbeiten](/docs/admin-portal-guide/menus/edit-a-menu/)— Aktualisieren der Konfiguration eines vorhandenen Menüs
- [Menü kopieren](/docs/admin-portal-guide/menus/copy-a-menu/)— Duplizieren Sie ein Menü als Ausgangspunkt
- [Ein Menü zuordnen](/docs/admin-portal-guide/menus/assign-a-menu/)— Verlinken Sie ein Menü zu bestimmten Speichern und Kanälen
- [Menü veröffentlichen](/docs/admin-portal-guide/menus/publish-a-menu/)— Schieben Sie ein Menü, um die Bestellkanäle zu leben
- [Löschen eines Menüs](/docs/admin-portal-guide/menus/delete-a-menu/)— Entfernen Sie ein Menü aus dem System

## Kategorien

- [Eine Kategorie erstellen](/docs/admin-portal-guide/menus/create-a-category/)— Fügen Sie ein Gruppierlabel hinzu, um Produkte innerhalb eines Menüs zu organisieren
- [Kategorie bearbeiten](/docs/admin-portal-guide/menus/edit-a-category/)— Name oder Konfiguration einer Kategorie aktualisieren
- [Kategorie kopieren](/docs/admin-portal-guide/menus/copy-a-category/)— Duplizieren einer Kategorie mit ihrer Konfiguration
- [Kategorie löschen](/docs/admin-portal-guide/menus/delete-a-category/)— Entfernen Sie eine Kategorie aus einem Menü
- [Metafields zu einer Kategorie hinzufügen](/docs/admin-portal-guide/menus/add-metafields-to-a-category/)— Benutzerdefinierte Daten an eine Kategorie für Integrationen anhängen

## Aufnäher

- [Einen Patch erstellen](/docs/admin-portal-guide/menus/create-a-patch/)— Erstellen Sie ein gezieltes Überschreiben für Menüänderungen
- [Einen Patch bearbeiten](/docs/admin-portal-guide/menus/edit-a-patch/)— Aktualisieren der Operationen oder Elemente eines Patches
- [Einen Patch kopieren](/docs/admin-portal-guide/menus/copy-a-patch/)— Vervielfältigen eines Patches als Ausgangspunkt
- [Löschen eines Patches](/docs/admin-portal-guide/menus/delete-a-patch/)— Entfernen Sie einen Patch aus dem System
- [Einen Patch zuordnen (Zu Patch hinzufügen)](/docs/admin-portal-guide/menus/assign-a-patch-add-to-patch-list/)— Fügen Sie einen Patch in die aktive Liste eines Speichers hinzu
- [Zuordnen eines Patches (Ersetzen vorhandene Liste)](/docs/admin-portal-guide/menus/assign-a-patch-replace-existing-list/)— Ersetzen Sie die gesamte Patchliste eines Speichers

## Name der Preise

- [Einen Namenspreis erstellen](/docs/admin-portal-guide/menus/create-a-named-price/)— Definieren Sie ein wiederverwendbares Preislabel für mehrere Produkte
- [Bearbeiten Name des Preises](/docs/admin-portal-guide/menus/edit-named-price/)— Aktualisieren des Wertes eines bereits genannten Preises
- [Gelöschter Name Preis](/docs/admin-portal-guide/menus/delete-named-price/)— Entfernen Sie den Namen des Systems

## Verwandte Domains

- [Tagteile](/docs/admin-portal-guide/dayparts/)— Verwalten org-Level Dayparts, Überschreitungen auf Lagerebene, zeitbasierte Preise, Verfügbarkeit und Sichtbarkeit der Kategorie
