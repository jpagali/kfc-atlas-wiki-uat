# Kategorie löschen

## Was diese Anleitung deckt

Entfernt permanent eine Kategorie aus dem System, wenn sie nicht mehr benötigt wird.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü zum Abschnitt **Menus***.

**Step 2:** Klicken Sie auf den Ordner **Kategorien**, um alle Kategorien anzuzeigen.

![Schritt 2](./img/step-2.png)

**Step 3:** Suchen Sie die Kategorie, die Sie löschen möchten, klicken Sie in der gleichen Zeile auf das **Aktionsmenü* (drei Punkte) und wählen Sie **Delete**.

![Schritt 3](./img/step-3.png)

**Step 4:** Ein Bestätigungsdialog wird angezeigt. Klicken Sie auf **Delete**, um die Kategorie dauerhaft zu entfernen.

![Schritt 4](./img/step-4.png)

:::caution
Die Löschung einer Kategorie wird sie aus allen Menüs entfernen, die sie verwenden. Produkte, die zuvor in dieser Kategorie verwendet wurden, werden nicht zugewiesen und können anderen Kategorien zugeordnet werden. Diese Aktion kann nicht rückgängig gemacht werden.
:::
