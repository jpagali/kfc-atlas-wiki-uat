# Löschen eines Patches

## Was diese Anleitung deckt

Entfernt permanent einen Patch aus dem System.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü zum Abschnitt **Menus***.

**Step 2:** Klicken Sie auf die Registerkarte **Patches*, um alle Patches anzuzeigen.

![Schritt 2](./img/step-2.png)

**Step 3:** Finden Sie den Patch, den Sie löschen möchten, klicken Sie in der gleichen Zeile auf das **Aktionsmenü* (drei Punkte) und wählen Sie **Delete***.

![Schritt 3](./img/step-3.png)

**Step 4:** Ein Bestätigungsdialog wird angezeigt. Klicken Sie auf **Delete**, um den Patch dauerhaft zu entfernen.

![Schritt 4](./img/step-4.png)

:::caution
Löschen eines Patches wird es aus allen Geschäften entfernen, wo es aktiv zugewiesen wird. Wenn das Patch in der Patchliste eines Speichers ist, wird es von dieser Liste entfernt. Diese Aktion kann nicht rückgängig gemacht werden.
:::
