# Löschen eines Menüs

## Was diese Anleitung deckt

Entfernt permanent ein Menü aus dem System, wenn es nicht mehr verwendet wird.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü zum Abschnitt **Menus***.

**Step 2:** Finden Sie das Menü, das Sie in der Menüliste löschen möchten, klicken Sie in der gleichen Zeile auf das **Aktionsmenü* (drei Punkte) und wählen Sie **Delete**.

![Schritt 2](./img/step-2.png)

**Step 3:** Ein Bestätigungsdialog wird angezeigt. Klicken Sie auf die **Delete** Schaltfläche, um das Menü dauerhaft zu entfernen.

![Schritt 3](./img/step-3.png)

:::caution
Das Löschen dieses Menüs wird es aus allen zugewiesenen Speichern entfernen. Allerdings werden die live veröffentlichten Menüs in diesen Läden nicht betroffen sein – nur zukünftige Menüeinträge werden beeinflusst. Diese Aktion kann nicht rückgängig gemacht werden.
:::
