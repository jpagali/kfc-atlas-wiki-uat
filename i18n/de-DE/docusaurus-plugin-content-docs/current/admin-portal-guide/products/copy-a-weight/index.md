# Kopieren Sie ein Gewicht

## Was diese Anleitung deckt

Dupliziert einen Gewichtseintrag, um die Rüstzeit bei der Erstellung ähnlicher Gewichtskonfigurationen zu reduzieren.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Weights**.

**Step 3:** Suchen Sie nach dem Gewicht, das Sie kopieren möchten, indem Sie den Gewicht-Namen oder Gewicht-Code in das Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Dreipunkt-Menü neben dem Gewicht, dann wählen Sie **Kopieren**.

![Schritt 2](./img/step-2.png)

**Step 5:** Das Kopierformular erscheint mit den Informationen des Originalgewichts. Aktualisieren Sie die Felder nach Bedarf. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| **Weight Code*** | Einzigartige Kennung für das neue Gewicht | Muss anders sein als das Original (z.B. „WT-LARGE-COPY“) |
| **Weight Name*** | Beschreibt die Portionsgröße | Kann gleich oder angepasst sein |
| **Max Gewicht** | numerischer Maximalgewichtswert | z.B. „500“ (in Gramm oder in Ihrer Einheit) |
| **Standard** | Sich als Standard markieren | Nur ein Gewicht sollte als Standard pro Option markiert werden |

**Step 6:** Wenn Sie fertig sind, alle Informationen hinzuzufügen, klicken Sie auf die **Kreate Gewicht** Taste.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Der **Weight Code** muss einzigartig sein. Sie können den gleichen Code wie das Originalgewicht nicht verwenden.
:::

:::caution
Klicken Sie auf **Cancel** verworfen alle unerwünschten Änderungen.
:::

:::tip
Sie können Gewichte nach Gewicht Name oder Gewicht Code suchen.
:::
