# Eine Option erstellen

## Was diese Anleitung deckt

Erstellt eine Anpassungsgruppe (z.B. „Größe“, „Spice Level“), die mit Produkten verknüpft werden kann, so dass Kunden ihre Bestellung mit Wahlen personalisieren können.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Optionen**.

![Schritt 2](./img/step-2.png)

**Step 3:** Klicken Sie auf die Schaltfläche **+ Neue Option** erstellen.

**Step 4:** Füllen Sie die Optionsdetails. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| **Optionscode*** | Einzigartige Kennung für diese Option | Verwenden Sie Großbuchstaben, Zahlen und Bindestriche (z.B. „OPT-SIZE“) |
| **Optionsname*** | Der Kundenname der Anpassungskategorie | z.B. „Größe“, „Flavor“, „Spice Level“ |

![Schritt 3](./img/step-3.png)

**Step 5:** Klicken Sie auf **Create Option*, um die Option zu speichern.

## Anmerkungen

:::caution
Klicken Sie auf **Cancel** verwerfen alle nicht gespeicherten Informationen.
:::

:::tip
Der **Option Code** muss einzigartig sein und kann nach der Erstellung nicht geändert werden, so wählen Sie es sorgfältig aus.
:::
