# Löschen eines Modifiers

## Was diese Anleitung deckt

Entfernt permanent einen Modifier aus dem System, wenn er nicht mehr benötigt wird.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Modifiers**.

![Schritt 2](./img/step-2.png)

**Step 3:** Suchen Sie nach dem Modifier, den Sie löschen möchten, indem Sie den Namen, den Code oder den Katalog Tag im Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Dreipunkt-Menü neben dem Modifier, dann wählen Sie **Delete**.

**Step 5:** Eine Bestätigungsmodalität zeigt alle Bereiche des Systems, wo dieser Modifier verwendet wird. Überprüfen Sie dies sorgfältig, um sicherzustellen, dass Sie den richtigen Modifier löschen.

**Step 6:** Klicken Sie auf die rote **Delete** Schaltfläche, um den Modifier dauerhaft zu entfernen.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Das Löschen eines Modifiers ist dauerhaft und kann nicht rückgängig gemacht werden. Der Modifier wird von allen Slots und Produkten entfernt, die es verwenden.
:::

:::tip
Sie können Modifikatoren nach Name, Code oder Catalog Tag suchen, um schnell den Artikel zu finden, den Sie löschen möchten.
:::

:::caution
Klicken Sie auf **Cancel**, wenn Sie nicht mit Löschung fortfahren möchten.
:::
