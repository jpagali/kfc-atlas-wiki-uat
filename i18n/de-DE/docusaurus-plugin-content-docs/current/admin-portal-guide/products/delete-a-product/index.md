# Produkt löschen

## Was diese Anleitung deckt

Entfernt permanent ein Produkt aus dem System, wenn es eingestellt oder im Fehler erstellt wurde.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Finden Sie das Produkt, das Sie löschen möchten. Sie können nach Produktname oder Produktcode suchen.

**Step 3:** Klicken Sie auf das Dreipunktmenü neben dem Produktnamen, dann wählen Sie **Delete**.

![Schritt 2](./img/step-2.png)

**Step 4:** Eine Bestätigungsmodalität zeigt alle Bereiche des Systems, wo dieses Produkt verwendet wird. Überprüfen Sie dies sorgfältig, um sicherzustellen, dass Sie das richtige Element löschen.

**Step 5:** Klicken Sie auf die rote **Delete** Schaltfläche, um das Produkt dauerhaft zu löschen.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Die Löschung eines Produkts ist dauerhaft und kann nicht rückgängig gemacht werden. Das Produkt wird aus allen Menüs und zugehörigen Bereichen des Katalogs entfernt.
:::

:::tip
Sie können Produkte nach Produktname oder Produktcode suchen, um schnell den Artikel zu finden, den Sie löschen möchten.
:::

:::caution
Klicken Sie auf **Cancel**, wenn Sie nicht mit Löschung fortfahren möchten.
:::
