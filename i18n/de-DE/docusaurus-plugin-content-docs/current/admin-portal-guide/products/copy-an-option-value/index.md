# Kopieren eines Optionswerts

## Was diese Anleitung deckt

Dupliziert einen Optionswert, um innerhalb einer Optionsgruppe schnell ähnliche Entscheidungen zu erstellen.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Optionswerte**.

**Step 3:** Suchen Sie nach dem Optionswert, den Sie kopieren möchten, indem Sie den Namen, den Code oder den Katalog Tag im Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Dreipunktmenü neben dem Optionswert, dann wählen Sie **Kopieren**.

![Schritt 2](./img/step-2.png)

**Step 5:** Das Kopierformular wird mit den Informationen des ursprünglichen Optionswerts angezeigt. Aktualisieren Sie die Felder nach Bedarf. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| **Optionswertcode*** | Einzigartige Kennung für den neuen Wert | Muss anders sein als das Original (z.B. „SIZE-XL-COPY“) |
| **Optionswert Name*** | Die Auswahl an Kunden | Kann gleich oder angepasst sein |
| ** Name anzeigen** | Kürzeres Etikett für begrenzten Bildschirmraum | Defaults to Option Value Name wenn leer gelassen |
| **Image** | Optionales Bild für diese Wahl | Toggle **Primary Image** nach Ja bei Bedarf. Klicken Sie auf **Ein weiteres Bild hinzufügen*, um mehr hinzuzufügen. |

**Step 6:** Wenn Sie die Bearbeitung abgeschlossen haben, klicken Sie auf die Schaltfläche ** Wählen Sie den Optionswert***.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Der **Optionswertcode** muss einzigartig sein. Sie können den gleichen Code wie den ursprünglichen Optionswert nicht verwenden.
:::

:::tip
Sie können Optionswerte nach Name, Code oder Katalog Tag suchen.
:::

:::caution
Klicken Sie auf **Cancel** verworfen alle unerwünschten Änderungen.
:::
