# Gewicht bearbeiten

## Was diese Anleitung deckt

Aktualisiert eine bestehende Gewichtsdefinition wie Name, Maximalwert oder Standardstatus.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Weights**.

![Schritt 2](./img/step-2.png)

**Step 3:** Suchen Sie nach dem Gewicht, das Sie bearbeiten möchten, indem Sie den Gewicht-Namen oder Gewicht-Code in das Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Dreipunkt-Menü neben dem Gewicht, dann wählen Sie **Bearbeiten**.

**Step 5:** Aktualisieren Sie die Gewichtsangaben. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| **Weight Code*** | Kennung | Kann nach der Schöpfung nicht geändert werden |
| **Weight Name*** | Beschreibt die Portionsgröße | z.B. „Regular“, „Large“, „Double“ |
| **Max Gewicht** | numerischer Maximalgewichtswert | z.B. „500“ (in Gramm oder in Ihrer Einheit) |
| **Standard** | Sich als Standard markieren | Nur ein Gewicht sollte als Standard pro Option markiert werden |

**Step 6:** Wenn Sie fertig sind, alle Informationen hinzuzufügen, klicken Sie auf die **Save** Taste.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Klicken Sie auf **Cancel** verworfen alle unerwünschten Änderungen.
:::

:::tip
Sie können Gewichte nach Gewicht Name oder Gewicht Code suchen, um schnell den Artikel zu finden, den Sie bearbeiten möchten.
:::

:::tip
Toggle **Default*** auf Ja, um dies für Kunden die vorgewählte Gewichtsoption zu machen.
:::
