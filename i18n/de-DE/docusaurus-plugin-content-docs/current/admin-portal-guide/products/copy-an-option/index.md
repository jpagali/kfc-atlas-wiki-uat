# Option kopieren

## Was diese Anleitung deckt

Dupliziert eine Optionsgruppe, um Zeit bei der Erstellung ähnlicher Anpassungssätze über mehrere Produkte zu sparen.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Optionen**.

**Step 3:** Suchen Sie nach der Option, die Sie kopieren möchten, indem Sie den Optionsnamen, den Optionscode oder den Katalog Tag im Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Dreipunktmenü neben der Option, dann wählen Sie **Kopieren**.

![Schritt 2](./img/step-2.png)

**Step 5:** Das Kopierformular wird mit den Informationen der ursprünglichen Option angezeigt. Aktualisieren Sie die Felder nach Bedarf. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| **Optionscode*** | Einzigartige Kennung für die neue Option | Muss anders sein als das Original. Verwenden Sie Großbuchstaben, Zahlen und Bindestriche (z.B. „OPT-SIZE-COPY“) |
| **Optionsname*** | Der Kundenname der Anpassungskategorie | Kann die gleiche sein wie das Original oder angepasst |

**Step 6:** Wenn Sie mit Ihren Edits fertig sind, klicken Sie auf **Kreate**.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Der **Optionscode** muss einzigartig sein. Sie können den gleichen Code wie die ursprüngliche Option nicht verwenden.
:::

:::tip
Sie können Optionen nach Option Name, Option Code oder Katalog Tag suchen.
:::

:::caution
Klicken Sie auf **Cancel** verworfen alle unerwünschten Änderungen.
:::
