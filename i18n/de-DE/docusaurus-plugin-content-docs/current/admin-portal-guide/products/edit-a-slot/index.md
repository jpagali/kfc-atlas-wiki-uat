# Ein Slot bearbeiten

## Was diese Anleitung deckt

Aktualisiert die Konfiguration eines vorhandenen Slots, wie sein Label, minimale/maximale Mengen, Modifizierungsmittel oder Gewichte.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Slots**.

**Step 3:** Suchen Sie nach dem Slot, den Sie bearbeiten möchten, indem Sie den Slot Name, Slot Code oder Tag im Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Dreipunktmenü neben dem Slot, dann wählen Sie **Bearbeiten**.

![Schritt 2](./img/step-2.png)

**Step 5:** Sie werden die Slot Details mit blauen Abschnitt Links sehen. Klicken Sie auf jeden blauen Link, um direkt zu diesem Abschnitt zu springen:
- **Basic Information** — Slotcode bearbeiten, Name, min/max
- **Modifier* — Modifikatoren hinzufügen, entfernen oder reorderieren
- **Weights* — Hinzufügen, Entfernen oder Reorder Gewichtsoptionen

**Step 6:** Bearbeiten Sie die Bereiche nach Bedarf. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| **Slot Code*** | Kennung | Kann nach der Schöpfung nicht geändert werden |
| **Slot Name*** | Beschreibt, was Anpassung dieser Slot bietet | z.B. „Sauce Selection“, „Cheese Optionen“ |
| **Min. | Mindest-Modifier-Auswahl erforderlich | 0 = optional |
| ** Höchstmenge** | Maximale Modifier-Auswahl erlaubt | Blättern Sie leer für unbegrenzt |

**Step 7:** Wenn Sie mit Ihren Edits fertig sind, klicken Sie auf die Schaltfläche **Save***.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Klicken Sie auf **Cancel** verworfen alle unerwünschten Änderungen.
:::

:::tip
Sie können direkt zu einem Abschnitt springen, indem Sie auf den blauen Abschnitt Link anstatt zu scrollen.
:::

:::tip
Sie können Slots nach Slot Name, Slot Code oder Tag suchen, um schnell den Artikel zu finden, den Sie bearbeiten möchten.
:::
