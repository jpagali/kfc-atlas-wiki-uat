# Löschen Sie das Gewicht

## Was diese Anleitung deckt

Entfernt eine Gewichtskonfiguration, wenn sie nicht mehr anwendbar ist.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Produkte**.

**Step 2:** Klicken Sie auf die Registerkarte **Weights**.

![Schritt 2](./img/step-2.png)

**Step 3:** Suchen Sie nach dem Gewicht, das Sie löschen möchten, indem Sie den Gewicht-Namen oder Gewicht-Code in das Suchfeld eingeben.

**Step 4:** Klicken Sie auf das Drei-Punkt-Menü neben dem Gewicht, dann wählen Sie * **Delete**.

**Step 5:** Eine Bestätigungsmodalität zeigt alle Bereiche des Systems, wo dieses Gewicht verwendet wird. Überprüfen Sie dies sorgfältig, um sicherzustellen, dass Sie das richtige Gewicht löschen.

**Step 6:** Klicken Sie auf die rote **Delete** Schaltfläche, um das Gewicht dauerhaft zu entfernen.

![Schritt 3](./img/step-3.png)

## Anmerkungen

:::caution
Eine Gewichtsabnahme ist dauerhaft und kann nicht rückgängig gemacht werden. Das Gewicht wird von allen Slots und Modifikatoren entfernt, die es verwenden.
:::

:::tip
Sie können Gewichte nach Gewicht Name oder Gewicht Code suchen, um schnell den Artikel zu finden, den Sie löschen möchten.
:::

:::caution
Klicken Sie auf **Cancel**, wenn Sie nicht mit Löschung fortfahren möchten.
:::
