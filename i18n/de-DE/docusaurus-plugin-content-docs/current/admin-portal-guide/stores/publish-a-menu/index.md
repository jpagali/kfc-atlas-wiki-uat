# Menü veröffentlichen

## Was diese Anleitung deckt

Veröffentlicht das zugewiesene Menü den Live-Bestellkanälen des Stores, wodurch das Menü für Kunden sichtbar wird.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Abschnitt **Stores**.

**Step 2:** Klicken Sie auf die **Publish Menu* Schaltfläche oder Tab. Damit wird der Workflow veröffentlicht.

![Schritt 2](./img/step-2.png)

**Step 3:** Wählen Sie die **store(s)* aus, für die Sie Menüs veröffentlichen möchten. Sie können nach Namen, Nummer oder Code suchen, wenn die Liste lang ist.

![Schritt 3](./img/step-3.png)

**Step 4:** Wählen Sie die **-Kanäle** aus, die Sie veröffentlichen möchten (z.B. Digital, Kiosk, In-Store). Sie können auf einmal mehrere Kanäle veröffentlichen.

![Schritt 4](./img/step-4.png)

**Step 5:** Überprüfen Sie die zu veröffentlichenden Änderungen. Klicken Sie auf **Review Changes*, wenn Sie überprüfen müssen, was an Kunden gesendet wird.

**Step 6:** Sobald Sie überprüft und bestätigt haben, klicken Sie auf die Schaltfläche **Publish**, um das Menü live in die ausgewählten Kanäle zu drücken.

![Schritt 5](./img/step-5.png)

:::tip
**Before publishing:** Überprüfen Sie, ob in den Menüeinstellungen des Speichers das richtige Menü jedem Kanal zugeordnet ist. Verwendung[Menü eines Stores anzeigen](/docs/admin-portal-guide/stores/view-a-stores-menu/)die Aufgaben zu bestätigen.
:::

:::caution
Das Veröffentlichen eines Menüs macht sofort Veränderungen für Kunden sichtbar. Überprüfen Sie alle Änderungen, bevor Sie auf Publish klicken.
:::
