# Geschäfte

Dieser Abschnitt umfasst alle geschäftsbezogenen Aufgaben im Byte Commerce Admin Portal, einschließlich Store-Setup, Menümanagement, Objektsicht und Spendenhandling.

:::note Byte POS Caveat
Store-Aufgaben, die **POS**, **device Authentifizierung** oder **store-Nummern mit POS-Aufzeichnungen** angeben, dass der Markt **Byte POS*** verwendet.

Wenn der Markt nicht Byte POS verwendet, sollten Teams **Byte Connect** im Rahmen von Byte Commerce onboarding behandeln, so dass der Speicherintegrationspfad **Byte Commerce -> Byte Connect -> POS** ist.
:::
