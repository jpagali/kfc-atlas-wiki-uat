# Kopie Promotion ID

## Was diese Anleitung deckt

Kopiert die einzigartige ID einer Promotion in Ihre Zwischenablage für die Verwendung in Support-Tickets, Systemintegrationen oder Cross-Referencing mit anderen Systemen.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Promotions**.

**Step 2:** Finden Sie die Promotion, die Sie kopieren möchten. Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte), dann wählen Sie **Copy ID**.

![Schritt 2](./img/step-2.png)

**Step 3:** Die Promotion-ID wurde in Ihre Zwischenablage kopiert. Sie können es jetzt in Support-Tickets, Integrationsformen oder andere Systeme einfügen, wenn nötig.

:::tip
Die Promotion ID ist eine eindeutige Kennung, die von Atlas vergeben wird. Technische Teams verwenden diese ID, um die Förderung in Integrationen, Debugging und Systemaufzeichnungen zu verweisen.
:::
