# Erstellen einer Store-Gruppe

## Was diese Anleitung deckt

Erstellt eine benannte Gruppierung von Geschäften, die Menü, Promotion oder Steuerkonfigurationen teilen — eine Grundstruktur für die Verwaltung mehrerer Standorte im Maßstab. (Dieser Leitfaden erstellt eine Filialgruppe aus dem Bereich Promotions. Sie können auch einen aus dem Bereich Store Groups in der Hauptnavigation erstellen.)

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Promotions**.

**Step 2:** Klicken Sie auf die Registerkarte **Store Groups*.

![Schritt 2](./img/step-2.png)

**Step 3:** Klicken Sie auf die Schaltfläche **+ Neue Store-Gruppe* erstellen.

![Schritt 3](./img/step-3.png)

**Step 4:** Geben Sie die Shop-Gruppendetails ein. Mit * markierte Felder sind erforderlich.

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| ** Store Group Name*** | Ein beschreibender Name für diese Gruppe | z.B. „NSW Franchise Group“, „Corporate Owned Stores“, „Breakfast Pilot 2024“. Sollte sich von anderen Gruppen deutlich unterscheiden. |
| **Store Group Tags** | Optionale Etiketten für Filterung und Berichterstattung | z.B. „pilot“, „korporiert“, „franchise“. Sie können mehrere Tags eingeben. |

![Schritt 4](./img/step-4.png)

**Step 5:** Wählen Sie Speicher aus, um dieser Gruppe hinzuzufügen. Verwenden Sie die bereitgestellte Tabelle, um Geschäfte zu finden und auszuwählen:

- Sie können ** filtern nach Store Number, Store Name oder Franchise Code**, um schnell bestimmte Läden zu finden.
- ** Den Schalter** neben jedem Speichernamen anklicken, um ihn der Gruppe hinzuzufügen.
- Ein Speicher kann mehreren Speichergruppen angehören.

![Schritt 5](./img/step-5.png)

**Step 6:** Überprüfen Sie die Zusammenfassung, die alle ausgewählten Läden und Gruppendetails zeigt. Klicken Sie auf **Kreate**, um die Speichergruppe zu speichern.

![Schritt 6](./img/step-6.png)

:::tip
Ein Speicher kann mehreren Speichergruppen angehören. Promotionen, Steuerregeln und Menüs werden auf der Ebene der Filiale verwaltet und gelten für alle Mitgliedsläden.
:::
