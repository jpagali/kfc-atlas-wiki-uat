# Erstellen Sie serialisierten Code

## Was diese Anleitung deckt

Erzeugt einzigartige Single-Use Promotion Codes gebunden an eine Promotion, verwendet für verfolgte Erlösung Kampagnen, wo jeder Kunde einen bestimmten Code erhält.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Promotions**.

**Step 2:** Finden Sie die Promotion, für die Sie serialisierte Codes erstellen möchten. Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte), dann wählen Sie **Serialized Code***.

![Schritt 2](./img/step-2.png)

**Step 3:** Klicken Sie auf die Schaltfläche **+ Erstellen Sie neuen serialisierten Code** (oder die entsprechende Schaltfläche auf Ihrem Bildschirm).

![Schritt 3](./img/step-3.png)

**Step 4:** Füllen Sie das serialisierte Codeformular mit den folgenden Details aus:

| Feld | Eingeben | Anmerkungen |
|-------|--------------|-------|
| ** Präfix** | Optionaler Text zum Start eines jeden Codes hinzufügen | z.B. „KFC-“, „BOGO2024-“. Blättern Sie leer, wenn kein Präfix benötigt wird. |
| **Quantität** | Anzahl der Codes zur Erzeugung | z.B. „100“, „5000“. Bestimmt, wie viele einzigartige Codes erstellt werden. |
| **Abgelaufenes Datum** | Wenn die Codes abgelaufen sind | Wählen Sie das Datum aus, nach dem Codes nicht eingelöst werden können. |

![Schritt 4](./img/step-4.png)

**Step 5:** Klicken Sie auf **Generieren Sie Codes**, um die Codes in die serialisierte Codeliste zu erstellen und hinzuzufügen.

:::tip
Serialisierte Codes sind einzigartige, einmalige Erlösungscodes. Jeder Code kann nur einmal eingelöst werden. Einmal generiert, werden Codes zur Förderung hinzugefügt und bereit für den Vertrieb an Kunden.
:::
