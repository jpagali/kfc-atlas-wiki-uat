# Promotionen für eine Store-Gruppe anzeigen

## Was diese Anleitung deckt

Listet alle Werbeaktionen, die einer Speichergruppe für Sicht- und Auditzwecke zugewiesen werden.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Promotions**.

**Step 2:** Klicken Sie auf die Registerkarte **Store Groups*.

![Schritt 2](./img/step-2.png)

**Step 3:** Finden Sie die Speichergruppe, für die Sie Werbeaktionen anzeigen möchten, indem Sie die Tabelle durchsuchen oder die Suchleiste verwenden.

![Schritt 3](./img/step-3.png)

**Step 4:** Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte) neben dem Speichergruppennamen, dann wählen Sie **Promotions***.

![Schritt 4](./img/step-4.png)

**Step 5:** Ein Schubladenpanel öffnet alle Aktionen, die dieser Speichergruppe zugeordnet sind. Die Liste enthält:

- **Promotion Name** — Der kundenorientierte Name der Promotion
- **Beschreibung** — Kurzbeschreibung der Aktion
- **Status** Ob die Promotion aktiv oder archiviert ist

**Step 6 (Optional):** Um eine Aktion aus der Speichergruppe zu entfernen, klicken Sie auf die Schaltfläche **Unlink** neben dem Werbenamen. Sie werden gebeten, die Entfernung zu bestätigen.

:::tip
Eine Werbeaktion von einer Filialgruppe zu verlinken, macht es sofort in allen Geschäften in dieser Gruppe nicht verfügbar.
:::
