# Archive a Promotion

## Was diese Anleitung deckt

Markiert eine Promotion als archiviert, um sie vom aktiven Gebrauch zu entfernen, während sie seinen Rekord für die historische Berichterstattung hält.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü auf den Abschnitt **Promotions**.

**Step 2:** Finden Sie die Promotion, die Sie archivieren möchten. Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte), dann wählen Sie **Archive***.

![Schritt 2](./img/step-2.png)

![Schritt 3](./img/step-3.png)

**Step 3:** Ein Warndialog wird angezeigt, dass die Archivierung der Aktion sie von zugewiesenen Speichergruppen entfernt. Klicken Sie auf die Schaltfläche **Ja, Archiv Promotion**, um zu bestätigen.

![Schritt 4](./img/step-4.png)

:::caution
Die Archivierung deaktiviert sofort die Promotion über alle zugewiesenen Speichergruppen und entfernt sie von kundenorientierten Kanälen. Der Werberekord wird zu historischen Zwecken aufbewahrt.
:::

## Archivierte Promotionen zu finden und wiederherzustellen

**Step 1:** In der Promotionsliste verwenden Sie die Filterspalte **Archiviert** und wählen Sie *****, um alle archivierten Aktionen anzuzeigen.

**Step 2:** Finden Sie die Aktion, die Sie wiederherstellen möchten, und klicken Sie auf den **Promotion Name*, um den Promotion Wizard zu öffnen.

**Step 3:** Klicken Sie auf die Schaltfläche **Unarchive**, um die Aktion zum aktiven Status wiederherzustellen.
