# Stores in einer Store-Gruppe anzeigen

## Was diese Anleitung deckt

Listet alle Geschäfte, die derzeit zu einer Filialgruppe gehören, für Sicht- und Verifikationszwecke.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Bereich **Store Groups**.

**Step 2:** Durchsuchen Sie die Tabelle Store Groups, um die Gruppe zu finden, die Sie sehen möchten, oder verwenden Sie die Suchleiste, um sie zu lokalisieren. Klicken Sie auf den **store Gruppennamen**, um seine Details anzuzeigen.

![Schritt 2](./img/step-2.png)

**Step 3:** Eine detaillierte Ansicht wird zeigen:

- ** Store Group Name** — Der Name der ausgewählten Gruppe
- **Store Group Tags* — Jede dieser Gruppe zugeordnete Tags
- **Stores Table* — Eine vollständige Liste aller Geschäfte in dieser Gruppe

Die Tabelle zeigt:
- **Store Number** — Einzigartige Kennung für jeden Speicher
- **Store Name** — Name des Lagerorts
- **Franchise Code** — Franchise-Kennung gegebenenfalls
- **Tax Rules** Ob diese Gruppe Steuerregeln assoziiert hat

![Schritt 3](./img/step-3.png)

:::tip
Die Stores-Tabelle listet alle Geschäfte, die derzeit zu dieser Store-Gruppe gehören. Um die Mitgliedschaft zu ändern, siehe[Eine Store-Gruppe bearbeiten](/docs/admin-portal-guide/store-groups/edit-a-store-group/).
:::
