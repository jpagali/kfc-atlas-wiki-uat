# Promotionen bearbeiten

## Was diese Anleitung deckt

Fügt oder entfernt Werbezuweisungen für eine bestimmte Speichergruppe, so dass Sie ändern, welche Promotionen in Member Stores aktiv sind.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Bereich **Store Groups**.

**Step 2:** Finden Sie die Store-Gruppe, deren Promotionen Sie ändern möchten. Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte) neben dem Speichergruppennamen.

![Schritt 2](./img/step-2.png)

**Step 3:** Klicken Sie im Dropdown-Menü auf **Promotionen**.

![Schritt 3](./img/step-3.png)

**Step 4:** Eine Werbeschublade wird geöffnet. Klicken Sie auf die Schaltfläche **Bearbeiten Promotions**, um den Bearbeitungsmodus einzutragen.

![Schritt 4](./img/step-4.png)

**Step 5:** Änderungen an den Werbeaufträgen vornehmen:

- **Um eine Promotion hinzuzufügen:** Überprüfen Sie die Checkbox neben den Werbenamen, die Sie hinzufügen möchten
- ** Um eine Aktion zu entfernen:** Überprüfen Sie das Kontrollkästchen neben den Werbenamen, die Sie entfernen möchten
- **Suche*** für bestimmte Aktionen mit der Suchleiste
- Toggle the **"Weitere Aktionen anzeigen"** wechseln, um Werbeaktionen zu sehen, die derzeit nicht zugewiesen sind (falls geprüft, können Sie alle verfügbaren Aktionen sehen; wenn nicht geprüft, sehen Sie nur aktuell zugeordnete).

![Schritt 5](./img/step-5.png)

**Step 6:** Überprüfen Sie Ihre Änderungen in der Zusammenfassung Abschnitt Anzeigen Promotionen hinzugefügt und entfernt. Klicken Sie auf die Schaltfläche **Assign**, um die Änderungen anzuwenden. (Die Schaltfläche Zuweisung wird erst aktiviert, nachdem Sie Änderungen vorgenommen haben.)

:::tip
Änderungen an Werbeaufträgen wirken sofort für alle Geschäfte in dieser Speichergruppe.
:::

:::tip
Sie können Werbe-Namen in der Suchleiste suchen, um schnell Aktionen zu finden, die Sie hinzufügen oder entfernen möchten.
:::
