# Import Promotions für eine Store-Gruppe

## Was diese Anleitung deckt

Bulk-importiert Werbezuweisungen von einer Speichergruppe zum anderen für großformatige Kampagnen-Setup oder Konfigurationsvervielfältigung.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Bereich **Store Groups**.

**Step 2:** Finden Sie die Store-Gruppe, die **receive** die Promotions (die Zielgruppe). Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte) neben dem Speichergruppennamen.

![Schritt 2](./img/step-2.png)

**Step 3:** Klicken Sie **Import Promotions** im Dropdown-Menü.

![Schritt 3](./img/step-3.png)

**Step 4:** Ein Dialog wird geöffnet. Wählen Sie die Speichergruppe aus, die Sie Werbeaktionen importieren möchten ** von** (die Quellgruppe). Sie können danach mit der Suchleiste suchen.

**Step 5 (Optional):** Entscheiden Sie, ob Sie alle oder spezifische Aktionen importieren:

| Option | Aktion |
|--------|--------|
| **Importieren Sie alle** | Überlassen Sie die Klinke ON, um jede Aktion aus der Quellgruppe zu importieren |
| **Importspezifisch** | Schalten Sie den "Import All" Schalter, dann überprüfen Sie die Boxen neben nur die Aktionen, die Sie importieren möchten |

![Schritt 4](./img/step-4.png)

**Step 6:** Überprüfen Sie die Importübersicht und klicken Sie auf **Save**, um den Import abzuschließen.

:::caution
**Important:** Importieren von Promotions wird ** ersetzen** alle vorhandenen Promotions im Zusammenhang mit der Ziel-Shop-Gruppe. Diese Aktion kann nicht rückgängig gemacht werden. Stellen Sie sicher, dass Sie vor der Bestätigung in die richtige Speichergruppe importieren.
:::

:::tip
Wenn Sie nur bestimmte Aktionen hinzufügen möchten, ohne bestehende zu ersetzen, verwenden Sie die[Promotionen bearbeiten](/docs/admin-portal-guide/store-groups/edit-promotions/)statt.
:::
