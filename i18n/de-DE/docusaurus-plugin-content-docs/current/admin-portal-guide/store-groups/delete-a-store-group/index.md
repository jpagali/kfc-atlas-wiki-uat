# Löschen einer Store Group

## Was diese Anleitung deckt

Entfernt permanent eine Filialgruppe aus dem System, zusammen mit allen zugehörigen Promotionen und Steuerregeln.

## Schritte

**Step 1:** Navigieren Sie mit dem linken Navigationsmenü in den Bereich **Store Groups**.

**Step 2:** Finden Sie die Speichergruppe, die Sie löschen möchten, indem Sie die Tabelle durchsuchen oder die Suchleiste verwenden. Klicken Sie auf die Schaltfläche **Aktionsmenü* (drei Punkte) neben dem Speichergruppennamen.

![Schritt 2](./img/step-2.png)

**Step 3:** Klicken Sie auf **Delete**.

**Step 4:** Ein Bestätigungsdialog wird angezeigt. Überprüfen Sie die Warnmeldung und klicken Sie auf die Schaltfläche **Delete Store Group**, um die dauerhafte Löschung zu bestätigen.

![Schritt 3](./img/step-3.png)

:::caution
**Diese Aktion ist dauerhaft und kann nicht rückgängig gemacht werden* Deleting a store group will:
- Entfernen Sie die Speichergruppe aus dem System
- Verlinken Sie alle Werbeaktionen, die dieser Speichergruppe zugeordnet sind
- Verlinken Sie alle Steuerregeln im Zusammenhang mit dieser Filialgruppe
- Die Mitgliederspeicher selbst werden nicht gelöscht, nur ihre Gruppierung und Aufgaben

:::tip
Wenn Sie die Speichergruppen-Konfiguration für die zukünftige Nutzung behalten müssen, beachten Sie **copying* vor dem Löschen.
:::
