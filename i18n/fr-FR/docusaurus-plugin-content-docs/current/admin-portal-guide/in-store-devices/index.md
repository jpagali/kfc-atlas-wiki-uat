# Dispositifs en magasin

Cette section couvre la gestion des terminaux et des kiosques POS (point de vente) dans les restaurants KFC, y compris la configuration, l'état et l'authentification des appareils.

:::note Octet POS Cavat
Les guides en magasin ci-dessous sont écrits pour les marchés en utilisant **Byte POS**.

Si un marché n'utilise pas Byte POS, **Byte Connect** doit être inclus dans le bundle d'embarquement Byte Commerce, et les équipes devraient comprendre le chemin de commande comme **Byte Commerce -> Byte Connect -> POS** plutôt que d'assumer la communication directe entre Byte Commerce et le POS du marché.
:::

## Thèmes

- [Générer un mot de passe unique](/docs/admin-portal-guide/in-store-devices/generate-one-time-password/)— Créer un code d'authentification temporaire pour la configuration ou la réauthentification des appareils.
- [Voir les détails du périphérique en magasin](/docs/admin-portal-guide/in-store-devices/view-in-store-device-details/)— Afficher tous les détails d'un appareil incluant le nom, le type, l'état et le menu publier l'historique.
- [Désactivation en cours de fabrication](/docs/admin-portal-guide/in-store-devices/deactivate-in-store/)— Désactiver un appareil de la réception des mises à jour de menus ou des commandes de traitement.

Besoin du contexte général? Lire[Byte Connect](/docs/byte-capabilities/enablement/byte-connect).
