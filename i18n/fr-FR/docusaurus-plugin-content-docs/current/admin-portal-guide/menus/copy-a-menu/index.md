# Copier un menu

## Ce que ce guide couvre

Dupliquer un menu existant à utiliser comme point de départ pour un nouveau menu, en économisant un temps de configuration important.

## Étapes

**Step 1:** Naviguez dans la section **Menus** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez le menu que vous voulez copier dans la liste des menus, cliquez sur le menu **action** (trois points) dans la même ligne, et sélectionnez **Copie**.

![Étape 2](./img/step-2.png)

**Step 3:** Le formulaire de détails du menu apparaîtra. Mettre à jour les champs suivants au besoin :

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code du menu** * | Un identifiant unique pour le nouveau menu | Utilisez les lettres majuscules, les chiffres et les tirets seulement — par exemple,`AU-BREAKFAST-2025`. Doit être différent du menu original. Impossible de changer après la création. |
| **Nom de famille** * | Un nom lisible par l'homme pour ce menu | Par exemple, "Australia Breakfast Menu 2025". Afficher dans la liste des menus et lors de l'attribution des menus aux magasins. |

Toutes les catégories, produits et paquets du menu original sont automatiquement copiés.

![Étape 3](./img/step-3.png)

**Step 4:** Une fois que vous avez entré le nouveau code et nom du menu, cliquez sur **Créer** pour enregistrer le menu copié.

![Étape 4](./img/step-4.png)

:::note :
Le menu copié aura la même structure de catégorie et les éléments que l'original. Modifiez-le après la création si vous devez apporter des modifications.
:::
