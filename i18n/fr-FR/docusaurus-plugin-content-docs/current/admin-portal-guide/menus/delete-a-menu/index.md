# Supprimer un menu

## Ce que ce guide couvre

Supprime définitivement un menu du système lorsqu'il n'est plus utilisé.

## Étapes

**Step 1:** Naviguez dans la section **Menus** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez le menu que vous voulez supprimer dans la liste des menus, cliquez sur le menu **action** (trois points) dans la même ligne, et sélectionnez **Supprimer**.

![Étape 2](./img/step-2.png)

**Step 3:** Une boîte de dialogue de confirmation apparaîtra. Cliquez sur le bouton **Supprimer** pour supprimer définitivement le menu.

![Étape 3](./img/step-3.png)

:::caution
Supprimer ce menu va le supprimer de tous les magasins assignés. Cependant, les menus publiés en direct dans ces magasins ne seront pas affectés — seules les futures affectations de menu seront touchées. Cette action ne peut être annulée.
:::
