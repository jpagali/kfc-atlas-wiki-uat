# Menus

Cette section couvre toutes les tâches de gestion de menu dans le portail Byte Commerce Admin, y compris la création de menus, leur organisation en catégories, la configuration des dépassements ciblés avec les correctifs et la gestion des structures de tarification.

## Menus

- [Créer un menu](/docs/admin-portal-guide/menus/create-a-menu/)— Construire une nouvelle structure de menu pour les magasins et les canaux
- [Modifier un menu](/docs/admin-portal-guide/menus/edit-a-menu/)— Mettre à jour la configuration d'un menu existant
- [Copier un menu](/docs/admin-portal-guide/menus/copy-a-menu/)— Dupliquer un menu comme point de départ
- [Attribuer un menu](/docs/admin-portal-guide/menus/assign-a-menu/)— Relier un menu à des magasins et canaux spécifiques
- [Publier un menu](/docs/admin-portal-guide/menus/publish-a-menu/)— Poussez un menu pour commander en direct
- [Supprimer un menu](/docs/admin-portal-guide/menus/delete-a-menu/)— Supprimer un menu du système

## Catégories

- [Créer une catégorie](/docs/admin-portal-guide/menus/create-a-category/)— Ajouter une étiquette de regroupement pour organiser les produits dans un menu
- [Modifier une catégorie](/docs/admin-portal-guide/menus/edit-a-category/)— Mettre à jour le nom ou la configuration d'une catégorie
- [Copier une catégorie](/docs/admin-portal-guide/menus/copy-a-category/)— Dupliquer une catégorie avec sa configuration
- [Supprimer une catégorie](/docs/admin-portal-guide/menus/delete-a-category/)— Supprimer une catégorie d'un menu
- [Ajouter des Metafields à une catégorie](/docs/admin-portal-guide/menus/add-metafields-to-a-category/)— Joindre les données personnalisées à une catégorie pour les intégrations

## Patches

- [Créer un patch](/docs/admin-portal-guide/menus/create-a-patch/)— Construisez une préséance ciblée pour les modifications de menu
- [Modifier un lot](/docs/admin-portal-guide/menus/edit-a-patch/)— Mettre à jour les opérations ou les éléments d'un patch
- [Copier un patch](/docs/admin-portal-guide/menus/copy-a-patch/)— Dupliquer un patch comme point de départ
- [Supprimer un lot](/docs/admin-portal-guide/menus/delete-a-patch/)— Supprimer un patch du système
- [Attribuer un lot (Ajouter à la liste des lots)](/docs/admin-portal-guide/menus/assign-a-patch-add-to-patch-list/)— Ajouter un correctif à la liste active d'un magasin
- [Attribuer un lot (remplacer la liste existante)](/docs/admin-portal-guide/menus/assign-a-patch-replace-existing-list/)— Remplacer la liste complète des patchs d'un magasin

## Prix

- [Créer un prix nommé](/docs/admin-portal-guide/menus/create-a-named-price/)— Définir une étiquette de prix réutilisable pour plusieurs produits
- [Modifier le prix nommé](/docs/admin-portal-guide/menus/edit-named-price/)— Mettre à jour la valeur d'un prix désigné existant
- [Supprimer Prix Nommé](/docs/admin-portal-guide/menus/delete-named-price/)— Enlever un prix indiqué du système

## Domaine associé

- [Jours](/docs/admin-portal-guide/dayparts/)— Gérer les parties de jour de niveau org, les dépassements de niveau de magasin, les prix en fonction du temps, la disponibilité et la visibilité de catégorie
