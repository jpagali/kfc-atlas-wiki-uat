# Supprimer une catégorie

## Ce que ce guide couvre

Supprime définitivement une catégorie du système lorsqu'elle n'est plus nécessaire.

## Étapes

**Step 1:** Naviguez dans la section **Menus** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur le dossier **Catégories** pour afficher toutes les catégories.

![Étape 2](./img/step-2.png)

**Step 3:** Trouvez la catégorie que vous voulez supprimer, cliquez sur le menu **action** (trois points) dans la même ligne, et sélectionnez **Supprimer**.

![Étape 3](./img/step-3.png)

**Step 4:** Une boîte de dialogue de confirmation apparaîtra. Cliquez sur **Supprimer** pour supprimer définitivement la catégorie.

![Étape 4](./img/step-4.png)

:::caution
Supprimer une catégorie la supprimera de tous les menus qui l'utilisent. Les produits précédemment classés dans cette catégorie seront non affectés et pourront être réaffectés à d'autres catégories. Cette action ne peut être annulée.
:::
