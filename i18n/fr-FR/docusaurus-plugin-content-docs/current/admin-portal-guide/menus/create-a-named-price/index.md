# Créer un prix nommé

## Ce que ce guide couvre

Définit une étiquette de prix réutilisable qui peut être appliquée à plusieurs produits ou variantes, permettant une gestion uniforme des prix à l'échelle. Mettre à jour tous les produits en utilisant un prix nommé en éditant le prix une fois.

## Étapes

**Step 1:** Naviguez dans la section **Menus** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Prix nommés** pour afficher tous les prix nommés existants.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **Créer un nouveau prix**.

![Étape 3](./img/step-3.png)

**Step 4:** Remplissez les détails de prix nommés. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Nom** * | Une étiquette qui identifie ce point de prix | Par exemple, « Prix du déjeuner pour les étudiants », « Prix de l'heure heureuse », « Prix de l'article premier ». Utilisé en interne pour identifier le prix. |
| **Prix** * | Le montant monétaire dans la monnaie de votre marché | Inscrivez uniquement les numéros — par exemple,`4.99`, `12.50`. Ne pas inclure le symbole de monnaie. Le montant que les clients paieront. |

![Étape 4](./img/step-4.png)

**Step 5:** Cliquez sur **Créer Prix Nommé** pour enregistrer.

:::tip
Une fois créé, vous pouvez appliquer ce prix nommé à plusieurs produits ou variantes. Lorsque vous devez changer le prix, mettez-le à jour une fois ici et tous les produits qui l'utilisent seront mis à jour automatiquement.
:::

:::note :
Utilisez la boîte de recherche en haut pour trouver les prix nommés existants. Vous pouvez ajuster le nombre de résultats affichés par page en utilisant le menu déroulant.
:::
