# Modifier une catégorie

## Ce que ce guide couvre

Mettre à jour un nom de catégorie, des paramètres d'affichage ou une configuration.

## Étapes

**Step 1:** Naviguez dans la section **Menus** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur le dossier **Catégories** pour afficher toutes les catégories.

![Étape 2](./img/step-2.png)

**Step 3:** Trouvez la catégorie à éditer, cliquez sur le menu **action** (trois points) dans la même ligne et sélectionnez **Edit**.

![Étape 3](./img/step-3.png)

**Step 4:** Mettre à jour l'un des champs suivants :

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Nom de la catégorie** | Le nom de l'affichage de cette catégorie | Par exemple, "Chicken", "Sides", "Desserts". Les changements seront visibles pour les clients. |
| **Afficher le nom** | Autre nom optionnel pour des canaux spécifiques | Utilisez une étiquette plus courte si nécessaire sur certaines plateformes. Laisser en blanc pour utiliser le nom de catégorie. |
| **Description** | Notes internes facultatives concernant cette catégorie | Visible seulement pour votre équipe — pas montré aux clients. |
| **Heures disponibles** | Fenêtre de temps optionnelle pour cette catégorie | Limiter la disponibilité à des heures précises, p.ex. « 6h00 - 11h00 ». Laisser en blanc pour toute la journée de disponibilité. |

![Étape 4](./img/step-4.png)

**Step 5:** Une fois vos modifications effectuées, cliquez sur **Enregistrer** pour les appliquer.

:::note :
Le code de catégorie ne peut être édité — il est permanent et utilisé pour les intégrations système. Les changements au nom de catégorie et à la disponibilité prennent effet lorsque le menu est republié.
:::
