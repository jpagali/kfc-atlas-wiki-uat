# Créer une catégorie

## Ce que ce guide couvre

Crée une étiquette de regroupement (par exemple 'Chicken', 'Sides', 'Desserts') pour organiser des produits dans un menu pour les clients qui naviguent sur l'interface de commande.

## Étapes

**Step 1:** Naviguez dans la section **Menus** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur le dossier **Catégories** pour afficher toutes les catégories existantes.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **Créer une nouvelle catégorie**.

![Étape 3](./img/step-3.png)

**Step 4:** Remplissez les détails de la catégorie. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code de catégorie** * | Un identifiant unique pour cette catégorie | Utiliser des lettres majuscules et des tirets, par exemple,`CAT-CHICKEN`, `CAT-SIDES`. Utilisé dans les intégrations système. Impossible de changer après la création. |
| **Nom de la catégorie** * | Le nom de l'affichage de cette catégorie | Par exemple, "Chicken", "Sides", "Desserts". C'est ce que les clients voient lors de la navigation dans le menu. |
| **Afficher le nom** | Autre nom optionnel pour des canaux spécifiques | Utilisez une étiquette plus courte si nécessaire sur certaines plates-formes (p. ex., « Boissons » au lieu de « Beverages »). |
| **Description** | Notes internes facultatives concernant cette catégorie | Par exemple, « Tous les articles de poulet grillé », « Disponible pendant les heures du petit déjeuner ». Visible seulement pour votre équipe. |
| **Heures disponibles** | Fenêtre de temps optionnelle pour cette catégorie | Limiter la disponibilité à des heures précises, p. ex. « 6h00 - 11h00 » pour les articles pour le petit déjeuner. Laisser en blanc pour toute la journée de disponibilité. |

![Étape 4](./img/step-4.png)

**Step 5:** Une fois que vous avez rempli les détails, cliquez sur **Créer la catégorie** pour enregistrer.

:::tip
Après avoir créé la catégorie, vous pouvez y ajouter des produits lors de la construction ou de l'édition d'un menu en utilisant le menu déroulant **Ajouter des produits/bundles**.
:::
