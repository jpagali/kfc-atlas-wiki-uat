# Modifier un produit

## Ce que ce guide couvre

Mettre à jour les informations d'un produit existant comme le nom, la description, les images, le prix ou la disponibilité pour refléter les changements de menu, les corrections ou les mises à jour de marque sans recréer l'article.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez le produit que vous voulez modifier. Vous pouvez rechercher en entrant le nom du produit ou le code du produit dans le champ de recherche.

**Step 3:** Cliquez sur le menu à trois points à côté du nom du produit, puis sélectionnez **Edit**.

![Étape 2](./img/step-2.png)

**Step 4:** Vous verrez le formulaire d'édition avec toutes les pages du processus de création. Pour sauter directement à une section, cliquez sur l'en-tête de la section bleue (par exemple, Informations de base, Options, Variantes). Pour naviguer étape par étape, cliquez sur **Suivant**.

**Step 5:** Faites vos changements. Les champs marqués d'un * sont obligatoires. Seul le bouton **Save** sera actif lorsque vous aurez apporté des modifications.

**Step 6:** Lorsque vous avez terminé vos modifications, cliquez sur le bouton **Enregistrer**.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::

:::tip
Vous pouvez rechercher des produits par nom de produit ou code de produit pour trouver rapidement l'élément que vous voulez modifier.
:::

:::tip
Cliquez sur les en-têtes de la section bleue pour passer directement à la section que vous souhaitez modifier au lieu de naviguer étape par étape.
:::
