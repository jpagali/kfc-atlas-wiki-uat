# Supprimer un poids

## Ce que ce guide couvre

Supprime une configuration de poids lorsqu'elle n'est plus applicable.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Poids**.

![Étape 2](./img/step-2.png)

**Step 3:** Recherchez le poids que vous voulez supprimer en entrant le nom de poids ou le code de poids dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté du poids, puis sélectionnez **Supprimer**.

**Step 5:** Un modal de confirmation apparaîtra montrant toutes les zones du système où ce poids est utilisé. Examinez ceci avec soin pour vous assurer que vous supprimez le poids correct.

**Step 6:** Cliquez sur le bouton rouge **Supprimer** pour enlever définitivement le poids.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
La suppression d'un poids est permanente et ne peut être annulée. Le poids sera retiré de toutes les fentes et les modificateurs qui l'utilisent.
:::

:::tip
Vous pouvez rechercher des poids par nom de poids ou code de poids pour trouver rapidement l'élément que vous voulez supprimer.
:::

:::caution
Cliquez sur **Annuler** si vous ne voulez pas procéder à la suppression.
:::
