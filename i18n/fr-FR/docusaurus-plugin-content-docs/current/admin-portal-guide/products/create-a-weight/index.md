# Créer un poids

## Ce que ce guide couvre

Définit une configuration de taille de poids ou de portion pour un produit, utilisé sur les marchés où les articles sont suivis ou évalués en fonction du poids.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Poids**.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **+ Créer un nouveau poids**.

**Step 4:** Remplissez les détails de poids. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code de poids** * | Identificateur unique pour ce poids | Utiliser des lettres majuscules, des chiffres et des tirets (p. ex. |
| **Nom du poids** * | Décrit la taille de la portion | Par exemple, "Regular", "Large", "Double" |
| ** Valeur maximale en poids** | Valeur du poids maximal numérique | Par exemple, 500 (en grammes ou votre unité) |
| **Défaut** | Basculer pour marquer comme défaut | Un seul poids doit être marqué par défaut par option |

![Étape 3](./img/step-3.png)

**Step 5:** Lorsque vous avez terminé d'ajouter toutes les informations, cliquez sur le bouton **Créer le poids**.

## Annexe

:::caution
Cliquez sur **Annuler** rejette toutes les informations non enregistrées.
:::

:::tip
Le **Code de poids** doit être unique et ne peut être modifié après la création.
:::

:::tip
Toggle **Défaut** à Oui pour faire de cela l'option de poids présélectionné pour les clients.
:::
