# Supprimer une valeur d'option

## Ce que ce guide couvre

Supprime un choix spécifique d'un groupe d'options lorsqu'il est arrêté ou n'est plus nécessaire.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Valeurs d'option**.

![Étape 2](./img/step-2.png)

**Step 3:** Recherchez la valeur d'option que vous souhaitez supprimer en entrant le nom, le code ou l'étiquette de catalogue dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté de la valeur de l'option, puis sélectionnez **Supprimer**.

**Step 5:** Un modal de confirmation apparaîtra montrant toutes les zones du système où cette option est utilisée. Passez en revue attentivement pour vous assurer que vous supprimez le bon choix.

**Step 6:** Cliquez sur le bouton rouge **Supprimer** pour supprimer définitivement la valeur de l'option.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
La suppression d'une valeur d'option est permanente et ne peut être annulée. Le choix sera retiré de tous les produits qui l'offrent.
:::

:::tip
Vous pouvez rechercher des valeurs d'option par nom, code ou étiquette de catalogue pour trouver rapidement l'élément que vous voulez supprimer.
:::

:::caution
Cliquez sur **Annuler** si vous ne voulez pas procéder à la suppression.
:::
