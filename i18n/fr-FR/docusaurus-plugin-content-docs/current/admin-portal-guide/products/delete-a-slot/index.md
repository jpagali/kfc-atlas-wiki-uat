# Supprimer une fente

## Ce que ce guide couvre

Supprime une fente du système lorsque la configuration n'est plus nécessaire.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Slots**.

![Étape 2](./img/step-2.png)

**Step 3:** Recherchez la fente que vous voulez supprimer en entrant le nom de la fente, le code de fente ou l'étiquette dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté de la fente, puis sélectionnez **Supprimer**.

**Step 5:** Un modal de confirmation apparaîtra montrant toutes les zones du système où ce slot est utilisé. Passez en revue attentivement pour vous assurer que vous supprimez la bonne fente.

**Step 6:** Cliquez sur le bouton rouge **Supprimer** pour supprimer définitivement la fente.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
La suppression d'une fente est permanente et ne peut être annulée. La fente sera retirée de tous les produits qui l'utilisent.
:::

:::tip
Vous pouvez rechercher des fentes par nom de fente, code de fente, ou étiquette pour trouver rapidement l'élément que vous voulez supprimer.
:::

:::caution
Cliquez sur **Annuler** si vous ne voulez pas procéder à la suppression.
:::
