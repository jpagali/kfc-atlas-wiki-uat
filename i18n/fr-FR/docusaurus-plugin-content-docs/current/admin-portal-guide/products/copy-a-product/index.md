# Copier un produit

## Ce que ce guide couvre

Dupliquer un produit existant comme point de départ, en réduisant la saisie des données lors de la création d'articles similaires dans le catalogue.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez le produit que vous voulez copier. Vous pouvez rechercher par nom de produit ou code de produit.

**Step 3:** Cliquez sur le menu à trois points à côté du nom du produit, puis sélectionnez **Copie**.

![Étape 2](./img/step-2.png)

**Step 4:** Le formulaire de copie apparaîtra avec la plupart des informations du produit original déjà remplies. Modifier les champs au besoin.

**Step 5:** Suivez les mêmes étapes que de créer un nouveau produit. Passez par chaque page (Informations de base, Options, Variantes, Fentes, Actions en vrac, Mots clés, Revue) en cliquant sur **Suivant** ou sautez directement à une section en cliquant sur son en-tête bleu.

**Step 6:** Lorsque vous avez terminé vos modifications, cliquez sur le bouton **Créer**. Le bouton ne sera cliquable qu'après avoir apporté des modifications.

![Étape 3](./img/step-3.png)

## Annexe

:::tip
Vous pouvez rechercher des produits par nom de produit ou code de produit pour trouver rapidement l'article que vous voulez copier.
:::

:::tip
Cliquez sur les en-têtes de section bleue pour passer directement à la section que vous voulez modifier au lieu de naviguer étape par étape.
:::

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::
