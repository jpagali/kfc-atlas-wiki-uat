# Modifier un poids

## Ce que ce guide couvre

Mettre à jour une définition de poids existante telle que le nom, la valeur maximale ou l'état par défaut.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Poids**.

![Étape 2](./img/step-2.png)

**Step 3:** Recherchez le poids que vous voulez modifier en entrant le nom de poids ou le code de poids dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté du poids, puis sélectionnez **Edit**.

**Step 5:** Mettre à jour les détails de poids. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code de poids** * | identificateur unique | Impossible de changer après la création |
| **Nom du poids** * | Décrit la taille de la portion | Par exemple, "Regular", "Large", "Double" |
| ** Valeur maximale en poids** | Valeur du poids maximal numérique | Par exemple, 500 (en grammes ou votre unité) |
| **Défaut** | Basculer pour marquer comme défaut | Un seul poids doit être marqué par défaut par option |

**Step 6:** Lorsque vous avez terminé d'ajouter toutes les informations, cliquez sur le bouton **Save**.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::

:::tip
Vous pouvez rechercher les poids par nom de poids ou code de poids pour trouver rapidement l'élément que vous voulez modifier.
:::

:::tip
Toggle **Défaut** à Oui pour faire de cela l'option de poids présélectionné pour les clients.
:::
