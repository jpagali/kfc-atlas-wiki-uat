# Créer une option

## Ce que ce guide couvre

Crée un groupe de personnalisation (p. ex., "Size", "Spice Level") qui peut être lié à des produits, permettant aux clients de personnaliser leur commande avec des choix.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Options**.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **+ Créer une nouvelle option**.

**Step 4:** Remplissez les détails de l'option. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code d'option** * | Identifiant unique pour cette option | Utiliser les lettres majuscules, les chiffres et les tirets (p. ex. |
| **Nom de l'option** * | Le nom de catégorie de personnalisation indiqué aux clients | Par exemple, Taille, Saveur, Niveau de vitesse |

![Étape 3](./img/step-3.png)

**Step 5:** Cliquez sur **Créer l'option** pour enregistrer l'option.

## Annexe

:::caution
Cliquez sur **Annuler** rejette toutes les informations non enregistrées.
:::

:::tip
Le **Code d'Option** doit être unique et ne peut pas être modifié après la création, alors choisissez-le soigneusement.
:::
