# Copier un poids

## Ce que ce guide couvre

Dupliquer une entrée de poids pour réduire le temps de configuration lors de la création de configurations de poids similaires.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Poids**.

**Step 3:** Recherchez le poids que vous voulez copier en entrant le nom de poids ou le code de poids dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté du poids, puis sélectionnez **Copie**.

![Étape 2](./img/step-2.png)

**Step 5:** Le formulaire de copie apparaîtra avec les renseignements originaux sur le poids. Mettre à jour les champs au besoin. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code de poids** * | Identificateur unique pour le nouveau poids | Doit être différent de l'original (p. ex. |
| **Nom du poids** * | Décrit la taille de la portion | Peut être le même ou personnalisé |
| ** Valeur maximale en poids** | Valeur du poids maximal numérique | Par exemple, 500 (en grammes ou votre unité) |
| **Défaut** | Basculer pour marquer comme défaut | Un seul poids doit être marqué par défaut par option |

**Step 6:** Lorsque vous avez terminé d'ajouter toutes les informations, cliquez sur le bouton **Créer le poids**.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
Le **Code de poids** doit être unique. Vous ne pouvez pas utiliser le même code que le poids initial.
:::

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::

:::tip
Vous pouvez rechercher les poids par nom de poids ou code de poids.
:::
