# Créer un modificateur

## Ce que ce guide couvre

Crée un complément ou un élément de personnalisation (p. ex., "Extra Cheese") qui peut être relié à des produits via des fentes, permettant aux clients de personnaliser leur commande avec des mises à niveau optionnelles ou requises.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Modificateurs**.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **+ Créer un nouveau modificateur**.

**Step 4:** Remplissez les détails du modificateur. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code de modification** * | Identifiant unique pour ce modificateur | Utiliser des lettres majuscules, des chiffres et des tirets (p. ex. |
| **Nom de la modification** * | Nom indiqué aux clients sur les canaux de commande | Par exemple, "Extra Cheese", "No Pickles", "Extra Sauce" |
| **Prix** | Frais supplémentaires pour la sélection de ce modificateur | Entrez`0`s'il n'y a pas de frais supplémentaires |
| **Image** | Image en option pour ce modificateur | Basculer **Image principale** à Oui pour définir comme image principale d'affichage. Cliquez sur **Ajouter une autre image** pour ajouter plus. |

![Étape 3](./img/step-3.png)

**Step 5:** Lorsque vous avez terminé d'ajouter toutes les informations, cliquez sur le bouton **Créer un changement**.

## Annexe

:::caution
Cliquez sur **Annuler** rejette toutes les informations non enregistrées.
:::

:::tip
Si ce modificateur est l'image principale affichée, n'oubliez pas de basculer **Image principale** à **Oui** après le téléchargement.
:::

:::tip
Vous pouvez ajouter plusieurs images en cliquant sur **Ajouter une autre image**.
:::
