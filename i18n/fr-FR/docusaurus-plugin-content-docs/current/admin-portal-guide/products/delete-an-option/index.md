# Supprimer une option

## Ce que ce guide couvre

Supprime un groupe d'options du système lorsqu'il n'est plus nécessaire.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Options**.

![Étape 2](./img/step-2.png)

**Step 3:** Recherchez l'option que vous souhaitez supprimer en entrant le nom d'option, le code d'option ou l'étiquette de catalogue dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté de l'option, puis sélectionnez **Supprimer**.

**Step 5:** Un mode de confirmation apparaîtra montrant toutes les zones du système où cette option est utilisée. Passez en revue attentivement pour vous assurer que vous supprimez l'option correcte.

**Step 6:** Cliquez sur le bouton rouge **Supprimer** pour supprimer définitivement l'option.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
La suppression d'une option est permanente et ne peut être annulée. L'option sera retirée de tous les produits qui l'utilisent.
:::

:::tip
Vous pouvez rechercher des options par nom d'option, code d'option ou balise de catalogue pour trouver rapidement l'élément que vous voulez supprimer.
:::

:::caution
Cliquez sur **Annuler** si vous ne voulez pas procéder à la suppression.
:::
