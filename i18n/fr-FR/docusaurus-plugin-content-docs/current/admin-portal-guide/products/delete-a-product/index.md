# Supprimer un produit

## Ce que ce guide couvre

Supprime définitivement un produit du système lorsqu'il est arrêté ou créé par erreur.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez le produit que vous voulez supprimer. Vous pouvez rechercher par nom de produit ou code de produit.

**Step 3:** Cliquez sur le menu à trois points à côté du nom du produit, puis sélectionnez **Supprimer**.

![Étape 2](./img/step-2.png)

**Step 4:** Un modal de confirmation apparaîtra montrant toutes les zones du système où ce produit est utilisé. Vérifiez attentivement ceci pour vous assurer que vous supprimez l'élément correct.

**Step 5:** Cliquez sur le bouton rouge **Supprimer** pour supprimer définitivement le produit.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
La suppression d'un produit est permanente et ne peut être annulée. Le produit sera retiré de tous les menus et zones associées du catalogue.
:::

:::tip
Vous pouvez rechercher des produits par nom de produit ou code de produit pour trouver rapidement l'élément que vous voulez supprimer.
:::

:::caution
Cliquez sur **Annuler** si vous ne voulez pas procéder à la suppression.
:::
