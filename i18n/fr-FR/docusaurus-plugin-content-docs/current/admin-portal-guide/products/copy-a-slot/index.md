# Copier une Fente

## Ce que ce guide couvre

Dupliquer une fente pour accélérer la configuration du produit lors de la création de structures de personnalisation similaires.

## Étapes

**Step 1:** Naviguez dans la section **Produits** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Slots**.

**Step 3:** Recherchez la fente que vous voulez copier en entrant le nom de la fente, le code de fente ou l'étiquette dans le champ de recherche.

**Step 4:** Cliquez sur le menu à trois points à côté de la fente, puis sélectionnez **Copier**.

![Étape 2](./img/step-2.png)

**Step 5:** Le formulaire de copie apparaîtra avec les informations de la fente originale. Mettre à jour les champs au besoin. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| ** Code du lot** * | Identifiant unique pour la nouvelle fente | Doit être différent de l'original |
| ** Nom du lot** * | Décrit ce que la personnalisation cette fente offre | Peut être le même ou personnalisé |
| ** Quantité minimale** | Sélection minimale du modificateur requise | 0 = facultatif |
| ** Quantité maximale** | Sélections maximales de modificateur autorisées | Laisser blanc pour illimité |

**Step 6:** Examinez toutes les sections (information de base, modifications, poids) et apportez les modifications nécessaires.

**Step 7:** Lorsque vous avez examiné vos modifications et êtes prêt, cliquez sur **Créer**.

![Étape 3](./img/step-3.png)

## Annexe

:::caution
Le code **Slot** doit être unique. Vous ne pouvez pas utiliser le même code que la fente d'origine.
:::

:::tip
Vous pouvez rechercher des fentes par nom de fente, code de fente, ou étiquette pour trouver rapidement l'article que vous voulez copier.
:::

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::
