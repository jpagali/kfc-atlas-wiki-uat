# Modifier une promotion

## Ce que ce guide couvre

Mettre à jour une configuration de promotion existante, comme les dates de validité, les valeurs de réduction ou les paramètres d'exigence et d'effet.

## Étapes

**Step 1:** Naviguez dans la section **Promotions** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez la promotion que vous souhaitez modifier en parcourant la table ou en utilisant la barre de recherche. Cliquez sur le nom **promotion** pour ouvrir l'assistant de promotion.

![Étape 2](./img/step-2.png)

![Étape 3](./img/step-3.png)

**Step 3:** Mettre à jour tous les détails de promotion (nom, nom d'affichage, description, type de flux, exigences, effets ou période de validité).

![Étape 4](./img/step-4.png)

**Step 4:** Lorsque vous avez terminé d'apporter des modifications, cliquez sur le bouton **Enregistrer** pour appliquer vos mises à jour.

:::note :
L'assistant de promotion vous guide dans chaque section. Vous pouvez cliquer sur n'importe quel numéro d'étape pour sauter à cette section ou procéder séquentiellement.
:::
