# Créer un code sérialisé

## Ce que ce guide couvre

Génére des codes promotionnels uniques à usage unique liés à une promotion, utilisés pour les campagnes de rachat suivies où chaque client reçoit un code distinct.

## Étapes

**Step 1:** Naviguez dans la section **Promotions** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez la promotion pour laquelle vous voulez créer des codes sériels. Cliquez sur le bouton de menu **action** (trois points), puis sélectionnez **Serialized Code**.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **+ Créer un nouveau code en série** (ou le bouton équivalent affiché sur votre écran).

![Étape 3](./img/step-3.png)

**Step 4:** Remplissez le formulaire de code sérialisé avec les détails suivants:

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Préfixe** | Texte optionnel à ajouter au début de chaque code | Par exemple, "KFC" et "BOGO2024". Laisser en blanc si aucun préfixe n'est nécessaire. |
| ** Quantité** | Nombre de codes à générer | Par exemple, 100, 50000. Détermine le nombre de codes uniques créés. |
| **Date d'expiration** | Lorsque les codes expirent | Sélectionnez la date après laquelle les codes ne peuvent pas être échangés. |

![Étape 4](./img/step-4.png)

**Step 5:** Cliquez sur **Générer des codes** pour créer et ajouter les codes à la liste des codes sérialisés.

:::note :
Les codes sérialisés sont des codes de rachat uniques. Chaque code ne peut être racheté qu'une seule fois. Une fois généré, les codes sont ajoutés à la promotion et prêts à être distribués aux clients.
:::
