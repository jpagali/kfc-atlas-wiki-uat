# Copier l'ID de promotion

## Ce que ce guide couvre

Copie un ID unique de promotion dans votre presse-papiers pour une utilisation dans les tickets de support, les intégrations de système ou les renvois avec d'autres systèmes.

## Étapes

**Step 1:** Naviguez dans la section **Promotions** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez la promotion pour laquelle vous voulez copier l'ID. Cliquez sur le bouton de menu **action** (trois points), puis sélectionnez **Copier ID**.

![Étape 2](./img/step-2.png)

**Step 3:** L'ID promotionnel a été copié dans votre presse-papiers. Vous pouvez maintenant le coller dans des tickets de support, des formulaires d'intégration ou d'autres systèmes au besoin.

:::note :
L'ID promotionnel est un identifiant unique attribué par Atlas. Les équipes techniques utilisent cet ID pour référencer la promotion dans les intégrations, le débogage et les enregistrements système.
:::
