# Apparaît dans le résultat de recherche (Déployer ou désactiver)

## Ce que ce guide couvre

Contrôle si un magasin apparaît dans les résultats de recherche orientés vers le client, permettant aux opérateurs de cacher un emplacement sans l'enlever complètement.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le nom du magasin **** (ou tout hyperlien bleu) pour afficher les détails du magasin, ou cliquez sur le menu ** à trois points** (•••) et sélectionnez **Edit**.

**Step 4:** Localiser le **Appareil dans le magasin Résultats de recherche** basculer et le mettre à l'état désiré:
- **Oui**: Store apparaît dans les résultats de la recherche client
- **No**: Store est caché des résultats de la recherche client mais n'est pas supprimé

**Step 5:** Cliquez sur le bouton **Enregistrer** pour appliquer la modification.

![Étape 3](./img/step-3.png)

:::tip
Utiliser **Non** pour cacher temporairement un emplacement aux clients tout en le conservant dans Atlas à des fins administratives (p. ex., pendant les rénovations, les fermetures saisonnières ou lors de la préparation d'un nouvel emplacement avant le lancement).
:::

:::caution
Cliquer sur **Annuler** à tout moment rejette votre changement.
:::
