# Modifier la liste des correctifs

## Ce que ce guide couvre

Gère la liste ordonnée des correctifs appliqués au menu d'un magasin, permettant aux opérateurs d'ajouter, de supprimer ou de réorganiser des correctifs qui dépassent le menu de base (p. ex., ajustements de prix, changements de disponibilité des articles).

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le menu ** à trois points** (••) pour ouvrir le menu des options.

**Step 4:** Cliquez sur **Menus** dans le menu déroulant. Ceci affiche les menus assignés à ce magasin.

![Étape 2](./img/step-2.png)

**Step 5:** Localisez le canal pour lequel vous voulez modifier les correctifs, et cliquez sur le bouton **plus de menu** (-) sur cette ligne.

**Step 6:** Cliquez sur **Modifier la liste des lots** dans le menu Options.

**Step 7:** Ajouter, supprimer ou réordonner des correctifs :

| Décision | Comment le faire |
|--------|------------|
| **Ajouter un lot** | Cliquez sur le bouton **+ Ajouter le patch** et recherchez le patch par nom |
| **Supprimer le lot** | Cliquez sur l'icône **delete/supprimer** à côté du nom du correctif |
| **Recommander les correctifs** | Faites glisser et déposez les correctifs pour modifier leur ordre (top = appliqué en premier) |

![Étape 3](./img/step-3.png)

**Step 8:** Consultez l'ordre final de la liste des patchs. Les patchs sont appliqués de haut en bas — le patch le plus haut est appliqué en premier.

**Step 9:** Cliquez sur **Enregistrer** pour appliquer les modifications de la liste des correctifs.

![Étape 4](./img/step-4.png)

:::note :
**Qu'est-ce que les patchs?** Les patchs sont des surcharges de menu spécifiques à un magasin. Ils peuvent ajuster le prix des articles, ajouter/supprimer des articles, ou changer la disponibilité pour ce magasin uniquement, superposé au menu de base.
:::

:::tip
**Patch order matters:** Les patchs sont appliqués de haut en bas. En cas de conflit de patchs, la première dans la liste a priorité.
:::

:::caution
Après l'édition des correctifs, publiez le menu pour faire les modifications en direct aux clients.
:::
