# Créer un magasin

## Ce que ce guide couvre

Enregistre un nouveau magasin dans Atlas avec les détails requis, le rendant disponible pour l'attribution de menus et la commande numérique.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur le bouton **+ Créer un nouveau magasin** dans le coin supérieur droit de l'écran.

![Étape 2](./img/step-2.png)

**Step 3:** Remplissez le formulaire de détails du magasin. Les champs marqués d'un * sont obligatoires.

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Nom du magasin** * | Affichage complet du nom du magasin | Par exemple, KFC George Street Sydney |
| ** Numéro du magasin** * | Identifiant numérique unique attribué par les opérations de marché | Doit correspondre au numéro de magasin attribué par Byte POS, ou à l'identificateur de magasin cartographié utilisé par Byte Connect pour les marchés de points de vente non attribués par Byte |
| **Code de la franchise** * | Code alphanumérique identifiant le franchisé | Fourni par votre gestionnaire régional |
| **Délai** | Le magasin local fuseau horaire | Requis pour l'article snooze et la précision future de la commande |

**Step 4:** Une fois que tous les champs requis sont remplis, le bouton **Save** devient actif. Cliquez sur **Enregistrer** pour créer le magasin.

![Étape 3](./img/step-3.png)

:::tip
Avant de créer un nouveau magasin, recherchez la liste des magasins par nom, numéro ou code de franchise pour confirmer qu'il n'existe pas déjà.
:::

:::note Octet POS Cavat
Si le marché n'utilise pas Byte POS, ne présumez pas qu' Byte Commerce se connectera directement au marché POS à partir du seul disque du magasin. **Byte Connect** doit être embarqué dans le cadre de Byte Commerce à bord.
:::

:::caution
Cliquez sur **Annuler** à tout moment rejette toutes les informations non enregistrées.
:::
