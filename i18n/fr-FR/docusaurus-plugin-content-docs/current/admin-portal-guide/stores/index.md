# Magasins

Cette section couvre toutes les tâches liées aux magasins dans le portail Byte Commerce Admin, y compris la configuration des magasins, la gestion des menus, la visibilité des articles et la gestion des dons.

:::note Octet POS Cavat
Les tâches de stockage qui mentionnent **POS**, **authentification des appareils** ou **numéros de magasin correspondant aux enregistrements POS** supposent que le marché utilise **Byte POS**.

Si le marché n'utilise pas Byte POS, les équipes devraient traiter **Byte Connect** comme faisant partie de Byte Commerce à bord afin que le chemin d'intégration du magasin soit **Byte Commerce -> Byte Connect -> POS**.
:::
