# Publier un menu

## Ce que ce guide couvre

Publie le menu assigné aux canaux de commande en direct du magasin, rendant les changements de menu visibles pour les clients.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur le bouton **Publier le menu** ou l'onglet. Cela ouvre le flux de travail de publication.

![Étape 2](./img/step-2.png)

**Step 3:** Sélectionnez le(s) **store(s)** pour lequel vous souhaitez publier des menus. Vous pouvez rechercher par nom de magasin, numéro ou code si la liste est longue.

![Étape 3](./img/step-3.png)

**Step 4:** Sélectionnez les **canaux** que vous souhaitez publier (p. ex. Digital, Kiosque, In-Store). Vous pouvez publier sur plusieurs canaux à la fois.

![Étape 4](./img/step-4.png)

**Step 5:** Examiner les modifications à publier. Cliquez sur **Review Changes** si vous devez vérifier ce qui sera envoyé aux clients.

**Step 6:** Une fois que vous avez examiné et confirmé, cliquez sur le bouton **Publier** pour appuyer sur le menu en direct vers les canaux sélectionnés.

![Étape 5](./img/step-5.png)

:::tip
**Before publishing:** Vérifiez que le bon menu est attribué à chaque canal dans les paramètres du menu du magasin. Utilisation[Afficher le menu d'un magasin](/docs/admin-portal-guide/stores/view-a-stores-menu/)pour confirmer les missions.
:::

:::caution
Publier un menu rend immédiatement les changements visibles pour les clients. Consultez tous les changements avant de cliquer sur Publier.
:::
