# Publier le menu

## Ce que ce guide couvre

Déclenche un menu publie directement depuis la vue Menus du magasin, poussant la configuration actuelle du menu en direct vers les canaux de commande.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le menu ** à trois points** (••) pour ouvrir le menu des options.

![Étape 3](./img/step-3.png)

**Step 4:** Cliquez sur **Menus** dans le menu déroulant. Ceci affiche les menus assignés à ce magasin par canal.

**Step 5:** Localisez la chaîne que vous voulez publier, et cliquez sur le bouton **plus de menu** (-) sur cette ligne.

**Step 6:** Cliquez sur **Publier le menu** dans le menu des options.

**Step 7:** Passez en revue la configuration du menu et les modifications en cours.

**Step 8:** Cliquez sur le bouton **Publier** pour appuyer sur le menu en direct pour les clients.

![Étape 4](./img/step-4.png)

:::tip
**When to use:** Utilisez cette méthode lors de la publication à partir de la vue menu d'un magasin. Utilisation[Publier un menu](/docs/admin-portal-guide/stores/publish-a-menu/)si vous avez besoin de publier des menus sur plusieurs magasins ou canaux à la fois.
:::

:::caution
Publier un menu rend immédiatement les changements visibles pour les clients. Consultez tous les changements avant de cliquer sur Publier.
:::
