# Autoriser les commandes futures (détourner ou désactiver)

## Ce que ce guide couvre

Permet ou désactive la capacité d'un magasin d'accepter les commandes passées à l'avance, pertinentes pour les capacités de précommande ou de livraison programmée.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le nom du magasin **** (ou tout hyperlien bleu) pour afficher les détails du magasin, ou cliquez sur le menu ** à trois points** (•••) et sélectionnez **Edit**.

**Step 4:** Localiser le **Autorise les ordres futurs** et le mettre à l'état désiré:
- **Oui**: Les clients peuvent passer des commandes à l'avance pour la livraison programmée ou le ramassage
- **Non**: Les clients ne peuvent passer des commandes que pour une exécution immédiate

**Step 5:** Cliquez sur le bouton **Enregistrer** pour appliquer la modification.

![Étape 3](./img/step-3.png)

:::note :
**Future Orders Requirement:** Ce réglage exige que le magasin ait configuré un canal de commande supporté. Confirmez avec votre gestionnaire régional si votre magasin supporte cette capacité.
:::

:::caution
Cliquer sur **Annuler** à tout moment rejette votre changement.
:::
