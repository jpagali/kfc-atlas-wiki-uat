# Afficher un magasin Publier l'historique / Copier le menu Publier l'ID de travail

## Ce que ce guide couvre

Affiche un journal de tous les travaux de publication de menu pour un magasin, y compris l'état et les horodatages. Vous pouvez afficher les détails de chaque travail de publication et copier le Menu Publier l'ID de travail à des fins de support.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Cliquez sur l'onglet **Histoire des publications** en haut de la page Boutiques.

![Étape 2](./img/step-2.png)

**Step 3:** Le tableau d'historique de publication montre:
- **Menu** — Nom du menu publié
- **Channel** — Chaîne de commande (numérique, Kiosque, In-Store, etc.)
- **Statut** — État de publication (succès, en attente, échec, etc.)
- **Timestamp** — Date et heure de la publication

Cliquez sur n'importe quelle ligne pour ouvrir tous les détails, y compris le **Menu Publish Job ID**.

![Étape 3](./img/step-3.png)

**Step 4:** Pour copier le **Menu Publier l'ID de travail** (utile pour contacter le support), cliquez sur l'icône **copie** à côté du champ ID de travail dans le panneau de détails.

:::tip
**When to use Menu Publish Job ID:** Lorsque vous contactez Atlas Support au sujet d'un problème de publication, incluez le Menu Publier l'ID de travail pour aider l'équipe de support à localiser rapidement le travail spécifique et diagnostiquer le problème.
:::
