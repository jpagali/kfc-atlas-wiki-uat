# Afficher les promotions

## Ce que ce guide couvre

Affiche toutes les promotions actuellement assignées à un magasin, vous permettant de rechercher et de filtrer par nom de promotion ou code de rachat.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le menu ** à trois points** (••) pour ouvrir le menu des options.

**Step 4:** Cliquez sur **Promotions** dans le menu déroulant. Ceci affiche toutes les promotions liées au magasin sélectionné.

![Étape 3](./img/step-3.png)

**Step 5:** Utilisez les options de recherche et de filtrage pour trouver des promotions spécifiques:
- Recherche par **Nom de promotion** pour trouver des promotions par nom d'affichage
- Rechercher par **Code de révocation** pour trouver des promotions par leur code de rachat ou de promo

Le tableau des promotions indique :
- **Nom de la promotion** — Afficher le nom de la promotion
- **Code de révocation** — Code que les clients entrent pour racheter la promotion
- **Situation** — Situation active, inactive ou autre
- **Valide De / À** — Dates de début et de fin de promotion

:::tip
Utilisez des filtres pour réduire la liste de promotion si votre magasin a de nombreuses promotions actives. Cela vous aide à vérifier rapidement quelles promotions sont en cours.
:::
