# Accepter les commandes en ligne (déplacer ou désactiver)

## Ce que ce guide couvre

Indique si un magasin accepte les commandes en ligne par les canaux numériques — utilisé pour désactiver rapidement la commande pendant les fermetures ou les problèmes opérationnels.

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le nom du magasin **** (ou tout hyperlien bleu) pour afficher les détails du magasin, ou cliquez sur le menu ** à trois points** (•••) et sélectionnez **Edit**.

**Step 4:** Localisez le **Accepter les commandes en ligne** basculer et le mettre à l'état désiré:
- **Oui**: Store accepte les commandes en ligne via les canaux numériques
- **Non**: Store n'accepte pas les commandes en ligne (les commandes sont temporairement désactivées)

**Step 5:** Cliquez sur le bouton **Enregistrer** pour appliquer la modification.

![Étape 3](./img/step-3.png)

:::tip
Utilisez **Non** pour désactiver rapidement la commande pendant les fermetures de magasin, la maintenance du système ou les pénuries de personnel sans avoir besoin de modifier d'autres paramètres de magasin.
:::

:::caution
Cliquer sur **Annuler** à tout moment rejette votre changement.
:::
