# Affichage d'un menu Store

## Ce que ce guide couvre

Affiche tous les menus assignés à un magasin, montrant quel menu est actif sur chaque canal de commande (numérique, Kiosque, In-Store, etc.).

## Étapes

**Step 1:** Naviguez dans la section **Stores** en utilisant le menu de navigation de gauche.

**Step 2:** Recherchez le magasin par **Nom**, **Numéro de magasin** ou **Code de franchise** à l'aide de la boîte de recherche.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous trouvez le magasin, cliquez sur le menu ** à trois points** (••) pour ouvrir le menu des options.

**Step 4:** Cliquez sur **Menus** dans le menu déroulant. Ceci affiche tous les menus assignés à ce magasin, organisés par canal.

![Étape 3](./img/step-3.png)

Le tableau de menu montre :
- **Channel** — Le canal de commande (par exemple, Digital, Kiosk, In-Store)
- **Nom de famille** — Le nom du menu assigné
- **Statut** — État actuel de la publication (publication, rédaction, etc.)

:::tip
De cette vue, vous pouvez également assigner de nouveaux menus, publier des menus, éditer des listes de patchs ou transférer des listes de patchs. Utilisez les boutons de menu ou le menu à trois points de chaque ligne de canal pour accéder à ces options.
:::
