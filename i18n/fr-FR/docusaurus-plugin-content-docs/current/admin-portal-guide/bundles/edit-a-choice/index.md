# Modifier un choix

## Ce que ce guide couvre

Mettre à jour le nom, les produits, les variantes ou les paramètres de quantité min/max d'un choix existant.

## Étapes

**Step 1:** Naviguez dans la section **Bundles** en utilisant le menu de navigation à gauche.

**Step 2:** Cliquez sur l'onglet **Choices** en haut de l'écran Bundles.

![Étape 2](./img/step-2.png)

**Step 3:** Trouvez le choix que vous souhaitez modifier. Vous pouvez rechercher par Nom de choix, Code de choix, Nom d'affichage de choix ou Étiquettes de catalogue.

**Step 4:** Cliquez sur le bouton *** (menu à trois points) dans la même ligne que le choix, puis sélectionnez **Edit**.

![Étape 3](./img/step-3.png)

**Step 5:** Mettre à jour l'un des champs suivants :
- ** Nom de choix** ou ** Nom d'affichage de choix**
- ** Quantité minimale** ou ** Quantité maximale**
- **Produits**: Ajouter ou supprimer les produits et leurs variantes

![Étape 4](./img/step-4.png)

**Step 6:** Cliquez sur **Enregistrer** pour effectuer vos modifications.

:::tip
Vous pouvez ajouter ou supprimer des produits et des variantes à tout moment sans affecter les paquets qui utilisent ce choix.
:::
