# Copier un ensemble

## Ce que ce guide couvre

Dupliquer un paquet existant pour créer rapidement des offres similaires avec les mêmes choix et configuration.

## Étapes

**Step 1:** Naviguez dans la section **Bundles** en utilisant le menu de navigation à gauche.

**Step 2:** Trouvez le paquet que vous voulez copier en recherchant par nom de groupe, code de groupe, étiquettes de catalogue ou étiquettes promo.

**Step 3:** Cliquez sur le bouton *** (menu à trois points) dans la même ligne que le paquet, puis sélectionnez **Copier**.

![Étape 2](./img/step-2.png)

**Step 4:** Un formulaire s'ouvre pour créer le nouveau paquet. Entrez un code unique** (obligatoire).

![Étape 3](./img/step-3.png)

**Step 5:** Ajustez le nom **Bundle** si nécessaire. Par défaut, ce sera "Copier de [Nom original]" - vous pouvez changer cela en quelque chose de plus descriptif.

**Step 6:** Consultez tous les autres champs (Description, Choix, etc.). Le paquet copié comprend tous les choix de l'original. Modifier les champs au besoin.

**Step 7:** Cliquez sur **Créer** pour terminer la copie du paquet.

:::tip
Le paquet copié hérite de tous les choix et configurations de l'original. Vous pouvez modifier l'un de ces détails avant ou après la création.
:::
