# Modifier un bundle

## Ce que ce guide couvre

Mettre à jour le nom, le code, les choix, les prix ou la description d'un paquet existant.

## Étapes

**Step 1:** Naviguez dans la section **Bundles** en utilisant le menu de navigation à gauche.

**Step 2:** Trouvez le paquet que vous voulez modifier. Vous pouvez rechercher à l'aide de la barre de recherche par nom de groupe, code de groupe, étiquettes de catalogue ou étiquettes promo.

**Step 3:** Cliquez sur le bouton *** (menu à trois points) dans la même ligne que le paquet, puis sélectionnez **Edit**.

![Étape 2](./img/step-2.png)

**Step 4:** Le même formulaire multi-page de création de faisceau s'ouvre. Mettre à jour les champs & #160;:
- Cliquez sur les en-têtes de section bleue pour sauter entre les pages (Info de base, Choix, Revue)
- Cliquez sur **Suivant** pour passer à travers les pages séquentiellement

![Étape 3](./img/step-3.png)

**Step 5:** Après toutes les modifications, cliquez sur **Enregistrer** pour effectuer les mises à jour.

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::
