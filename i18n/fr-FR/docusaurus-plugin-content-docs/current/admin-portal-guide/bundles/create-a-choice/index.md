# Créer un choix

## Ce que ce guide couvre

Crée une fente de sélection à l'intérieur d'un paquet (par exemple, « Choisissez votre côté » ou « Choisissez votre boisson »), permettant aux clients de personnaliser leur combo en sélectionnant parmi les produits disponibles.

## Étapes

**Step 1:** Naviguez dans la section **Bundles** en utilisant le menu de navigation à gauche.

**Step 2:** Cliquez sur l'onglet **Choices** en haut de l'écran Bundles.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur le bouton **+ Créer un nouveau choix**.

![Étape 3](./img/step-3.png)

**Step 4:** Remplissez les détails du choix :

| Champ | Quoi entrer | Annexe |
|-------|--------------|-------|
| **Code de choix** * | identificateur unique | Utilisez des lettres majuscules, des chiffres et des tirets, par exemple:`CHOICE-SIDE` |
| **Nom de choix** * | Étiquette montrée aux clients | Par exemple, "Choisissez votre côté", "Choisissez votre verre" |
| ** Nom de l'affichage des chèques** | Nom plus court pour un espace limité | Par défaut pour choisir le nom si laissé vide |
| ** Quantité minimale** | Sélection minimale requise | Réglé sur`0`de rendre le choix facultatif |
| ** Quantité maximale** | Sélection maximale autorisée | Par exemple,`1`pour un seul choix,`2`pour le double choix |

**Step 5:** Ajoutez des produits et des variantes à ce choix :

- Cliquez sur le champ **Produits** et recherchez le produit que vous souhaitez inclure
- Sélectionnez le produit — une liste des variantes disponibles apparaît
- Vérifiez les variantes que vous voulez que les clients puissent sélectionner (par exemple, "Frises régulières", "Grandes frites")
- Cliquez sur **Ajouter** pour inclure ce produit

**Step 6:** Répétez l'étape 5 pour tout produit ou variante supplémentaire que vous souhaitez dans ce choix.

**Step 7:** Cliquez sur **Créer un choix** pour enregistrer le choix.

![Étape 4](./img/step-4.png)

:::tip
Vous pouvez rechercher des choix plus tard par Choice Name, Choice Code, Choice Display Name ou Catalog Tags pour trouver rapidement ce dont vous avez besoin.
:::
