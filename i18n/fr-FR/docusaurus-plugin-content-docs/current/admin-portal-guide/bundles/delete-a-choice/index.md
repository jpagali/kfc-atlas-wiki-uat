# Supprimer un choix

## Ce que ce guide couvre

Enlevez définitivement un choix et ses produits/variants associés du système.

## Étapes

**Step 1:** Naviguez dans la section **Bundles** en utilisant le menu de navigation à gauche.

**Step 2:** Cliquez sur l'onglet **Choices** en haut de l'écran Bundles.

![Étape 2](./img/step-2.png)

**Step 3:** Trouvez le choix que vous souhaitez supprimer par recherche par Nom de choix, Code de choix, Nom d'affichage de choix, ou Tags de catalogue.

**Step 4:** Cliquez sur le bouton *** (menu à trois points) dans la même ligne que le choix, puis sélectionnez **Supprimer**.

![Étape 3](./img/step-3.png)

**Step 5:** Un mode de confirmation apparaît. Cliquez sur **Confirmer** pour supprimer le choix, ou cliquez en dehors du mode ou **Annuler** pour le garder.

![Étape 4](./img/step-4.png)

:::caution
Cette action est permanente. Supprimer un choix :
- Supprimer de tous les paquets qui l'utilisent
- Supprimer toutes les valeurs de choix et les variantes correspondantes
- Supprimer tout prix associé à ce choix

Vous ne pouvez pas annuler cette action. Les ensembles qui dépendent de ce choix ne fonctionneront plus correctement jusqu'à ce que vous ajoutiez un choix de remplacement.
:::
