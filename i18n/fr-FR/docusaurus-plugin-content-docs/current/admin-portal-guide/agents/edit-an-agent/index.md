# Modifier un agent

## Ce que ce guide couvre

Mettre à jour les détails du compte, les rôles ou les permissions d'accès d'un agent existant.

## Étapes

**Step 1:** Naviguez dans la section **Agents** en utilisant le menu de navigation de gauche.

**Step 2:** Cherchez l'agent que vous voulez modifier. Vous pouvez rechercher par :
- Prénom
- Nom de famille
- Adresse électronique
- Nom d'utilisateur

Entrez l'un de ces détails et la liste sera filtrée.

![Étape 2](./img/step-2.png)

**Step 3:** Une fois que vous avez trouvé l'agent, cliquez sur le bouton **=** (menu à trois points) dans la même ligne, puis sélectionnez **Edit**.

![Étape 3](./img/step-3.png)

**Step 4:** Le formulaire d'édition s'ouvre avec les mêmes sections que la création d'agent :
- **Information sur l'agent** (Page 1) : Prénom, Nom, Email, Nom d'utilisateur
- **Rôles et autorisations** (Page 2) : Cochez les cases Rôle
- **Examen** (Page 3) : Résumé de tous les changements

Cliquez sur les en-têtes de la section bleue pour sauter entre les pages, ou cliquez sur **Suivant** pour les déplacer séquentiellement.

**Step 5:** Faire les changements nécessaires aux détails ou aux rôles de l'agent. Vous pouvez :
- Mettre à jour Prénom, nom ou adresse électronique
- Ajouter ou supprimer des rôles en cochant ou en décochant des cases
- **Note** : Le nom d'utilisateur ne peut pas être modifié après la création

**Step 6:** Une fois que vous avez fait toutes les modifications, cliquez sur **Enregistrer** pour les engager.

![Étape 4](./img/step-4.png)

**Step 7:** Après l'enregistrement, cliquez sur **Quit** pour quitter l'écran d'édition et retourner à la liste des agents.

:::note :
Les rôles et permissions actuels de l'agent sont affichés en haut pour une référence rapide. Vous pouvez supprimer tous les rôles si nécessaire, mais vous ne pouvez pas supprimer un compte d'agent entièrement — seulement enlever leurs permissions.
:::

:::caution
Cliquez sur **Annuler** rejette tous les changements non enregistrés.
:::
