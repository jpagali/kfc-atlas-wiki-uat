# Agents

Cette section couvre toutes les tâches de création et de gestion d'agents utilisateurs (membres du personnel ayant accès au portail) dans le portail Byte Commerce Admin.

## Thèmes

- [Créer un Agent](/docs/admin-portal-guide/agents/create-an-agent/)— Configurer un nouveau compte utilisateur avec des rôles et des permissions spécifiques.
- [Modifier un agent](/docs/admin-portal-guide/agents/edit-an-agent/)— Mettre à jour les détails de compte, les rôles ou les autorisations d'un agent existant.
