# Supprimer un groupe Store

## Ce que ce guide couvre

Enlève définitivement un groupe de magasins du système, ainsi que toutes les promotions et les règles fiscales connexes.

## Étapes

**Step 1:** Naviguez vers la section **Groupes de magasins** en utilisant le menu de navigation de gauche.

**Step 2:** Trouvez le groupe de magasins que vous voulez supprimer en parcourant la table ou en utilisant la barre de recherche. Cliquez sur le bouton de menu **action** (trois points) à côté du nom du groupe de magasins.

![Étape 2](./img/step-2.png)

**Step 3:** Cliquez sur **Supprimer**.

**Step 4:** Une boîte de dialogue de confirmation apparaîtra. Passez en revue le message d'avertissement et cliquez sur le bouton **Delete Store Group** pour confirmer la suppression permanente.

![Étape 3](./img/step-3.png)

:::caution
**Cette action est permanente et ne peut être annulée.** La suppression d'un groupe de magasins :
- Retirer le groupe de magasins du système
- Déconnectez toutes les promotions attribuées à ce groupe de magasins
- Déconnectez toutes les règles fiscales associées à ce groupe de magasins
- Les magasins membres eux-mêmes ne sont pas supprimés, seulement leur regroupement et leurs missions

:::tip
Si vous devez conserver la configuration du groupe de magasins pour une utilisation future, considérez **copier** avant de supprimer.
:::
