# Afficher les magasins dans un groupe de magasins

## Ce que ce guide couvre

Liste tous les magasins appartenant actuellement à un groupe de magasins à des fins de visibilité et de vérification.

## Étapes

**Step 1:** Naviguez vers la section **Groupes de magasins** en utilisant le menu de navigation de gauche.

**Step 2:** Parcourez la table Groupes Store pour trouver le groupe que vous voulez voir, ou utilisez la barre de recherche pour le localiser. Cliquez sur le nom du groupe **store** pour voir ses détails.

![Étape 2](./img/step-2.png)

**Step 3:** Une vue détaillée sera ouverte montrant:

- **Nom du groupe* — Le nom du groupe sélectionné
- **Store Group Tags** — Toute balise attribuée à ce groupe
- **Tableau des stocks** — Liste complète de tous les magasins de ce groupe

Le tableau affiche :
- **Numéro de magasin** — Identifiant unique pour chaque magasin
- **Nom du magasin** — Nom du magasin
- **Code de franchise** — Identifiant de franchise, le cas échéant
- ** Règles fiscales** — Si ce groupe a des règles fiscales associées

![Étape 3](./img/step-3.png)

:::note :
La table des magasins énumère tous les magasins appartenant actuellement à ce groupe de magasins. Pour modifier l'adhésion au magasin, voir[Modifier un groupe de magasins](/docs/admin-portal-guide/store-groups/edit-a-store-group/).
:::
