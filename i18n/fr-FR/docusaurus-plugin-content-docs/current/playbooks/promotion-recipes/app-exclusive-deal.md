# Offre exclusive

## Objectif

20% de réduction disponible uniquement sur l'application mobile KFC.

## Exigences

| Domaine | Champ | Valeur |
|---|---|---|
| **Channel** | **Choisir les canaux** | `Mobile`, `iOS`, `Android` |

## Effets

| Domaine | Champ | Valeur |
|---|---|---|
| **Niveau de commande** | **Type d'effet de prix** | `Percent Off` |
| **Niveau de commande** | **En pourcentage** | `20` |
