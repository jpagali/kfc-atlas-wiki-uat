# Branche Deeplink

## Ce qu'il fait

Branch Deeplink étend les capacités de liaison de la plate-forme Atlas en utilisant le **Branche.io SDK**, une plate-forme de liaison et d'attribution profonde de tiers utilisée largement dans les applications mobiles.

Lorsque KFC Universal Links relie les utilisateurs au contenu de l'application, Branch ajoute une couche d'intelligence en haut : il suit *d'où* un utilisateur est venu, gère les liens profonds différés pour les utilisateurs qui n'ont pas encore l'application, et fournit des données d'attribution qui connecte l'activité marketing à l'application installs et commandes.

## Capacités clés

| Capacité | Désignation des marchandises |
|---|---|
| **Reliure profonde différée** | Un utilisateur clique sur un lien Branch, installe l'application, et est pris à l'écran correct de l'application au premier lancement — même si l'application n'était pas installée quand ils ont cliqué |
| **Attribution** | Links trace quelle campagne, canal, ou créative conduit l'installation ou la session |
| **Cross-plateforme routage** | Branche gère iOS, Android et les retombées web à partir d'un seul lien |
| **Gestion unifiée des liens** | Les équipes de marketing peuvent créer et gérer des liens profonds via le tableau de bord de la Branche sans intervention technique |

## La relation avec les liens universels

KFC Universal Links and Branch Deeplinks a des objectifs complémentaires :

|  | Lien universel | Branche Deeplink |
|---|---|---|
| **Utilisation primaire** | Acheminement direct (app installée) | Affectation + lien différé |
| **Applique non installée** | Retour sur le Web | Peut déclencher la redirection App Store + lien profond différé |
| **Commercialisation | Numéro | Oui |
| **Géré par** | Génie | Marketing + Ingénierie |
