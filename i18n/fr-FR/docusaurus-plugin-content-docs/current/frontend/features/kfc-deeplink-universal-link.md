# KFC Deeplink

## Ce qu'il fait

La fonction KFC Universal Link permet à toute URL KFC prise en charge de s'ouvrir directement à l'intérieur de l'application mobile KFC, ce qui amène l'utilisateur directement à l'écran correspondant de l'application plutôt qu'à une page de navigateur.

Lors de l'installation de l'application ****, il est possible de relier directement les utilisateurs à l'écran correspondant (p. ex. une page produit, une promo ou un restaurant). Lorsque l'application ** n'est pas installée**, le même lien revient gracieusement à l'expérience web.

Le résultat est une voie transparente et sans friction depuis n'importe quel point de contact — un courriel de marketing, un QR code in-restaurant, une annonce sociale, une notification de poussée — directement au bon endroit dans l'application.

## Pourquoi ça compte

Sans Liens Universels, chaque lien externe vers le contenu KFC s'ouvre dans un navigateur, même lorsque le client a installé l'application. Cela crée des frictions inutiles et des possibilités de conversion perdues.

Universal Links comble cette lacune. L'URL reste la même. La destination s'adapte à ce que le client a installé.

## Portée

| Domaine | Détail |
|---|---|
| ** Domaines pris en charge** | URLs appartenant à KFC configurées par marché |
| **Route** | Path-based — chaque carte du chemin URL vers une destination dans l'application |
| **Applique installée** | L'utilisateur est directement sur l'écran de l'application |
| **Applique non installée** | L'utilisateur est envoyé à l'expérience Web équivalente |
| **Plateformes** | iOS, Android |

## Points d'entrée

Les liens universels sont généralement déclenchés à partir de :

- Emails marketing et campagnes SMS
- Codes QR (dans le restaurant, emballage, extérieur)
- Publicités sur les médias sociaux et publications organiques
- Notifications poussées
- Flux de transfert Web-à-app
