# Enquêtes auprès des clients

Les sondages d'engagement des clients aident KFC à saisir les commentaires structurés des clients après une commande, une visite ou une interaction de soutien. Dans Atlas, cette capacité est généralement utilisée pour **KFC Listens** programmes de style qui mesurent la satisfaction, identifient les problèmes de service, et les informations de surface pour les équipes de marché.

## Ce que fait cette capacité

Les enquêtes sur l'engagement des clients permettent aux marchés:

- Inviter les clients à partager leurs commentaires après l'achat
- Mesurer la satisfaction entre les canaux, les magasins et les campagnes
- Réponses aux enquêtes sur les itinéraires dans une plate-forme de rapport pour l'analyse
- Mesures de suivi pour les détracteurs, les plaintes ou les processus de récupération

## Où il apparaît dans le parcours client

Les invitations peuvent être placées à différents points de contact selon la configuration du marché:

| Point de contact | Exemple de comportement |
|---|---|
| Confirmation de commande | Demander après la commande ou sur l'écran de confirmation |
| Réception par courriel | Inclure un sondage CTA dans le courriel de réception |
| Notification de poussée | Envoyer une invitation de suivi après l'exécution |
| Garantie en magasin | Code QR ou URL de réception conduisant à l'enquête |

## Pourquoi les marchés l'utilisent

KFC Écoutes et programmes similaires donnent aux opérateurs un signal direct des clients sur la qualité des aliments, la vitesse, la convivialité du personnel et la précision de la commande. Cette rétroaction aide les marchés :

- Performance du magasin de référence
- Détecter les points de douleur récurrents du client
- Valider les changements de campagne ou de produit
- Récupération de service prioritaire pour les réponses à faible teneur en soufre

## Modèle d'exploitation typique

| Domaine | Propriétaire |
|---|---|
| Conception et logique de l'enquête | Équipe CX / Insights |
| Règles de déclenchement d'invitation | Produit numérique ou équipe CRM |
| Rapports de réponse | Opérations sur le marché / Perspectives |
| Travaux de récupération de services | Assistance à la clientèle ou opérations de stockage |
