# Guide: Contenu de la catégorie

Les catégories sont la façon dont le menu KFC est organisé — regroupement des produits en sections telles que poulet, côtés, boissons, ou Desserts. Contentful vous permet de contrôler la **séquence** des produits dans une catégorie et de gérer **sous-catégories** pour une structure de menu plus granulaire.

:::info
Les catégories elles-mêmes sont créées et gérées dans **Byte Admin Portal**. C'est là que vous contrôlez le séquençage et la structure des sous-catégories. Les produits qui n'ont pas été configurés dans Contentful sont ** cachés dans le menu**.
:::

---

## Où vous le voyez au front

** Navigation de niveau** — Les catégories apparaissent comme des éléments de navigation de haut niveau dans le menu (p. ex. "Burgers", "Sides", "Boissons"). Cliquer sur une catégorie amène le client à une grille de produits.

**Sous-catégories** — Au sein d'une page de catégorie, les sous-catégories agissent comme des onglets ou des groupes de filtres (p. ex. dans "Chicken" vous pourriez avoir "Recette originale", "Zinger", "Wraps"). Les produits sont imbriqués dans chaque sous-catégorie.

**Séquence du produit** — L'ordre dans lequel les produits apparaissent dans une catégorie ou sous-catégorie est contrôlé dans Contentful. Si aucune séquence n'est définie dans Contentful, l'ordre revient à ce qui est défini dans Byte Admin Portal.

---

## Que configurer dans Contentful

### Produits de séquençage dans une catégorie

1. Une fois que la catégorie a été créée dans le portail Byte Admin avec des produits assignés, accédez à la rubrique **Catégorie** dans Contentful
2. Faites défiler vers le bas à la section **Menu Items**
3. Cliquez sur **Ajouter le contenu existant** et sélectionnez les produits ou les paquets que vous souhaitez inclure — cliquez sur **Sélectionner les entrées** pour confirmer
4. Faites glisser la poignée droite de chaque entrée pour les réorganiser
5. Cliquez sur **Modifications de publication**

### Création d'une sous-catégorie

1. Cliquez sur **Ajouter Entrée** (le bouton bleu sur la droite)
2. Rechercher et sélectionner **Sous-catégorie**
3. Remplissez tous les champs requis (nom, étiquettes d'affichage, etc.)
4. Cliquez sur **Publier**

### Ajouter une sous-catégorie à une catégorie

1. Naviguez vers la rubrique de catégorie pertinente
2. Cliquez sur **Contenu**, puis sur **Ajouter Entrée** et recherchez **Catégorie**
3. Sélectionnez le code de catégorie souhaité, remplissez les champs requis et ajoutez la sous-catégorie existante comme entrée liée
4. Cliquez sur **Publier**

### Sous-catégories de séquence

1. Naviguez vers l'entrée de catégorie souhaitée
2. Cliquez sur la poignée à trois points du côté droit de chaque entrée de sous-catégorie
3. Faites glisser et déposez à la position souhaitée
4. Cliquez sur **Modifications de publication**

### Produits de séquençage dans une sous-catégorie

1. Naviguer dans la catégorie qui contient la sous-catégorie
2. Cliquez sur la sous-catégorie souhaitée
3. Utiliser la poignée à trois points pour faire glisser et déposer les produits dans l'ordre
4. Cliquez sur **Publier**

## Cas d'utilisation

** Lancement de nouveaux produits** — Lorsqu'un nouvel élément est ajouté dans le portail Byte Admin, ajoutez-le et placez-le dans la bonne catégorie Contentful afin qu'il apparaisse au bon endroit sur le menu immédiatement.

**Restructuration du menu de saison** — Commandez des produits ou faites la promotion d'une sous-catégorie (p. ex., déplacez les «Spéciaux de vacances» en haut du menu) pour une période promotionnelle sans déploiement de code.

**Séquençage spécifique au marché** — Différents marchés peuvent vouloir des commandes différentes pour la même catégorie. L'espace Contentful de chaque marché peut être configuré indépendamment.

**Présentation de la sous-catégorie** — Ajouter un nouveau groupe de produits (p. ex., « Plant-Based » ou « Shareboxes ») comme sous-catégorie dans une catégorie existante sans soutien technique.
