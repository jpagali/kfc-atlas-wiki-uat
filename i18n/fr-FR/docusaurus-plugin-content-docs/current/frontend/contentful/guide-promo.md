# Guide: Contenu promotionnel

Le contenu promotionnel est utilisé pour afficher les coupons et les informations promotionnelles aux clients dans le panier et à la caisse. Il peut également faire apparaître une promotion en tant que carte de menu, ce qui la rend extensible dans le menu comme s'il s'agissait d'un produit régulier.

:::info
Une promotion ** doit être créée dans le portail Byte Admin d'abord** avant de pouvoir être configurée comme une entrée de contenu promotionnel dans Contentful.
:::

---

## Où vous le voyez au front

**Basket / Checkout** — Lorsqu'un client a un élément admissible dans son panier, une tuile promotionnelle apparaît montrant le nom d'affichage, la description et tout texte requis de la promotion (p. ex. « Dépenser 20 $ pour débloquer cette offre »).

** Carte Menu** — Les promotions peuvent en option apparaître comme une carte directement dans la grille de menu — avec une image, un titre et une description — afin que les clients puissent les découvrir et les ajouter à leur panier sans avoir besoin d'un code de coupon.

---

## Que configurer dans Contentful

1. Dans Contentful, accédez au type de contenu **Promotion** et ouvrez-le
2. Cliquez sur **Sélectionner à partir de Yum** pour voir la liste des promotions créées dans le portail Byte Admin
3. Sélectionnez la promotion à configurer

**Configurer le nom de l'affichage** — Le nom affiché au client sur l'interface utilisateur (par exemple « Zinger Box Deal »). Réglez ça par local.

**Configurer la description** — Contexte supplémentaire affiché à côté de la promotion (p. ex. « Comprend un Burger Zinger, des frites régulières et une boisson »). Par région.

**Configurer le texte des exigences** — Toutes les conditions que le client doit remplir pour recevoir la promotion (p. ex., « Ajouter un gros repas pour être admissible »). Par région.

**Configurer l'image de la carte de menu** — Ajouter une image pour faire surface à la promotion comme une carte de menu verticale dans la grille de menu. Télécharger ou sélectionner dans les médias existants.

5. Cliquez sur **Publier** — ou utilisez le menu déroulant pour programmer la promotion pour aller en direct à une date et une heure précises

---

## Cas d'utilisation

**Afficher un nouveau coupon à la caisse** — Configurez la copie et l'imagerie d'une promotion afin que les clients la voient clairement dans le panier lorsqu'ils sont admissibles à l'utiliser.

**Démarrage d'un menu visible** — Ajoutez une image de carte de menu pour faire surface à la promotion en tant qu'élément de navigation dans le menu, ce qui la rend accessible sans code promo.

** Campagnes limitées dans le temps** — Utilisez l'édition programmée pour activer et désactiver le contenu d'une promotion en synchronisation avec une fenêtre de campagne marketing.

**Promotions locales** — Définir différents noms et descriptions d'affichage par localité pour adapter la messagerie promotionnelle à la langue et au ton de chaque marché.
