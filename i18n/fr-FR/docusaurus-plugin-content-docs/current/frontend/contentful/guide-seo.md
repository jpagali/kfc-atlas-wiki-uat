# Guide: Mots clés et gestion

Champs SEO dans Contentful laissez les marchés contrôler comment les pages KFC apparaissent dans les résultats des moteurs de recherche — y compris les titres de page dans les onglets de navigateur, les descriptions dans les extraits de recherche Google, et si une page doit être indexée du tout.

---

## Où vous le voyez au front

Les champs SEO n'apparaissent pas visiblement aux clients sur la page elle-même — ils apparaissent en deux endroits:

**Onglet navigateur** — Le Meta Title est affiché comme étiquette de l'onglet lorsqu'un client a la page ouverte dans son navigateur.

**Google search results** — Le Meta Title et la Meta Description apparaissent dans l'extrait de recherche de Google quand une page KFC se classe pour une requête. Une description bien écrite augmente le clic à partir des résultats de recherche.

**Crawlability toggle** — Contrôle si Google est autorisé à indexer la page du tout. Lorsque défini à **Oui**, Google peut découvrir et classer la page. Régler à **Non** pour masquer la page des résultats de recherche (utile pour les pages non encore prêtes pour le trafic public).

---

## Que configurer dans Contentful

Les champs SEO sont intégrés dans plusieurs types de contenu — vous ne configurez pas SEO dans une section séparée autonome. Recherchez le bloc SEO à l'intérieur:

- **Page d'information** Type de contenu
- **Page d'accueil** Type de contenu
- **Catégorie** Type de contenu
- ** Page d'atterrissage des produits** *(sur feuille de route)*
- **Produit** *(sur feuille de route)*

Dans n'importe laquelle de ces entrées, localisez la section **SEO** et configurez :

**Titre de la méta** — Le titre affiché dans l'onglet navigateur et comme le titre dans les résultats de recherche Google. Gardez-le concis (50 à 60 caractères recommandés) et incluez le terme clé pour la page.

**Meta Description** — Un bref résumé ci-dessous le titre dans les résultats de recherche Google. Visez 150–160 caractères. Rendre descriptif et convaincant pour encourager les clics.

**Crawlable toggle** — Set to **Oui** (par défaut) pour permettre à Google d'indexer et de classer la page. Régler à **No** pour empêcher l'indexation.

Tous les champs SEO sont **locale-aware** — vous pouvez définir différents Meta Titres et Descriptions par langue, ce qui est important pour les marchés opérant en plusieurs langues.

---

## Cas d'utilisation

**Page de la catégorie SEO** — Définir un titre et une description de Meta riche en mots clés pour chaque catégorie de menu (p. ex. « KFC Chicken Burgers — Commander en ligne ») pour améliorer la visibilité de la recherche biologique.

** Optimisation de la page d'accueil** — Craft a convaincant Meta Description pour la page d'accueil qui résume l'expérience KFC et lance des clics à partir de requêtes de recherche de marque.

**Contenu de mise en scène en cache** — Lors de la préparation d'une page pour une campagne future, réglez la bascule Crawlable à **Non** jusqu'à ce que la page soit prête à être rendue publique, puis passez-la à **Oui** le jour du lancement.

**Référencement en plusieurs langues** — S'assurer que chaque localité possède son propre Meta Title et Description afin que les marchés non anglais apparaissent correctement dans les résultats de recherche en langues locales.

**Les pages d'information** — FAQ, sur les pages, et les pages de contenu nutritionnel peuvent toutes avoir une copie SEO personnalisée pour capturer le trafic de recherche longue queue.
