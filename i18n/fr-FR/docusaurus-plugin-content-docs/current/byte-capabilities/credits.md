# Crédits

Les capacités d'octets ont été étudiées et documentées par :

[**Varima Henry** et **Wesley Weidemann** dans Confluence](https://yumbrands.atlassian.net/wiki/spaces/ITBFR/overview?homepageId=3880354415)pour voir leur travail.

Ce travail a été réalisé à partir de 11 documents sources couvrant les spécifications fonctionnelles et techniques du portail Byte Helium, Byte Commerce Backend et Byte Portal — distillés dans un format accessible aux équipes de mise en marché, de lancement et de fonctions croisées.
