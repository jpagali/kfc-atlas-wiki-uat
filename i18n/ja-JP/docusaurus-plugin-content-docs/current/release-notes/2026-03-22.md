# リリース 2026.03.22

**2026年3月22日** にリリースされました。

このリリースでは、Brand Experience Team の org chart 改善に加えて、日本語コピーの英語ソースとの整合性を高め、Japanese admin sidebar も bilingual label で読みやすく整理しました。特に、誰が誰にレポートするのか、どこが dotted-line relationship なのかに加え、日本語 UI でどのセクションに入るのかも以前より追いやすくなっています。

## ハイライト

- **Brand Experience の reporting line がより自然に読める構成へ**
  Haley を中心に、Mark、Justin、Wes、Don の関係が視覚的に整理され、Brand 側の構造を追いやすくしました。

- **Current Team の focus 表示が Jim の直下で展開**
  `KFC Brand Experience`、`Byte Engineering`、`Shared Support` を選ぶと、overview を隠して、その branch を Jim の下に直接展開する drill-down 型の挙動へ変更しました。

- **Byte Engineering 側の上下関係をより明示**
  Matthew、Jonathan、Brendan、Phong、Steve、Don の関係が、現在の reporting context に沿って分かりやすく整理されました。

- **org chart がサイトの light / dark theme に追従**
  埋め込みの org chart が親ドキュメントのテーマ変更を受け取り、dark mode でも connector line が見やすくなるよう調整しました。

- **Brand Experience ページの重複テキストを整理**
  図の下に並んでいた補足説明や人物リストを外し、ページの主役が org chart になるよう軽量化しました。

- **日本語ナビゲーションとコピーを英語ソース寄りに整理**
  ホームページ、主要ドキュメント、日本語 admin sidebar を見直し、`Stores / 店舗` のような bilingual label と、より 1 対 1 に近い用語へ揃えました。

## 変更内容

### Brand Experience org chart

- Haley の役職表記を `Director, Head of Brand Experience` に更新しました
- Mark の役職表記を `Atlas Product Manager` に更新しました
- Don の配置や dotted-line の見せ方を調整し、Brand 側での見え方を整理しました
- dark mode で connector line が背景に埋もれにくいよう、線のコントラストを改善しました

### Current Team の表示挙動

- focus CTA を押したとき、overview をそのまま残すのではなく、選択した branch が Jim の直下に切り替わるよう変更しました
- `Current Team` を押すと、focus を解除して全体構造へ戻る挙動を追加しました
- Jim の下にあった不要な overview connector を削除し、ノイズを減らしました

### Byte Engineering / Shared Support

- Jim の下に Matthew を置き、現在の engineering branch の文脈が分かるようにしました
- Jonathan、Brendan、Phong、Steve、Don の current-state relationship を明示的に整理しました
- focus panel の構造不整合を直し、Engineering と Support が正しく切り替わるようにしました

### Future Team とページ整理

- Future Experience から不要な `Program Manager TBD Contractor Role` ボックスを削除しました
- `Market Product Owners` を KFC 側として扱う見た目に揃えました
- ページ本文にあった説明ブロックや、図の下の人物一覧を削除し、org chart 中心のページ構成にしました

### 日本語 localisation cleanup

- 日本語ホームページの stats と summary を見直し、モジュール名や platform attribution を英語ソースにより近づけました
- 日本語 admin sidebar を `Stores / 店舗`、`Menus / メニュー`、`Dayparts / デイパート` のような bilingual label に変更しました
- 高頻度で見られる日本語 docs の一部で、旧 translation pass から残っていた product name のズレや、途中で英語が混ざった手順文を整理しました

## ひとことで言うと

このリリースは、新しい機能追加というより「org chart と日本語体験を信頼できる表示に整える」ための改善です。Current Team の drill-down は分かりやすくなり、Brand と Engineering の reporting line だけでなく、英語版と日本語版を見比べたときの用語やナビゲーションの一貫性も高まりました。
