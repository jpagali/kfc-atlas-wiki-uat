# リリース 2026.03.20

**2026年3月20日** にリリースされました。

このリリースでは、Byte Commerce のシステムアーキテクチャ図の操作性を改善し、固定したノートを見失わずに図を探索できるようにしたほか、Atlas Wiki 全体の日本語ローカライズを大きく前進させ、Byte capability の technical source から元の FRD 参照へたどりやすくしました。

## ハイライト

- **固定したアーキテクチャノートを表示したまま図をスクロール可能に**
  ノートを固定した状態でも、その下にあるアーキテクチャマップを引き続きスクロールでき、参照しながら確認できます。

- **固定モードを維持したままノート内容を更新**
  一度ノートを固定した後に別のシステムボックスをクリックすると、固定を解除せずに内容だけがすぐ切り替わります。

- **埋め込みアーキテクチャのサイズ調整を改善**
  プラットフォームエクスプローラーとドキュメント iframe がアーキテクチャパネルに合わせてリサイズされるようになり、固定ノートが埋め込み内で切れなくなりました。

- **日本語カバレッジが全ドキュメントへ拡大**
  Byte Capabilities、リリースノート、新しい daypart ガイドを含め、すべての英語ドキュメントに日本語版が揃いました。

- **プラットフォームエクスプローラーを日本語対応**
  日本語のメンタルモデルページから開くエクスプローラーで、外側 UI、階層ビュー、システムアーキテクチャ表示まで日本語で確認できるようになりました。

- **Byte capability の FRD 参照を元要件へ直接リンク**
  Byte capability ドキュメント内の technical source ブロックで、明示されている Helium FRD と、承認済みの Portal / Commerce Backend FRD を元の要件ページへ直接たどれるようにしました。

- **Admin Portal Guide の Dayparts をトップレベル化**
  daypart ドキュメントを Menus 配下のサブカテゴリとして扱うのではなく、Stores や Menus と並ぶ独立セクションとして見つけやすくしました。

## 変更内容

### システムアーキテクチャ図

- `Pin this note` の操作を復元し、固定前でもホバーやクリックでプレビューできるようにしました
- 固定したノートを、背後の図をスクロールしても残る永続オーバーレイとして表示するよう更新しました
- ボックスを切り替えても固定モードを維持し、再固定せずに比較できるようにしました

### 埋め込みエクスプローラーのサイズ調整

- アーキテクチャ図からプラットフォームエクスプローラーのラッパーへ高さを通知する仕組みを追加しました
- プラットフォームエクスプローラーからドキュメント iframe へ高さを返す仕組みを追加しました
- メンタルモデルページの埋め込みが、エクスプローラーからのライブ高さ変更を監視するよう更新しました

### 日本語ローカライズ

- これまで欠けていた Byte Capabilities のドメイン、enablement、reference、release notes、daypart ドキュメントに日本語版を追加しました
- 日本語の navbar、footer、ホームページ、What's New、Contribute、Atlas Peek の文言を更新し、英語フォールバックを減らしました
- 新しく翻訳したドキュメントを検索できるよう、日本語検索インデックスを再生成しました

### プラットフォームエクスプローラーのローカライズ

- 日本語メンタルモデルページから platform explorer へ locale を渡すよう更新しました
- エクスプローラー外側の見出し、タブ、パネルラベル、フッター文言を日本語化しました
- システムアーキテクチャ表示のラベル、凡例、パネル UI、各ボックス説明も日本語化しました

### Byte Capability の参照リンク改善

- Byte capability の technical source セクションにある明示的な `FRD-HEL-*` 参照を、対応する Helium 要件アンカーへ直接リンクしました
- 必要な箇所では Helium の FRD 範囲表記を個別リンクへ展開し、各要件を直接開けるようにしました
- 正式な対応表がある `FRD-BCOM-*` と、承認済み正規化に基づく `FRD-PORTAL-*` をリンクし、未対応コードは平文のまま残しました

### Admin Portal Guide

- Dayparts を Menus 配下ではなく、Admin Portal Guide のトップレベルセクションへ昇格しました
- daypart の概要、定義、店舗別カスタマイズ、価格 / 提供可否、カテゴリ表示設定ガイドを新しい Dayparts セクション配下へ再配置しました
- Admin Portal Guide のトップページ、サイドバー、Stores / Menus からの導線を更新し、新しい Dayparts 階層へ揃えました

## ひとことで言うと

アーキテクチャ図がかなり使いやすくなり、日本語体験も大幅に改善し、Byte capability ドキュメントも参照元として使いやすくなりました。さらに、Dayparts も Menus の下位項目ではなく、Admin Portal Guide の独立ドメインとして見つけやすくなっています。ノートを固定したままシステムマップを見続けられるだけでなく、technical source にある要件参照から元の FRD に直接ジャンプできるようになっています。
