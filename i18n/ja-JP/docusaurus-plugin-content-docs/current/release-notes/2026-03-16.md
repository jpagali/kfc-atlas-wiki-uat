# リリース 2026.03.16

**2026 年 3 月 16 日**に公開されました。

このリリースでは、Atlas Wiki のホームページをよりシネマティックな KFC ブランド表現に刷新し、あわせて Admin Portal ガイドのナビゲーション構造も整理して、より辿りやすくしました。

## ハイライト

- **ホームページのヒーローに KFC の映像演出を追加**
  Atlas Wiki のトップページに、見出し、検索、主要 CTA の背後で再生されるフルブリードの KFC 調理映像を追加しました。

- **トップページがよりブランドらしい第一印象に**
  ヒーローの見た目に動きと存在感を持たせつつ、主要な検索導線とオンボーディング導線は引き続き前面に保っています。

- **ホームページの市場フラグが Windows でも安定表示に**
  市場カードのフラグ表示をプラットフォーム依存の絵文字から画像ベースのアイコンに切り替え、Mac 以外の端末でも日本とオーストラリアのカードが正しく表示されるようにしました。

- **主要市場カードのサイズを拡大**
  Australia のような市場名も不自然な折り返しや欠けが起きないよう、上部の市場カードを少し広く調整しました。

- **Admin Portal のナビゲーション構造を整理**
  サイドバー構造を見直し、主要セクションがより自然に理解できるようにすることで、パンくずや閲覧体験の一貫性を高めました。

- **モバイルでも読みやすさを維持**
  ヒーローのオーバーレイとレイヤー構成を調整し、小さい画面でもテキスト、検索、CTA が読みやすい状態を保てるようにしました。

- **ホームページに Atlas Peek 導線を追加**
  ホームページから専用の preview ページへ遷移できる `Atlas Peek` 導線を追加し、よりアプリらしい注文プロトタイプを確認できるようにしました。

## 変更内容

### ホームページ体験

- ホームページにシネマティックなフルブリード動画ヒーローを追加
- ヒーローの見出し、検索、CTA は動画の上に重ねて表示
- 動きのある映像上でも読みやすさを保つため、KFC ブランドのオーバーレイ表現を追加
- Front-end Guide と Admin Portal Guide への既存導線は維持
- `Atlas Peek` CTA とホームページ上の紹介ブロックを追加
- ホームページの市場フラグを絵文字からローカル画像アセットへ置き換え、クロスプラットフォーム表示を安定化
- Windows でも市場名がきれいに収まるよう、主要市場カードのサイズを拡大

### Atlas Peek プレビュー

- より豊かな prototype review のため、Atlas Wiki に専用の `/sneak-peek` ページを追加
- staged slide deck ではなく、モバイル注文アプリらしく振る舞う standalone Atlas Peek prototype を埋め込み
- 店舗選択、メニュー閲覧、商品詳細、カート、チェックアウト、完了までを持つ stateful な iPhone 風フローへ刷新
- add-to-cart の導線を明確化し、メニューからチェックアウトまでの流れが直感的に分かるよう改善
- prototype 内で英語と日本語を切り替えられるようにし、1 セッション内で両 locale を確認可能に
- 小さなブラウザ幅でも使いやすいよう、preview の viewport と scrolling 挙動を調整

### ナビゲーションと情報設計

- Admin Portal サイドバーのグルーピングを再構成し、全体構造が把握しやすい形に調整
- Admin Portal 内の主要セクション入口をより分かりやすく整理
- ネストされたセクションの見せ方を改善し、閲覧やパンくず文脈を把握しやすく調整

### レスポンシブ対応

- 背景動画が操作要素の背面に留まるよう、ヒーローのレイヤー構成を調整
- モバイルでの可読性を保つため、小さい画面向けのオーバーレイ挙動を最適化

## わかりやすく言うと

このリリースで、Atlas Wiki を開いた瞬間の印象がより KFC らしくなり、Atlas Peek prototype もレビューしやすくなり、同時に Admin Portal ドキュメントも迷いにくく辿りやすくなりました。
