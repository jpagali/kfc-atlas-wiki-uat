# リリース 2026.03.15

**2026 年 3 月 15 日**に公開されました。

このリリースでは、Atlas Wiki に専用の `What's New` 体験を追加し、サイトの主要導線からアクセスできるようにしました。あわせて、新しいページと導線が英語・日本語の両方で正しく動作するよう、ローカライズ対応も行いました。

## ハイライト

- **新しい `What's New` ページを公開**
  注目リリース、新機能、最近の改善、今後の機能を、より整理された形で確認できる専用ページを追加しました。

- **`What's New` をメインナビゲーションに追加**
  `Playbooks` の横に新しいトップナビ項目を追加し、`About` はトップナビから外しつつフッターでは引き続き利用できるようにしました。

- **フッターに twin カード導線を追加**
  既存の `The 12th Secret` に加えて、`What's New` カードを並べ、デスクトップでの発見性を高めました。

- **モバイルは軽量な導線を維持**
  モバイルでは、不要なフッター混雑を避けるため、`What's New` はハンバーガーメニュー経由でアクセスする形を維持しています。

- **日本語ルーティングとページ内容にも対応**
  新ページとリリースノート導線を日本語でも利用できるようにし、`en-US` と `ja-JP` の両方で動作するようにしました。

## 変更内容

### ナビゲーションと導線

- トップナビに `What's New` を追加
- トップナビから `About` を削除
- `About Atlas Wiki` はフッターから引き続きアクセス可能
- デスクトップ向けに `What's New` のフッター twin カードを追加
- モバイルはハンバーガーメニュー中心の導線を維持

### What's New 体験

- アプリ内に新しい `/whats-new` ページを追加
- ダークな KFC ブランドのローンチハブとしてデザイン
- ページ構成を以下に整理:
  - ロードマップ概要
  - 注目リリース
  - 新機能
  - 最近の改善
  - 近日公開
- 注目リリース領域のビジュアル表現を強化

### ローカライズ

- ナビバーの `What's New` に日本語コピーを追加
- `/ja-JP/whats-new` の日本語ページ内容を追加
- リリース関連導線が両ロケールで辿れるよう、日本語の release-notes も整備
- Atlas Chat のサジェストリンクからドキュメントを開く際、日本語ユーザーが page not found に遷移することがあったルーティング不具合を修正
- Atlas Peek プロトタイプの言語切り替えをシェルごとに見直し、モバイルアプリシェルでは上部ツールバー、Web シェルでは Web ナビ内から `EN/JP` を切り替えられるように改善

### レイアウト調整

- twin カード追加によってフッター列が窮屈になりすぎないよう、余白を調整
- 日本語フッターのリンクや CTA テキストが読みやすくなるよう、行間と文字サイズを改善

## わかりやすく言うと

このリリースで、ローンチ情報は見つけにくい changelog ではなく、実際のプロダクト導線として使える体験になりました。デスクトップではナビとフッターから、モバイルではハンバーガーメニューから見つけられ、英語・日本語の両方で安全に使えます。
