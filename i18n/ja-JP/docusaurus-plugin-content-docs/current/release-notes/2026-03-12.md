# リリース 2026.03.12

**2026 年 3 月 12 日**に公開されました。

このリリースは、Admin Portal Guide を大きく拡張したことが中心です。多数の運用向けコンテンツを追加し、ナビゲーションやリンクを整理し、記事ページをより使いやすくしました。

## ハイライト

- **Admin Portal Guide を大幅に拡張**
  運用トピックを大きく追加し、Atlas Wiki 内でより広い業務カバレッジを持つようになりました。

- **ナビゲーションとスラッグを整理**
  サイドバーのドキュメント ID、Markdown リンク、インデックスパスを修正し、拡大するガイドが Docusaurus 内で正しく解決されるようにしました。

- **記事構造を改善**
  Admin Portal Guide の内容を、よりドキュメント向きの構造に再編成し、記事の整理と画像まわりの扱いも改善しました。

- **画像表示をより安定化**
  Figma のスプラッシュ画像を削除し、画像ズームを追加し、SSR に関わる画像表示の問題を修正しました。

## わかりやすく言うと

このリリースで Admin Portal Guide が一気に実務で使いやすい形になりました。運用コンテンツが増え、大規模追加に伴う粗さが整えられ、ガイド全体のナビゲーションと保守性が改善されています。
