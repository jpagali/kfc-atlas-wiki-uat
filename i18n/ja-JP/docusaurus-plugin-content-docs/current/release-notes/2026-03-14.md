# リリース 2026.03.14

**2026 年 3 月 14 日**に公開されました。

このリリースでは、投稿ワークフローの拡張と、共有ナビゲーションの安定性回復に注力しました。大きな追加点は、新しい寄稿用 Contribution Studio と、ロケール切り替えをより安全な標準挙動へ戻したことです。

## ハイライト

- **新しい Contribution Studio を追加**
  リポジトリやプルリクエストの知識がなくても、記事を下書きし、プレビューし、Markdown をダウンロードし、公開依頼メールを開けるようになりました。

- **Contribute CTA をサイト導線に追加**
  ナビバーに `Contribute` を追加し、フッターには `The 12th Secret` として寄稿導線を設置しました。

- **日本語でも Contribution Studio を利用可能に**
  スタジオ画面の主要コピーを日本語対応し、両ロケールで新しい寄稿体験を使えるようにしました。

- **ロケール切り替えを標準機能へ戻して安定化**
  カスタム実装は不安定だったため、Docusaurus 標準のロケールドロップダウンへ戻し、EN/JP の挙動をフレームワーク側に任せる形にしました。

## わかりやすく言うと

このリリースで、git を触らずに Atlas Wiki へ寄稿できる、洗練された新しい導線が追加されました。同時に、ローカライズ挙動も安全な標準方式へ戻したことで、寄稿体験と多言語ナビゲーションの両方が安定しています。
