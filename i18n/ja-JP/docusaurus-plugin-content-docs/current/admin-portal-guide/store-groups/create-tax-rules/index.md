# 税ルールを作成する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で税ルールを作成する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Store Groups 画面に移動します。
**ステップ 2:** Once you find the store group you are looking for, click on the stacked dots to open the option window.

![Step 2](./img/step-2.png)

**ステップ 3:** taxes をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** this ボタン to create a new tax rule をクリックします。

![Step 4](./img/step-4.png)

**ステップ 5:** Type in the tax names and when the dates the taxes are effective from. You can also create or add a tax rule group that already exists.

![Step 5](./img/step-5.png)

**ステップ 5:** Make more selections or remove stores that already selected

![Step 6](./img/step-6.png)

**ステップ 6:** a matching condition in this dropdown. (For cart conditions) を選択します。

![Step 7](./img/step-7.png)

**ステップ 7:** a matching condition in this dropdown. (For product filters) を選択します。

![Step 8](./img/step-8.png)

**ステップ 8:** Press this ボタン to add tax. This will open up the field below and allows you input the rate mode, what is apply to, or tax amount %

**ステップ 9:** 入力します taxes on external fees

**ステップ 9:** the tax rule をクリックします。。

![Step 9](./img/step-9.png)

## 注意事項

:::note
You can click on each step to navigate throughout each flow
:::

:::note
This will be automatically selected to see other stores than you selected uncheck this
:::

:::note
Here you can review all of the information you filled out in each step.
:::

:::note
You can edit, copy, and delete tax rules.
:::
