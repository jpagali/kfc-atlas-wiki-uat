# 税ルールグループを作成する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で税ルールグループを作成する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Store Groups 画面に移動します。
**ステップ 2:** Once you find the store group you are looking for, click on the stacked dots to open the option window.

![Step 2](./img/step-2.png)

**ステップ 3:** taxes をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** on tax rule groups tab をクリックします。

![Step 4](./img/step-4.png)

**ステップ 5:** this ボタン to create a new tax rule group をクリックします。

**ステップ 6:** Create the name, display name, description and then hit create tax group to make the tax rule group

![Step 5](./img/step-5.png)

## 注意事項

:::note
You can edit, copy, and delete tax rule groups.
:::
