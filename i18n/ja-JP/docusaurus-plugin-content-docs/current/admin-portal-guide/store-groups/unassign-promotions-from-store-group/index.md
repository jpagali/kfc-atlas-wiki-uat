# ストアグループからプロモーションを解除する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でストアグループからプロモーションを解除する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Store groups 画面に移動します。
**ステップ 2:** To see all promotions assigned to a certain store group click this ボタン and the click Promotions.

![Step 2](./img/step-2.png)

**ステップ 3:** To unassign a promotions from a store group, click this unassign ボタン.

![Step 3](./img/step-3.png)

**ステップ 4:** After clicking Unassign on a promotion, a modal will pop up to make sure this was your intent. If so Just click Unassign Promotion.

![Step 4](./img/step-4.png)

## 追加情報

- ストアグループ - ストアグループからプロモーションを解除する
- On this screen you will see all the  store groups created. You can search for them and create a new group from this screen. As well as Edit, Delete and see the promotions assigned to that store group.
- Unassign a Promotion Dialogue
