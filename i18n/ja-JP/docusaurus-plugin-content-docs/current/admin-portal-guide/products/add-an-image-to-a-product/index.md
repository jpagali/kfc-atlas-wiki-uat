# 商品に画像を追加する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で商品に画像を追加する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Products 画面に移動します。
**ステップ 2:** the 3 dots to reveal a panel. Click Copy をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** each field as needed を入力します。

**ステップ 4:** 完了したら、click Save。

![Step 3](./img/step-3.png)

## 注意事項

:::note
商品は名称またはコードで検索できます。
:::

:::note
画面サイズによっては、他の列を表示するためにスクロールが必要です。
:::

:::note
There are other options in the window  but for this step we are just looking at Images. Others are discussed else where. Please go to the Table of Contents to find where.
:::

:::note
If you need to stop click Cancel. Please be aware that your info will not be saved.
:::

:::note
If you need this image to be the Primary image shown, click to toggle to Yes.
:::

:::note
If you need to add another image, click here and follow the same steps as in Step 3.
:::
