# 商品をコピーする

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で商品をコピーする手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Products 画面に移動します。
**ステップ 2:** the 3 dots to reveal a panel. Click Copy をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** Go through each step like you would creating a Product to add or change any fields as needed. Fill in each “*”必須項目 and other valuable information.

**ステップ 4:** When you have finished each step click Next or jump to the area you need by clicking on it in the stepper.

**ステップ 5:** 完了したら、click Create. The ボタン will turn blue as you make changes。

![Step 3](./img/step-3.png)

## 注意事項

:::note
商品は名称またはコードで検索できます。
:::

:::note
画面サイズによっては、他の列を表示するためにスクロールが必要です。
:::

:::note
There are other options in the window  but for this step we are just looking at Copy. Others are discussed else where. Please go to the Table of Contents to find where.
:::

:::note
Depending on your screen size you may need to scroll to see all the fields in each step.
:::

:::note
The product you copied will have most all of it’s info shown.
:::

:::note
作業を中止する場合はここをクリックしてください。入力内容は保存されません。
:::
