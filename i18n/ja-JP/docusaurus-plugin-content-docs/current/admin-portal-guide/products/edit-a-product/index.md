# 商品を編集する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で商品を編集する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Products 画面に移動します。
**ステップ 2:** the 3 dots to reveal a panel. Click Edit をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** To be taking to the area you need to make Edits to, click on the blue header of that section or to go step by step, click on Next. Make sure you fill in each “*”必須項目 and other valuable information.

**ステップ 4:** 完了したら、with your edits click Save. As you make changes the Save ボタン will be clickable。

![Step 3](./img/step-3.png)

## 注意事項

:::note
画面サイズによっては、すべての編集セクションを表示するためにブラウザを広げるか、スクロールする必要があります。
:::

:::note
作業を中止する場合はここをクリックしてください。入力内容は保存されません。
:::

:::note
商品は名称またはコードで検索できます。
:::

:::note
画面サイズによっては、他の列を表示するためにスクロールが必要です。
:::

:::note
There are other options in the window  but for this step we are just looking at Edit. Others are discussed else where. Please go to the Table of Contents to find where.
:::
