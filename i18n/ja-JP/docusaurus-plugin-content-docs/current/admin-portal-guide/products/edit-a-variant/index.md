# バリエーションを編集する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でバリエーションを編集する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Products 画面に移動します。

**ステップ 2:** the Variants tab をクリックします。
**ステップ 3:** You can search Variants by entering the Product Name or Option Values or by Tags.

**ステップ 4:** the 3 dots to reveal a panel. Click Edit をクリックします。

![Step 2](./img/step-2.png)

**ステップ 5:** each “*”必須項目 and other valuable information を入力します。

**ステップ 5:** the 3 dots and choose Availability. A drawer will pull out for you to edit をクリックします。

![Step 3](./img/step-3.png)

**ステップ 5:** the 3 dots and click Delete.  Please be aware that this can not be reversed and all Variant info will be deleted をクリックします。

**ステップ 6:** To add a Variant Code click inside the red box to reveal this drop down to fill in. Click Save when you are done.

![Step 4](./img/step-4.png)

**ステップ 6:** each “*”必須項目 and other valuable information を入力します。

**ステップ 6:** 完了したら、editing click the Save ボタン。

![Step 5](./img/step-5.png)

**ステップ 6:** any “*”必須項目 and other valuable information as needed in all drawers を入力します。

**ステップ 6:** each field as needed を入力します。

![Step 6](./img/step-6.png)

**ステップ 7:** 完了したら、editing click the Save ボタン。

**ステップ 7:** 完了したら、with your edits, click Save。

**ステップ 7:** 完了したら、with your adds/edits, click Save. The Save ボタン works the same way in all of the drawers。

**ステップ 8:** 完了したら、editing click the Save ボタン。

![Step 7](./img/step-7.png)
