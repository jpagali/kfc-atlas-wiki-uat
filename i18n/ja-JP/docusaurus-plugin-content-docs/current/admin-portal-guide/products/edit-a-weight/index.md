# 重みを編集する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で重みを編集する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Products 画面に移動します。
**ステップ 2:** the Weights tab をクリックします。

![Step 2](./img/step-2.png)

**ステップ 4:** the 3 dots to reveal a panel. Click Edit をクリックします。

**ステップ 4:** each “*”必須項目 and other valuable information を入力します。

**ステップ 5:** 完了したら、adding in all the information, click the Create Weight ボタン。

![Step 3](./img/step-3.png)

## 注意事項

:::note
There are other options in the window  but for this step we are just looking at Edit. Others are discussed else where. Please go to the Table of Contents to find where.
:::

:::note
作業を中止する場合はここをクリックしてください。入力内容は保存されません。
:::

:::note
Please read the box for important information before you save.
:::
