# スロットを編集する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でスロットを編集する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Products 画面に移動します。

**ステップ 2:** the Slots tab をクリックします。
**ステップ 3:** You can search Slots by entering the Names or Codes or by Tags.

**ステップ 4:** the 3 dots to reveal a panel. Click Edit をクリックします。

![Step 2](./img/step-2.png)

**ステップ 5:** on either of the blue links to go to that area and edit をクリックします。

**ステップ 6:** Edit the areas as needed making sure to fill in each “*”必須項目 and any other valuable information.

![Step 3](./img/step-3.png)

## 注意事項

:::note
There are other options in the window  but for this step we are just looking at Edit. Others are discussed else where. Please go to the Table of Contents to find where.
:::
