# 選択肢を作成する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で選択肢を作成する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Bundles 画面に移動します。
**ステップ 2:** on the choices tab をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** on the create new choice ボタン をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** Create a choice code and name

**ステップ 5:** Add a product the variants you want to include that are under the product

**ステップ 6:** Hit create choice.

![Step 4](./img/step-4.png)

## 追加情報

- バンドル - 選択肢を作成する
- Search by Bundle Name, Bundle Code, Catalog Tags, Promo Tags
- Search by Choice Name, Choice Code, Choice Display Name, and Catalog Tags
- Optional: Add more variants if needed and go through the same process you did for Step 5
