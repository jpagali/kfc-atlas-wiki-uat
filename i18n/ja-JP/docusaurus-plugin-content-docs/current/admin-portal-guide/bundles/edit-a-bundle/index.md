# バンドルを編集する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でバンドルを編集する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Bundles 画面に移動します。
**ステップ 2:** this  ボタン in the same row your bundle is in and then hit Edit をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** Go through this flow (same flow as creation) make changes and after you are done hit save.

![Step 3](./img/step-3.png)
