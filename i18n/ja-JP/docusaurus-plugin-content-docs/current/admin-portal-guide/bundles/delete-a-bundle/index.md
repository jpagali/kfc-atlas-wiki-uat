# バンドルを削除する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でバンドルを削除する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Bundles 画面に移動します。
**ステップ 2:** this  ボタン in the same row your bundle is in and then hit Delete をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** to delete をクリックします。

![Step 3](./img/step-3.png)

## 注意事項

:::note
Cancel at anytime if necessary
:::

:::note
Keep in my mind that deleting the bundle you selected will automatically take of any menu or category it is in.
:::
