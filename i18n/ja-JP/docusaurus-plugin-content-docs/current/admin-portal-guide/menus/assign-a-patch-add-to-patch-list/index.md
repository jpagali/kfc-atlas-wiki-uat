# パッチを割り当てる（Add to Patch List）

## このガイドで扱う内容

既存のパッチを削除せずに、店舗のアクティブな patch list に新しいパッチを追加する手順です。パッチは順番に適用され、後から重ねた内容が前の内容に上書きされます。

## 手順

**ステップ 1:** 左側のナビゲーションから **Menus** セクションに移動します。

**ステップ 2:** **Patches** タブをクリックして、利用可能なパッチ一覧を表示します。

![Step 2](./img/step-2.png)

**ステップ 3:** **Create New** をクリックして、新しいパッチ割り当てを開始します。

![Step 3](./img/step-3.png)

**ステップ 4:** **Add Patch to List** を選択して、既存の patch list にパッチを追加します。

![Step 4](./img/step-4.png)

**ステップ 5:** ドロップダウンから割り当てたい **Patch** を選択します。

![Step 5](./img/step-5.png)

**ステップ 6:** patch list 内での **Position** を選択します。複数のパッチが同じ項目を対象にしている場合、この順序が適用順になります。

![Step 6](./img/step-6.png)

**ステップ 7:** この割り当てを適用する **Channel** を選択します。

**ステップ 8:** このパッチを受け取る **Stores** を選択します。

**ステップ 9:** **Summary** で内容を確認し、**Save** をクリックして選択した店舗の patch list に追加します。

![Step 7](./img/step-7.png)

## 注意事項

:::note
この操作で、選択した店舗のアクティブな patch list に新しいパッチが追加されます。既存のパッチがある場合も、選択した位置に応じてそのまま保持されます。
:::
