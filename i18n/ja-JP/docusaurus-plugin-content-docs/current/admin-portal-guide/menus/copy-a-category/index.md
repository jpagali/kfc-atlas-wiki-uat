# カテゴリをコピーする

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でカテゴリをコピーする手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Menu 画面に移動します。
**ステップ 2:** on the categories folder をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** this  ボタン in the same row the category you’re looking for is in and then hit Copy をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** Create a category code and rewrite the category name if needed. The information you want to and hit save afterwards. (By default information from the original category like images and tags will be copied)

![Step 4](./img/step-4.png)
