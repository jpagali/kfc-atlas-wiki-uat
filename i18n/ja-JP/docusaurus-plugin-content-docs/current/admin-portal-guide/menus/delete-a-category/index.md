# カテゴリを削除する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でカテゴリを削除する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Menu 画面に移動します。
**ステップ 2:** on the categories folder をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** this  ボタン in the same row the category you’re looking for is in and then hit Delete をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** delete ボタン to delete the category をクリックします。

![Step 4](./img/step-4.png)

## 注意事項

:::note
Deleting a category will remove from all menus
:::
