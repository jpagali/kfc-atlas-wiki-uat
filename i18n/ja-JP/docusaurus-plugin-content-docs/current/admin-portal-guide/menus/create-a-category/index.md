# カテゴリを作成する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でカテゴリを作成する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Menu 画面に移動します。
**ステップ 2:** on the categories folder をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** on the categories folder をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** Fill this out. Create a category code and name and other options if needed then click the create category ボタン.

![Step 4](./img/step-4.png)
