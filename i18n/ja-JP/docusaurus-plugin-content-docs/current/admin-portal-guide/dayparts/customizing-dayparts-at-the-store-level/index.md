# 店舗レベルでデイパートをカスタマイズする

## 概要

デイパート自体は組織レベルでのみ作成できますが、オケージョンと時間枠は各店舗ごとにカスタマイズできます。

**スクリーンショットプレースホルダー:** 組織デフォルトと店舗上書きを示す、店舗レベルのデイパート継承図をここに配置

## Byte Portal で店舗レベルのデイパートをカスタマイズする方法

**ステップ 1:** ダッシュボードからサイドバーの **Stores** をクリックします。

**ステップ 2:** 対象の店舗を選択します。

**ステップ 3:** 右上のナビゲーションバーにある **Dayparts** タブをクリックします。

**スクリーンショットプレースホルダー:** 右上ナビゲーションに Dayparts タブが表示された店舗詳細画面をここに配置

**ステップ 4:** 編集したいデイパートの行の右端にある **ケバブアイコン** をクリックし、**Edit** を選択します。

**ステップ 5:** 必要に応じて変更します。既存の時間枠の編集、時間枠の追加、時間枠の削除が可能です。

**スクリーンショットプレースホルダー:** オケージョンと時間枠を編集できる店舗レベルのデイパート編集画面をここに配置

**ステップ 6:** 店舗レベルの設定を保存します。

このデイパートの **Status** 列には **Modified** が表示され、組織レベルのデフォルトとは異なる設定になっていることがわかります。

**スクリーンショットプレースホルダー:** Status 列に Modified が表示された店舗の Dayparts テーブルをここに配置

:::tip
初期状態では、すべての店舗が組織レベルの同じデイパート設定を継承します。本当に違うオケージョンや時間枠が必要な店舗だけを上書きしてください。
:::

:::note
デイパートの定義は組織レベルでのみ行えますが、対応オケージョンと時間枠は各店舗で調整できます。
:::

## デイパートを組織レベル設定へ戻す方法

**ステップ 1:** ダッシュボードからサイドバーの **Stores** をクリックします。

**ステップ 2:** 対象の店舗を選択します。

**ステップ 3:** 右上のナビゲーションバーにある **Dayparts** タブをクリックします。

**ステップ 4:** 編集したいデイパートの行の右端にある **ケバブアイコン** をクリックし、**Edit** を選択します。

**ステップ 5:** **Reset to Default** をクリックします。

**スクリーンショットプレースホルダー:** Reset to Default 操作が見える店舗レベルのデイパート編集画面をここに配置

**ステップ 6:** **Save** をクリックします。

このデイパートの **Status** 列には **Default** が表示され、組織レベル設定を継承している状態に戻ります。

**スクリーンショットプレースホルダー:** Status 列に Default が表示された店舗の Dayparts テーブルをここに配置
