# デイパートを定義する

## 概要

デイパートは、メニュー、商品、バンドルと同じ階層の **組織レベル** で設定します。一度定義すると、その組織内のすべての店舗ですぐに利用できるようになります。

各デイパートは、名前、一意のデイパートコード、対応するオケージョンの一覧で構成されます。各オケージョンには独自の時間枠を持たせることができます。初回作成時は、1 セットの時間枠を選択したすべてのオケージョンへ一括適用できます。作成後は、オケージョンごとに時間枠を個別調整できます。

**スクリーンショットプレースホルダー:** 組織、デイパート、オケージョン、時間枠の階層図をここに配置

## Byte Portal でデイパートを定義する方法

**ステップ 1:** 左側ナビゲーションから **Menus** を開きます。

**ステップ 2:** **Dayparts** タブをクリックして、既存のデイパート一覧を表示します。

**スクリーンショットプレースホルダー:** Dayparts タブと Create New Daypart ボタンが見える Menus 画面をここに配置

**ステップ 3:** **Create New Daypart** をクリックします。

**ステップ 4:** **name** と一意の **daypart code** を入力します。

| 項目 | 入力内容 | 補足 |
|-------|--------------|-------|
| **Name** | データ管理者とエンドユーザーの両方に表示される、わかりやすい名称 | 例: `Breakfast` |
| **Daypart code** | 外部システムで利用される、ブランド側で定義された一意の識別子 | 公開メニューにもこのコードが含まれます |

**スクリーンショットプレースホルダー:** Daypart Name と Daypart Code の項目がある Create New Daypart ドロワーをここに配置

**ステップ 5:** **Add time frame** をクリックします。

**ステップ 6:** このデイパートで利用する **occasions** を選択します。すべてのオケージョンに適用したい場合は、ドロップダウン内の **Select All** をクリックします。

**ステップ 7:** **曜日**（必須）、**開始時間** と **終了時間**（必須）、**開始日** と **終了日**（任意）を入力します。

**スクリーンショットプレースホルダー:** occasions、曜日、開始時間、終了時間、開始日、終了日がある Add Time Frame ドロワーをここに配置

**ステップ 8:** **Create time frame** をクリックします。

**ステップ 9:** 選択したすべてのオケージョンには、初期状態で同じ時間枠が適用されます。オケージョン名をクリックすると、個別に時間枠を編集できます。

**スクリーンショットプレースホルダー:** デイパート編集画面内で、オケージョンごとにまとめられた時間枠をここに配置

**ステップ 10:** **Save** をクリックします。

新しく作成したデイパートは、**Dayparts** タブの一覧に表示されます。

**スクリーンショットプレースホルダー:** 新しいデイパート行が表示された Dayparts テーブルをここに配置

## デイパートを編集する

**ステップ 1:** ダッシュボードからサイドバーの **Menus** をクリックします。

**ステップ 2:** **Dayparts** タブをクリックします。

**ステップ 3:** 編集したいデイパートの行の右端にある **ケバブアイコン** をクリックし、**Edit** を選択します。

**スクリーンショットプレースホルダー:** Edit が選択された Dayparts テーブルの行アクションメニューをここに配置

## デイパートを削除する

**ステップ 1:** ダッシュボードからサイドバーの **Menus** をクリックします。

**ステップ 2:** **Dayparts** タブをクリックします。

**ステップ 3:** 削除したいデイパートの行の右端にある **ケバブアイコン** をクリックし、**Delete** を選択します。

**スクリーンショットプレースホルダー:** Delete が選択された Dayparts テーブルの行アクションメニューをここに配置

このデイパートに紐づく価格設定や提供可否設定を含む、すべての関連設定が削除されます。

:::tip
最初にデイパートを作成する際、対応オケージョンが同じ挙動なら、まずは共通の時間枠をまとめて適用するのがおすすめです。必要なら後でオケージョンごとに細かく調整できます。
:::

:::note
デイパートは **組織レベル** で作成されますが、対応オケージョンと営業時間は **店舗レベル** でカスタマイズできます。
:::
