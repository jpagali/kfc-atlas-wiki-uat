# カテゴリのデイパート表示を設定する

## 概要

デイパートは Byte Portal 全体で利用でき、カテゴリの表示制御にも使えます。この機能はカテゴリ内商品の注文可否には影響しません。あくまで、利用側インターフェース（Byte POS、Byte Kiosk、Byte Commerce など）に対して、設定されたデイパート外ではそのカテゴリを非表示にしたり、表示優先度を下げたりするためのヒントを与えるものです。

**スクリーンショットプレースホルダー:** 組織レベルのデイパートがカテゴリに紐づく表示制御図をここに配置

## カテゴリにデイパート表示を設定する

**ステップ 1:** ダッシュボードからサイドバーの **Menus** をクリックします。

**ステップ 2:** **Categories** タブをクリックします。

**ステップ 3:** 対象のカテゴリを選択します。

**スクリーンショットプレースホルダー:** Categories タブが選択され、カテゴリ行を開ける Menus 画面をここに配置

**ステップ 4:** **Dayparts** セクションまでスクロールします。

**ステップ 5:** このカテゴリを表示したいデイパートを選択します。複数選択できます。何も選択しない場合、そのカテゴリは営業時間全体で表示されます。

**スクリーンショットプレースホルダー:** 1 つ以上のデイパートが選択されたカテゴリドロワーの Dayparts セクションをここに配置

**ステップ 6:** ドロワーフォームの **Save** をクリックします。

:::tip
カテゴリ表示は、そのカテゴリ内の商品が注文可能かどうかを変えるものではありません。利用側インターフェースへの表示ヒントとしてのみ機能します。
:::
