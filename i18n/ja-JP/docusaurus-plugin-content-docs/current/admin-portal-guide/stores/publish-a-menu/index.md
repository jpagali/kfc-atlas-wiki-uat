# メニューを公開する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でメニューを公開する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Stores 画面に移動します。
**ステップ 2:** Publish Menu を選択します。

![Step 2](./img/step-2.png)

**ステップ 3:** Stores を選択します。

![Step 3](./img/step-3.png)

**ステップ 4:** the channels you would like to publish the menu to を選択します。

![Step 4](./img/step-4.png)

**ステップ 5:** Publish を選択します。

![Step 5](./img/step-5.png)
