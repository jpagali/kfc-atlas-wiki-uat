# 店舗詳細を編集する

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal で店舗詳細を編集する手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Stores 画面に移動します。
**ステップ 2:** Once you find the store you are looking for, click on the stacked dots to open the option window.

![Step 2](./img/step-2.png)

**ステップ 3:** on Edit をクリックします。

**ステップ 4:** Edit each area as needed making sure to fill in each “*”必須項目.

**ステップ 5:** 完了したら、the Save ボタン will become active to click and save your new store。

![Step 3](./img/step-3.png)

## 注意事項

:::note
店舗は名称、番号、またはフランチャイズコードで検索できます。
:::

:::note
There are other options in the window  but for this step we are just looking at Edit. Others are discussed else where. Please go to the Table of Contents to find where.
:::

:::note
作業を中止する場合はここをクリックしてください。入力内容は保存されません。
:::
