# クーポンコード割引

## 目的

コード `SAVE10` を入力したお客様に $10 オフを適用する施策です。

## Basic Information

| Field | Value |
|---|---|
| **Automatic** | `No` |
| **Add Redemption Code** | `SAVE10` |

## Requirements

| Area | Field | Value |
|---|---|---|
| **Optional** | **Max Total Uses Per Customer** | `1` にすると 1 顧客 1 回に制限できます |
| **Optional** | **Max Number of Times a Promotion Can Be Used** | 全体の利用上限を設定します |

## Effects

| Area | Field | Value |
|---|---|---|
| **Order Level** | **Pricing Effect Type** | `Amount Off` |
| **Order Level** | **Amount Off** | `$10.00` |
