# アプリ限定オファー

## 目的

KFC モバイルアプリでのみ利用できる 20% オフ施策です。

| Area | Field | Value |
|---|---|---|
| **Channel** | **Select Channels** | `Mobile`, `iOS`, `Android` |

## Effects

| Area | Field | Value |
|---|---|---|
| **Order Level** | **Pricing Effect Type** | `Percent Off` |
| **Order Level** | **Percent Off** | `20` |
