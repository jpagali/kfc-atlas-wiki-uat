# 登録顧客限定オファー

## 目的

ログイン済みのロイヤルティ会員だけに 15% オフを提供する施策です。

## Requirements

| Area | Field | Value |
|---|---|---|
| **User Status** | **Is Registered Customer?** | `Yes` |

## Effects

| Area | Field | Value |
|---|---|---|
| **Order Level** | **Pricing Effect Type** | `Percent Off` |
| **Order Level** | **Percent Off** | `15` |
