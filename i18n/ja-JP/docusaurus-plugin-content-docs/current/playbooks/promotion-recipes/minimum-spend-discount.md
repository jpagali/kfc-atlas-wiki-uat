# 最低購入金額割引

## 目的

注文金額が $30 を超えた場合に、$5 オフを適用する施策です。

## Requirements

| Area | Field | Value |
|---|---|---|
| **Subtotal** | **Minimum Subtotal** | `$30.00` |
| **Subtotal** | **Calculation Mode** | `Actual Subtotal` |

## Effects

| Area | Field | Value |
|---|---|---|
| **Order Level** | **Pricing Effect Type** | `Amount Off` |
| **Order Level** | **Amount Off** | `$5.00` |
