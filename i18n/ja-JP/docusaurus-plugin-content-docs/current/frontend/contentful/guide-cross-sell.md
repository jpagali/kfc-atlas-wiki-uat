# ガイド: カート内クロスセル

Dynamic Cross-Sell を使うと、カートに入っている内容に応じて、どの商品をおすすめ表示するかをメニュー担当者が設定できます。適切なタイミングで適切な商品を見せるためのアップセル機能です。

---

## フロントエンドでの表示箇所

**Cart / Basket section** — 顧客が追加した商品一覧の下に、「You might also like」のようなおすすめ行が表示されます。ここに表示される商品は固定ではなく、カート内の商品構成によって変わります。

たとえば、顧客が Zinger Burger を追加した場合、Fries、Drink、Dessert などが表示されることがあります。別の商品が入っていれば、まったく異なる組み合わせが表示されます。

この機能は **Web** と **Mobile App** の両方で表示されます。

---

## Contentfulで設定する内容

1. Contentful で **Content Type: Dynamic Content Type** で絞り込む
2. **Add Entry** → **Dynamic Cross Sell** を選ぶ

このエントリには 3 つの主要項目があります。

**Name / Title** — 管理用ラベルです。顧客には表示されません。  
例: `Cross-sell for Zinger Burger`

**Menu Item** — トリガーになる商品です。この商品がカートに入ったときに、下のおすすめ商品が表示されます。**Select from Yum** から選択します。

**Cross Sell Products** — トリガー商品がカートにあるときに表示するおすすめ商品です。**Add Products** から複数選択できます。

3. **Publish** を押してルールを有効化する

---

## 利用例

**ドリンクのアップセル** — バーガーが入ったときにドリンクをおすすめし、セット化を促す。

**サイドメニュー提案** — 単品チキンが入ったときにポテトやサラダを出す。

**デザート提案** — メイン商品があるときにアイスやパイを見せ、平均客単価を上げる。

**キャンペーン連動表示** — 期間限定商品を、対象商品がカートに入ったときだけ強調表示する。
