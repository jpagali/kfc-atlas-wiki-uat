# ガイド: ホームページコンテンツ

ホームページは、お客様が KFC サイトに訪れたとき最初に見るページです。体験全体の印象を決める場所なので、常に最新で、その市場に合った見せ方に保つことが重要です。

---

## フロントエンドでの表示箇所

ホームページでは主に次の 2 要素が表示されます。

**Hero Banner** — ホームページ上部のフル幅バナーです。最も目立つ枠であり、通常はキャンペーン開始、季節プロモーション、新商品訴求などに使います。バナーは **locale ごと** に設定できるため、市場別に異なる画像を表示できます。

**Homepage Title** — ホーム上に表示される見出しで、これも locale ごとに編集できます。市場に合わせたあいさつや訴求文を入れる用途に向いています。

:::tip Visual Merchandising Guideline
ホームページバナーのメッセージはできるだけ**シンプル**にしてください。ディールの細かい説明は Deals ページ側に任せ、1〜2個の訴求要素に絞り、画像内テキストを増やしすぎないようにします。
:::

バナー画像は **3840×606px（PNG）** を基準にアップロードします。この中で中央の 2:1 比率領域が「安全表示領域」です。

- **右側** — マーケティング文言の配置に適しています
- **左側** — 中〜大画面ではローカライゼーションウィジェットに重なる可能性があります
- **グレーの余白領域** — 画面幅に応じて切り取られたり伸びたりするため、補助的なビジュアルだけに使います

## Contentfulで設定する内容

### ホームページタイトルを更新する

1. Contentful で **Home page** Content Type を検索する
2. Homepage エントリを開く
3. **Title - English** フィールドを探して文言を更新する
4. Preview で表示を確認し、問題なければ公開する

### ヒーローバナーを更新する

1. Homepage エントリを開き、**Background Images** までスクロールする
2. 更新したい locale の **Hero** エントリをクリックする
3. **Extra Small / Small / Medium / Large / Extra Large** の 5 つの画像サイズ枠が表示される
4. 対象画像または設定ボタンをクリックして編集する

**既存アセットを差し替える場合（推奨）:**

1. 画像枠をクリックして **Header Banner Asset** エディタを開く
2. **Open File Selector** を押して新しい画像をアップロードする
3. Contentful は JPEG、PNG、WebP などに対応している
4. 画像をアップロードまたはドラッグ＆ドロップする
5. **Publish Changes** を押して反映する

市場ごとに異なるバナーが必要な場合は、locale ごとに同じ手順を繰り返します。

## 利用例

**季節キャンペーンの開始** — 大型プロモーションや新商品施策の開始時に、新しいヒーローバナーへ差し替えます。Launch を使えば深夜公開も自動化できます。

**市場別ローカライズ** — ある市場では夏施策、別市場では冬施策など、同じタイミングでも locale ごとに異なる訴求ができます。

**A/B 的なバナー比較** — デプロイなしで locale ごとに異なるメッセージ訴求を試せます。

**鮮度維持** — ホームのヒーローを定期的に入れ替えることで、サイトの印象を古く見せにくくできます。
