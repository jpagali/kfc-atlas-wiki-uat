# ガイド: SEOタグと管理

Contentful の SEO フィールドを使うと、KFC の各ページが検索結果でどう見えるかを市場側で管理できます。ブラウザタブのタイトル、Google 検索結果の説明文、そもそも検索エンジンにインデックスさせるかどうかまで制御できます。

---

## フロントエンドでの表示箇所

SEO フィールド自体は、ページ内でお客様に直接見えるものではありませんが、主に次の場所に影響します。

**Browser tab** — Meta Title がブラウザタブ名として表示されます。

**Google search results** — Meta Title と Meta Description が Google 検索結果のスニペットに使われます。説明文が適切だとクリック率改善につながります。

**Crawlability toggle** — Google にそのページをクロール・インデックスさせるかどうかを制御します。**Yes** なら検索結果対象、**No** なら非公開ページのように扱えます。

---

## Contentfulで設定する内容

SEO フィールドは独立メニューではなく、複数の Content Type に埋め込まれています。以下のエントリ内にある **SEO** セクションを探してください。

- **Information Page**
- **Home Page**
- **Category**
- **Product Landing Page**（今後対応予定）
- **Product**（今後対応予定）

設定する主な項目:

**Meta Title** — ブラウザタブ名、および検索結果見出しに使われます。50〜60 文字程度を目安に、ページの主要キーワードを含めます。

**Meta Description** — 検索結果でタイトルの下に表示される説明文です。150〜160 文字程度を目安に、内容が分かりやすくクリックしたくなる文にします。

**Crawlable toggle** — **Yes** でインデックス許可、**No** で検索対象外にします。

SEO フィールドは **locale 対応** です。多言語市場では、言語ごとに別の Meta Title / Meta Description を設定できます。

---

## 利用例

**カテゴリページの SEO 強化** — 各メニューカテゴリにキーワード入りのタイトルと説明文を設定し、自然検索流入を狙う。

**ホームページ最適化** — ブランド検索時にクリックされやすい説明文を設定する。

**公開前ページの検索除外** — 将来施策ページを先に作っておき、公開準備が整うまでは Crawlable を **No** にしておく。

**多言語 SEO** — 言語ごとに適切な Meta 情報を設定し、各市場の検索言語に合わせて表示させる。

**情報ページの流入獲得** — FAQ、会社情報、栄養ページなども個別に SEO を調整できる。
