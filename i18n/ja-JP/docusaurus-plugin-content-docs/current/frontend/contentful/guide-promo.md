# ガイド: プロモーションコンテンツ

Promotion Content は、クーポンや販促情報をカートやチェックアウト画面でお客様に見せるための設定です。必要に応じて、通常商品と同じようにメニューカードとしてプロモーションを表示することもできます。

:::info
プロモーションは **先に Byte Commerce Admin Portal で作成しておく必要があります**。その後、Contentful の Promotion Content として表示設定を行います。
:::

---

## フロントエンドでの表示箇所

**Basket / Checkout** — 顧客が条件を満たす商品をカートに入れると、プロモーションタイルが表示されます。Display Name、Description、Requirement Text などが表示されます。

**Menu card** — プロモーションを画像付きカードとしてメニューグリッドに表示し、クーポンコードなしでも見つけられるようにできます。

---

## Contentfulで設定する内容

1. Contentful で **Promotion** Content Type を開く
2. **Select from Yum** を押して Byte Commerce Admin Portal で作成済みのプロモーション一覧を表示する
3. 設定したいプロモーションを選択する

**Display Name を設定する** — UI 上でお客様に見える名称です。locale ごとに設定します。

**Description を設定する** — プロモーションの追加説明です。locale ごとに設定します。

**Requirement Text を設定する** — 利用条件や発動条件です。locale ごとに設定します。

**Menu Card Image を設定する** — メニューカードとして見せる場合の画像です。既存メディアを使うか、新しくアップロードします。

5. **Publish** を押す  
   必要ならドロップダウンから特定日時での公開設定も行います

---

## 利用例

**新しいクーポンの表示** — 顧客が利用条件を満たしたとき、カート内で分かりやすく表示させる。

**メニュー上でディールを見せる** — プロモーションを通常商品のように表示し、顧客が発見しやすくする。

**期間限定施策** — スケジュール公開を使って、キャンペーン期間に合わせて出し入れする。

**市場別ローカライズ** — Display Name や Description を locale ごとに変え、市場に合った訴求にする。
