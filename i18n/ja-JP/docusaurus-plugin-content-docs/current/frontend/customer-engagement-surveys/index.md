# カスタマーエンゲージメント調査

カスタマーエンゲージメント調査は、注文、来店、またはサポート対応の後に、お客様から構造化されたフィードバックを収集するための仕組みです。Atlas では、主に **KFC Listens** のようなプログラムで利用され、満足度の把握、サービス課題の特定、市場チーム向けのインサイト抽出に使われます。

## この機能でできること

カスタマーエンゲージメント調査により、市場は以下を実施できます。

- 購入後のお客様にフィードバックを依頼する
- チャネル、店舗、キャンペーンごとの満足度を測定する
- 回答結果をレポーティング基盤へ連携し、分析に活用する
- 低評価、苦情、サービスリカバリーの対象に対してフォローアップを開始する

## カスタマージャーニー上のどこで表示されるか

調査への導線は、市場設定に応じて複数のタッチポイントに配置できます。

| タッチポイント | 表示例 |
|---|---|
| 注文完了 | チェックアウト後や確認画面で案内を出す |
| メールレシート | レシートメール内にアンケート CTA を含める |
| プッシュ通知 | 受け渡し完了後にフォローアップ案内を送る |
| 店頭掲示物 | QR コードやレシート URL からアンケートへ誘導する |

## 市場がこれを使う理由

KFC Listens のようなプログラムは、料理の品質、提供スピード、スタッフ対応、注文精度などについて、お客様から直接シグナルを得る手段になります。このフィードバックは市場にとって次の価値があります。

- 店舗パフォーマンスの比較
- 継続的に発生している課題の発見
- キャンペーンや商品変更の効果確認
- 低評価回答への優先的なサービスリカバリー

## 代表的な運用モデル

| 領域 | 主な担当 |
|---|---|
| アンケート設計とロジック | CX / Insights チーム |
| 招待トリガー設定 | デジタルプロダクトまたは CRM チーム |
| 回答レポーティング | 市場運営 / Insights |
| サービスリカバリー運用 | カスタマーサポートまたは店舗運営 |
