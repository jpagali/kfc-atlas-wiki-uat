# KFC Deeplink | Universal Link

## 機能概要

KFC Universal Link を使うと、対応する KFC の URL を開いたときに、ブラウザではなく KFC モバイルアプリ内の該当画面へ直接遷移できます。

**アプリがインストールされている場合**は、商品詳細、プロモーション、店舗ページなど、対応するアプリ内画面へ直接移動します。**アプリが入っていない場合**は、同じリンクが自然に Web 体験へフォールバックします。

その結果、マーケティングメール、店頭 QR、SNS 広告、プッシュ通知など、どの接点からでも、ユーザーを迷わせずに適切なアプリ体験へつなげられます。

## なぜ重要か

Universal Links がない場合、KFC コンテンツへの外部リンクは、ユーザーがアプリを持っていてもブラウザで開いてしまいます。これは不要な離脱要因になり、コンバージョン機会の損失にもつながります。

Universal Links を使えば、URL はそのまま、遷移先だけがユーザーの環境に応じて最適化されます。

## 対応範囲

| 項目 | 内容 |
|---|---|
| **対応ドメイン** | 市場ごとに設定された KFC 管理ドメイン |
| **ルーティング方式** | パスベース。各 URL パスがアプリ内の遷移先にマッピングされる |
| **アプリインストール済み** | 直接アプリ内画面へ遷移 |
| **アプリ未インストール** | 対応する Web 画面へ遷移 |
| **対応プラットフォーム** | iOS、Android |

## 主な流入元

Universal Links は主に次の導線で使われます。

- マーケティングメールや SMS
- QR コード（店頭、パッケージ、屋外掲出など）
- SNS 広告やオーガニック投稿
- プッシュ通知
- Web からアプリへの送客導線

## 対応プラットフォーム

| プラットフォーム | 対応状況 |
|---|---|
| iOS | ✅（Apple Universal Links） |
| Android | ✅（Android App Links） |
| Web | ✅（フォールバック遷移先） |
