Guía del Portal de Admin

Esta guía cubre todas las tareas disponibles en el Portal de Admin de Comercio Byte. Seleccione una sección abajo para empezar.

Reconocimiento de la fuente

Esta Guía del Portal de Admin se construyó a partir de la Guía de usuario de Prototipos ** del Equipo Chito y se adaptó a Atlas Wiki.

Abrir la Guía de usuario de Prototipo Byte Team en Figma

  • YumKnowledge Base: Guía de usuario del Portal de Admin
Byte POS Caveat

Las tareas del Portal de Admin que se refieren POS, in-store devices, o la comunicación tienda-a-POS suponen que el mercado está utilizando Byte POS.

Si un mercado es no usando Byte POS, Byte Connect debe estar a bordo como parte del paquete Byte Commerce para que el camino de integración se convierta en Byte Commerce - título Byte Connect - propiedad POS.

Tiendas​

Administrar ubicaciones de restaurantes, menús y guardar los ajustes.

Menús​

Crear y organizar menús y categorías de productos para cada tienda.

Salidas del día​

Gestione el comportamiento del menú basado en el tiempo, incluyendo la configuración de org-level daypart, overrides de nivel de tienda, precios, disponibilidad y visibilidad de la categoría.

Productos​

Definir elementos de menú individuales, ingredientes, precios y variantes.

Bundles​

Construye ofertas de comida y combos agrupando productos juntos.

Promociones​

Crear y gestionar ofertas promocionales y campañas de descuento.

Grupos de tiendas​

Organizar múltiples tiendas y gestionar ajustes y promociones a nivel de grupo.

Agentes​

Crear y gestionar cuentas de usuario con acceso y permisos específicos del portal.

Atención al cliente​

Buscar pedidos y clientes, emitir reembolsos, y ayudar con problemas de cuenta.

Dispositivos intraestables​

Administrar terminales y quioscos POS, incluyendo la configuración de dispositivos y la autenticación.
